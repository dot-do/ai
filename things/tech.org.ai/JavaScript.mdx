# JavaScript

JavaScript is a high-level, interpreted programming language that is one of the core technologies of the World Wide Web, alongside HTML and CSS. Originally created for client-side web development, JavaScript has expanded to server-side development (Node.js), mobile apps, desktop applications, and more.

## Key Features

- **Runs Everywhere**: Browsers, servers (Node.js), mobile, desktop
- **Event-Driven**: Asynchronous programming with callbacks, promises, async/await
- **Dynamically Typed**: Flexible variable types
- **First-Class Functions**: Functions as values
- **Prototype-Based OOP**: Inheritance through prototypes

## Example Code

```javascript
// Hello World
console.log("Hello, World!");

// Function (arrow syntax)
const greet = (name) => {
  return `Hello, ${name}!`;
};

// Async/Await
async function fetchData(url) {
  try {
    const response = await fetch(url);
    const data = await response.json();
    return data;
  } catch (error) {
    console.error('Error:', error);
  }
}

// Class
class Developer {
  constructor(name, languages) {
    this.name = name;
    this.languages = languages;
  }

  introduce() {
    return `I'm ${this.name}, I code in ${this.languages.join(', ')}`;
  }
}

const dev = new Developer('Alice', ['JavaScript', 'TypeScript', 'Python']);
```
