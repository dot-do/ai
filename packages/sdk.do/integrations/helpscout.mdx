# Help Scout Integration

Help Scout is a customer support platform with shared inbox capabilities and a knowledge base solution for customer service teams.

## Features

- **Conversations**: Manage email, chat, and phone conversations
- **Customers**: Maintain customer profiles and contact information
- **Mailboxes**: Organize conversations into shared inboxes
- **Tags**: Categorize and organize conversations
- **OAuth 2.0**: Secure authentication
- **Webhooks**: Real-time event notifications with SHA-1 signature verification

## Quick Start

```typescript
import { HelpscoutClient } from '@dotdo/sdk.do/integrations/helpscout'

const helpscout = new HelpscoutClient({
  clientId: process.env.HELPSCOUT_CLIENT_ID,
  clientSecret: process.env.HELPSCOUT_CLIENT_SECRET,
})

// Create a conversation
const conversation = await helpscout.conversation.create({
  subject: 'Need help',
  type: 'email',
  mailboxId: 123,
  customer: {
    email: 'customer@example.com',
  },
  threads: [
    {
      type: 'customer',
      text: 'I need help with my account',
    },
  ],
})

// Update conversation status
await helpscout.conversation.update({
  id: conversation.id,
  status: 'closed',
})
```

## Authentication

Help Scout uses OAuth 2.0 authentication:

```bash
export HELPSCOUT_CLIENT_ID=your_client_id
export HELPSCOUT_CLIENT_SECRET=your_client_secret
```

## Resources

### Conversations

Manage customer conversations:

```typescript
// Create conversation
const conversation = await helpscout.conversation.create({
  subject: 'Billing Question',
  type: 'email',
  mailboxId: 123,
  customer: {
    email: 'customer@example.com',
    firstName: 'John',
  },
  threads: [
    {
      type: 'customer',
      text: 'I have a question about my invoice',
    },
  ],
  status: 'active',
})

// Get conversation
const conv = await helpscout.conversation.get({ id: 456 })

// List conversations
const conversations = await helpscout.conversation.list({
  mailbox: 123,
  status: 'active',
})

// Update conversation
await helpscout.conversation.update({
  id: 456,
  status: 'closed',
})
```

### Customers

Manage customer profiles:

```typescript
// Create customer
const customer = await helpscout.customer.create({
  firstName: 'Jane',
  lastName: 'Smith',
  emails: [
    {
      type: 'work',
      value: 'jane@company.com',
    },
  ],
  phones: [
    {
      type: 'work',
      value: '+1-555-0100',
    },
  ],
})

// Get customer
const customer = await helpscout.customer.get({ id: 789 })

// Search customers
const customers = await helpscout.customer.list({
  query: 'jane@company.com',
})

// Update customer
await helpscout.customer.update({
  id: 789,
  firstName: 'Jane',
  lastName: 'Doe',
})
```

### Mailboxes

Access mailbox information:

```typescript
// Get mailbox
const mailbox = await helpscout.mailbox.get({ id: 123 })

// List all mailboxes
const mailboxes = await helpscout.mailbox.list()
```

### Tags

Work with conversation tags:

```typescript
// List tags
const tags = await helpscout.tag.list()
```

## Webhooks

Help Scout supports webhooks with SHA-1 HMAC signature verification:

```typescript
import { HelpscoutWebhookHandler, WebhookEventRouter } from '@dotdo/sdk.do/integrations/helpscout'

const handler = new HelpscoutWebhookHandler(process.env.HELPSCOUT_WEBHOOK_SECRET)
const event = await handler.handleRequest(request)

const router = new WebhookEventRouter()
router.onConversationCreated(async (event) => {
  console.log('New conversation:', event.data)
})

router.onConversationStatus(async (event) => {
  console.log('Status changed:', event.data)
})

await router.route(event)
```

## Best Practices

1. **Rate Limits**: 400 requests per minute per account
2. **OAuth Tokens**: Refresh tokens before expiration
3. **Webhooks**: Verify SHA-1 signatures for security
4. **Pagination**: Use page-based pagination (default 50, max 100)
5. **Conversation Types**: Support email, chat, and phone conversations
6. **Customer Search**: Use email addresses for precise customer lookup
