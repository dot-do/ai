# Zendesk Integration

Zendesk is a leading customer support and ticketing platform that helps businesses manage customer service operations.

## Features

- **Tickets**: Create and manage support tickets
- **Users**: Manage customers, agents, and admins
- **Organizations**: Segment customers into organizations
- **Comments**: Add comments and notes to tickets
- **API Key Auth**: Simple authentication with API tokens
- **Webhooks**: Real-time event notifications with signature verification

## Quick Start

```typescript
import { ZendeskClient } from '@dotdo/sdk.do/integrations/zendesk'

const zendesk = new ZendeskClient({
  apiKey: process.env.ZENDESK_API_TOKEN,
  subdomain: 'your-subdomain', // your-subdomain.zendesk.com
})

// Create a ticket
const ticket = await zendesk.ticket.create({
  subject: 'Need help with billing',
  comment: {
    body: 'Customer inquiry about billing',
  },
  priority: 'normal',
  status: 'new',
})

// Update ticket
await zendesk.ticket.update({
  id: ticket.id,
  status: 'open',
  comment: {
    body: 'Working on this issue',
  },
})
```

## Authentication

Zendesk uses API token authentication:

```bash
export ZENDESK_API_TOKEN=your_api_token
export ZENDESK_SUBDOMAIN=your-subdomain
```

## Resources

### Tickets

Manage support tickets:

```typescript
// Create ticket
const ticket = await zendesk.ticket.create({
  subject: 'Order issue',
  comment: {
    body: 'Customer needs help with order #1234',
  },
  priority: 'high',
  tags: ['order', 'urgent'],
})

// Get ticket
const ticket = await zendesk.ticket.get({ id: 123 })

// Update ticket
await zendesk.ticket.update({
  id: 123,
  status: 'solved',
  comment: {
    body: 'Issue resolved',
  },
})

// List tickets
const tickets = await zendesk.ticket.list({ per_page: 50 })
```

### Users

Manage users:

```typescript
// Create user
const user = await zendesk.user.create({
  name: 'John Doe',
  email: 'john@example.com',
  role: 'end-user',
})

// Get user
const user = await zendesk.user.get({ id: 456 })

// List agents
const agents = await zendesk.user.list({ role: 'agent' })
```

### Organizations

Manage organizations:

```typescript
// Create organization
const org = await zendesk.organization.create({
  name: 'Acme Inc',
  domain_names: ['acme.com'],
  details: 'Enterprise customer',
})

// Get organization
const org = await zendesk.organization.get({ id: 789 })

// List organizations
const orgs = await zendesk.organization.list()
```

### Comments

View ticket comments:

```typescript
// List comments
const comments = await zendesk.comment.list({ ticket_id: 123 })
```

## Webhooks

Zendesk supports webhooks with HMAC SHA256 signature verification:

```typescript
import { ZendeskWebhookHandler, WebhookEventRouter } from '@dotdo/sdk.do/integrations/zendesk'

const handler = new ZendeskWebhookHandler(process.env.ZENDESK_WEBHOOK_SECRET)
const event = await handler.handleRequest(request)

const router = new WebhookEventRouter()
router.onTicketCreated(async (event) => {
  console.log('New ticket:', event.data)
})

await router.route(event)
```

## Best Practices

1. **Rate Limits**: Zendesk enforces rate limits (varies by plan)
2. **Pagination**: Use pagination for large datasets (max 100 per page)
3. **Ticket Status**: Follow status flow: new → open → pending → solved → closed
4. **Organization**: Use organizations to segment customers
5. **Tags**: Use tags for categorization and automation
6. **Webhooks**: Verify webhook signatures for security
