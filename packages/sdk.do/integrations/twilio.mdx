# Twilio Cloud Communications

Twilio is the leading cloud communications platform that enables developers to programmatically make and receive phone calls, send and receive text messages, and perform other communication functions using web service APIs.

## Features

- **SMS & MMS**: Send and receive text messages with media attachments globally
- **Voice**: Make and receive voice calls with programmable call flows (TwiML)
- **Verify**: Two-factor authentication and phone number verification
- **Video**: Programmable video for web and mobile applications
- **Conversations**: Multi-channel messaging (SMS, WhatsApp, Facebook Messenger)
- **Phone Numbers**: Purchase and manage phone numbers in 100+ countries
- **Call Recording**: Record and store voice conversations
- **Conferences**: Multi-party conference calls with programmable controls

## Quick Start

```typescript
import { TwilioClient } from '@dotdo/sdk.do/integrations/twilio'

const twilio = new TwilioClient({
  accountSid: process.env.TWILIO_ACCOUNT_SID,
  authToken: process.env.TWILIO_AUTH_TOKEN,
})

// Send an SMS
const message = await twilio.sendSMS({
  from: '+15551234567',
  to: '+15559876543',
  body: 'Hello from Twilio!',
})

// Make a call
const call = await twilio.makeCall({
  from: '+15551234567',
  to: '+15559876543',
  twiml: '<Response><Say>Hello World</Say></Response>',
})
```

## Authentication

Twilio uses Account SID and Auth Token for authentication. Set these as environment variables:

```bash
export TWILIO_ACCOUNT_SID=ACxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
export TWILIO_AUTH_TOKEN=your_auth_token_here
```

**Important**: Never expose your Auth Token in client-side code or public repositories.

## Webhooks

Twilio sends webhooks for asynchronous events. Always verify webhook signatures:

```typescript
import { TwilioWebhooks } from '@dotdo/sdk.do/integrations/twilio'

// Verify webhook signature
const isValid = TwilioWebhooks.verifySignature(
  'https://example.com/webhook', // Full URL
  req.body, // Webhook parameters
  req.headers['x-twilio-signature'],
  process.env.TWILIO_AUTH_TOKEN
)

if (!isValid) {
  return res.status(403).send('Invalid signature')
}

// Parse webhook event
const event = TwilioWebhooks.parseEvent(req.body)

// Handle event
switch (event.type) {
  case 'message.delivered':
    console.log('Message delivered:', event.data.MessageSid)
    break
  case 'call.completed':
    console.log('Call completed:', event.data.CallSid)
    break
}
```

### Common Webhook Events

**SMS/Message Events**:

- `message.sent` - Message has been sent
- `message.delivered` - Message delivered to recipient
- `message.failed` - Message failed to send
- `message.received` - Incoming message received

**Call Events**:

- `call.initiated` - Call initiated
- `call.ringing` - Phone is ringing
- `call.answered` - Call answered
- `call.completed` - Call completed
- `call.failed` - Call failed

**Recording Events**:

- `recording.available` - Recording available for download
- `recording.failed` - Recording failed

**Conference Events**:

- `conference.start` - Conference started
- `conference.end` - Conference ended
- `conference.participant.join` - Participant joined
- `conference.participant.leave` - Participant left

## TwiML (Twilio Markup Language)

TwiML is Twilio's XML-based language for controlling voice and messaging:

```typescript
import { createTwiML, say, play, hangup } from '@dotdo/sdk.do/integrations/twilio'

// Simple greeting
const twiml = say('Welcome to our service!')

// Complex call flow
const twiml = createTwiML(`
  <Say voice="alice">Thank you for calling</Say>
  <Pause length="1"/>
  <Gather action="/handle-input" method="POST" numDigits="1">
    <Say>Press 1 for sales, 2 for support</Say>
  </Gather>
  <Say>We didn't receive any input</Say>
  <Redirect>/main-menu</Redirect>
`)
```

## Rate Limits

Twilio enforces rate limits based on your account type:

- **Default**: 60 requests per second
- **High Volume**: 1000+ requests per second (contact sales)

Handle rate limit errors with exponential backoff.

async function send2FACode(phoneNumber: string) {

async function verify2FACode(phoneNumber: string, code: string) {

  })

### Make Automated Reminder Call

```typescript
async function sendReminderCall(phoneNumber: string, reminderText: string) {
  return await twilio.makeCall({
    from: '+15551234567',
    to: phoneNumber,
    twiml: `
      <Response>
        <Say voice="alice">${reminderText}</Say>
        <Say>Press any key to confirm</Say>
        <Gather numDigits="1" action="/confirm-reminder"/>
      </Response>
    `,
  })
}
```

### Set Up Conference Call

```typescript
async function setupConferenceCall(participants: string[]) {
  const conferenceName = `Meeting-${Date.now()}`

  // Call each participant
  for (const participant of participants) {
    await twilio.makeCall({
      from: '+15551234567',
      to: participant,
      twiml: `
        <Response>
          <Say>Joining conference call</Say>
          <Dial>
            <Conference>${conferenceName}</Conference>
          </Dial>
        </Response>
      `,
    })
  }

  return conferenceName
}
```
