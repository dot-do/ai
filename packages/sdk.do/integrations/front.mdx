# Front Integration

Front is a customer communication hub that combines email, chat, SMS, and other channels into shared team inboxes for collaborative customer support.

## Features

- **Messages**: Send and manage messages across channels
- **Conversations**: Handle customer conversations collaboratively
- **Contacts**: Manage customer contact information
- **Tags**: Organize conversations with tags
- **Comments**: Add internal team comments to conversations
- **Webhooks**: Real-time event notifications
- **API Key Auth**: Simple authentication

## Quick Start

```typescript
import { FrontClient } from '@dotdo/sdk.do/integrations/front'

const front = new FrontClient({
  apiKey: process.env.FRONT_API_TOKEN,
})

// Send a message
const message = await front.message.send({
  channel_id: 'cha_123',
  to: ['customer@example.com'],
  subject: 'Re: Your inquiry',
  body: 'Thanks for reaching out...',
})

// Update conversation
await front.conversation.update({
  conversation_id: 'cnv_123',
  status: 'open',
  assignee_id: 'tea_456',
})
```

## Authentication

Front uses API token authentication:

```bash
export FRONT_API_TOKEN=your_api_token
```

## Resources

### Messages

Send and retrieve messages:

```typescript
// Send message
const message = await front.message.send({
  channel_id: 'cha_123',
  to: ['customer@example.com'],
  subject: 'Welcome!',
  body: '<p>Welcome to our service!</p>',
  text: 'Welcome to our service!',
})

// Get message
const msg = await front.message.get({ message_id: 'msg_123' })

// List conversation messages
const messages = await front.message.list({
  conversation_id: 'cnv_123',
})
```

### Conversations

Manage conversations:

```typescript
// Get conversation
const conversation = await front.conversation.get({
  conversation_id: 'cnv_123',
})

// List conversations
const conversations = await front.conversation.list({
  q: 'status:open',
})

// Update conversation
await front.conversation.update({
  conversation_id: 'cnv_123',
  status: 'archived',
  assignee_id: 'tea_456',
  tag_ids: ['tag_123', 'tag_456'],
})
```

### Contacts

Manage customer contacts:

```typescript
// Create contact
const contact = await front.contact.create({
  name: 'John Doe',
  handles: [
    { handle: 'john@example.com', source: 'email' },
    { handle: '+15551234567', source: 'phone' },
  ],
  custom_fields: {
    company: 'Acme Inc',
    plan: 'pro',
  },
})

// Update contact
await front.contact.update({
  contact_id: contact.id,
  custom_fields: {
    plan: 'enterprise',
  },
})

// List contacts
const contacts = await front.contact.list({ q: 'acme' })
```

### Tags

Organize with tags:

```typescript
// Create tag
const tag = await front.tag.create({
  name: 'urgent',
  description: 'Urgent customer issues',
})

// List tags
const tags = await front.tag.list()

// Delete tag
await front.tag.delete({ tag_id: tag.id })
```

### Comments

Add internal comments:

```typescript
// Add comment
const comment = await front.comment.create({
  conversation_id: 'cnv_123',
  body: 'Customer called, follow up tomorrow',
  author_id: 'tea_456',
})

// List comments
const comments = await front.comment.list({
  conversation_id: 'cnv_123',
})
```

## Webhooks

Front supports webhooks for real-time notifications:

```typescript
import { FrontWebhookHandler, WebhookEventRouter } from '@dotdo/sdk.do/integrations/front'

const handler = new FrontWebhookHandler(process.env.FRONT_WEBHOOK_SECRET)
const event = await handler.handleRequest(request)

const router = new WebhookEventRouter()
router.onConversationCreated(async (event) => {
  console.log('New conversation:', event.data)
})

await router.route(event)
```

## Best Practices

1. **Rate Limits**: Front enforces rate limits - implement exponential backoff
2. **Shared Inboxes**: Use team assignments for collaborative support
3. **Tags**: Use tags for workflow and prioritization
4. **Custom Fields**: Store additional contact data in custom fields
5. **Webhooks**: Subscribe to relevant events only
6. **Pagination**: Use cursor-based pagination for large datasets
