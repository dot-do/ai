# Zoom Integration

Zoom is a leading video conferencing and virtual meeting platform with comprehensive API support for meetings, webinars, users, and recordings.

## Features

- **Meetings**: Schedule and manage Zoom meetings
- **Webinars**: Create and manage webinars
- **Users**: Manage Zoom users and accounts
- **Recordings**: Access and manage cloud recordings
- **OAuth 2.0**: Secure authentication
- **Webhooks**: Real-time event notifications

## Quick Start

```typescript
import { ZoomClient } from '@dotdo/sdk.do/integrations/zoom'

const zoom = new ZoomClient({
  clientId: process.env.ZOOM_CLIENT_ID,
  clientSecret: process.env.ZOOM_CLIENT_SECRET,
})

// Create a meeting
const meeting = await zoom.meeting.create({
  userId: 'me',
  topic: 'Team Sync',
  type: 2,
  start_time: '2025-10-15T10:00:00Z',
  duration: 60,
})

console.log('Join URL:', meeting.join_url)
```

## Authentication

Zoom uses OAuth 2.0 authentication. Register your app in the Zoom App Marketplace:

```bash
export ZOOM_CLIENT_ID=your_client_id
export ZOOM_CLIENT_SECRET=your_client_secret
```

## Resources

### Meetings

Schedule and manage meetings:

```typescript
// Create meeting
const meeting = await zoom.meeting.create({
  userId: 'me',
  topic: 'Weekly Standup',
  type: 2, // Scheduled meeting
  start_time: '2025-10-15T10:00:00Z',
  duration: 30,
  settings: {
    join_before_host: true,
    mute_upon_entry: true,
  },
})

// Update meeting
await zoom.meeting.update({
  meetingId: meeting.id,
  topic: 'Updated Topic',
  duration: 45,
})

// Delete meeting
await zoom.meeting.delete({ meetingId: meeting.id })
```

### Webinars

Create and manage webinars:

```typescript
// Create webinar
const webinar = await zoom.webinar.create({
  userId: 'me',
  topic: 'Product Launch',
  type: 5,
  start_time: '2025-10-20T14:00:00Z',
  duration: 120,
})

// Get webinar
const webinarInfo = await zoom.webinar.get({ webinarId: webinar.id })
```

### Users

Manage Zoom users:

```typescript
// Get user
const user = await zoom.user.get({ userId: 'me' })

// List users
const users = await zoom.user.list({ status: 'active' })
```

### Recordings

Access cloud recordings:

```typescript
// List recordings
const recordings = await zoom.recording.list({
  userId: 'me',
  from: '2025-10-01',
  to: '2025-10-31',
})

// Get recording
const recording = await zoom.recording.get({ meetingId: 'meeting-id' })

// Delete recording
await zoom.recording.delete({ meetingId: 'meeting-id' })
```

## Webhooks

Zoom supports webhooks for real-time event notifications:

```typescript
import { ZoomWebhookHandler, WebhookEventRouter } from '@dotdo/sdk.do/integrations/zoom'

const handler = new ZoomWebhookHandler(process.env.ZOOM_WEBHOOK_SECRET)
const event = await handler.handleRequest(request)

const router = new WebhookEventRouter()
router.onMeetingStarted(async (event) => {
  console.log('Meeting started:', event.data)
})

await router.route(event)
```

## Best Practices

1. **OAuth Tokens**: Refresh tokens before expiration
2. **Rate Limits**: Zoom enforces rate limits (varies by plan)
3. **Webhooks**: Verify webhook signatures
4. **Meeting Types**: Use correct type codes (1=instant, 2=scheduled, etc.)
5. **Timezones**: Always specify timezone for scheduled meetings
6. **Pagination**: Use pagination for large result sets
