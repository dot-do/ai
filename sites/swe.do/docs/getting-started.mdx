# Getting Started with swe.do

Build your first Software Engineer agent in 5 minutes.

## Installation

```bash
pnpm add sdk.do
```

## Your First Software Engineer Agent

```typescript
import $, { agent } from 'sdk.do'

const agentInstance = await agent.create({
  $type: 'Agent',
  name: 'Software Engineer',
  capabilities: ['code_generation', 'debugging', 'testing', 'code_review', 'refactoring', 'documentation'],
  model: 'claude-sonnet-4.5',
})

// Execute task
const result = await agentInstance.execute({
  task: 'Feature implementation',
  context: {
    /* your context */
  },
})
```

- **Code Generation**: Code generation
- **Debugging**: debugging
