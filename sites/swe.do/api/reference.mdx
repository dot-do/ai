# Software Engineer Agent API Reference

Complete API documentation for swe.do.

## Agent Creation

```typescript
const softwareengineer = await agent.create({
  $role: $.SoftwareEngineer,
  name: string,
  capabilities: string[],
  tools: Record<string, Tool>,
  model: string
})
```

## Execution

```typescript
const result = await softwareengineer.execute({
  task: string,
  context?: Context
})
```

## Configuration

```typescript
await softwareengineer.configure({
  model?: string,
  temperature?: number,
  capabilities?: string[]
})
```

## Events

```typescript
softwareengineer.on('task_complete', (result) => {
  console.log(result)
})
```
