# Basic Software Engineer Agent Usage

Simple examples for swe.do.

## Example: Implement REST API endpoint

```typescript
import $, { agent } from 'sdk.do'

const softwareengineer = await agent.create({
  $role: $.SoftwareEngineer,
  model: 'claude-sonnet-4.5',
})

const result = await softwareengineer.execute({
  task: 'Implement REST API endpoint',
})
```

See [Advanced Patterns](./advanced-patterns) for complex examples.
