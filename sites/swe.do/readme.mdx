# swe.do

**Software Engineer Agent for Autonomous Operations**

swe.do provides autonomous software engineer agents that handle Writing production-quality code, implementing features, fixing bugs, writing tests, code review. Build digital workers that can implement rest api endpoint with the same expertise as experienced professionals.

## Quick Start

```bash
pnpm add sdk.do
```

```typescript
import $, { agent } from 'sdk.do'

const softwareengineer = await agent.create({
  $type: 'Agent',
  $role: $.SoftwareEngineer,
  name: 'Software Engineer Agent',
  capabilities: ['code_generation', 'debugging', 'testing', 'code_review', 'refactoring', 'documentation'],
  tools: ['github', 'vscode', 'terminal', 'debugger', 'test_runner'],
  model: 'claude-sonnet-4.5',
})

const result = await softwareengineer.execute({
  task: 'Implement REST API endpoint',
  context: {
    /* your context */
  },
})
```

## Integration with .do Platform

```typescript
import $, { agent, db, on, send, every } from 'sdk.do'

const softwareengineer = await agent.create({
  $role: $.SoftwareEngineer,
  tools: { db, on, send },
})

await on($.Task.created, async (task) => {
  await softwareengineer.execute(task)
})
```
