# validate.do

**Validate data, schemas, and business rules with type-safe validation patterns**

validate.do provides powerful utilities for data validation in your Business-as-Code applications. Built with semantic patterns, type safety, and integration with the entire `.do` platform ecosystem.

## Quick Start

```bash
pnpm add validate.do
```

```typescript
import $, { validate } from 'validate.do'

// Basic usage
const result = await validate({
  // Your data
  data: 'example',
  // Options
  options: {},
})

console.log('Result:', result)
```

## Key Features

### Semantic Patterns

Use `$.Subject.predicate.Object` patterns for type-safe operations:

```typescript
import $ from 'validate.do'

// Semantic patterns
const result = await validate($.Data, {
  // Configuration
})
```

### Type Safety

Full TypeScript support with comprehensive type definitions:

```typescript
import type { ValidateOptions, ValidateResult } from 'validate.do'

const options: ValidateOptions = {
  // Type-safe configuration
}

const result: ValidateResult = await validate(options)
```

### Integration Ready

Seamlessly integrates with other `.do` platform services:

```typescript
import { validate } from 'validate.do'
import { db, ai } from 'sdk.do'

// Integrate with database
const data = await db.list($.Data)
const result = await validate(data)

// Integrate with AI
const processed = await ai.generate('process', { data: result })
```

## TypeScript Support

validate.do is built with TypeScript and provides full type safety:

```typescript
import type { ValidateOptions, ValidateResult, ValidateError } from 'validate.do'

// Type-safe operations
const options: ValidateOptions = {
  // Configuration
}

try {
  const result: ValidateResult = await validate(options)
  console.log('Success:', result)
} catch (error: ValidateError) {
  console.error('Error:', error.message)
}
```

## Best Practices

### Use Semantic Patterns

```typescript
// Good - semantic patterns
const result = await validate($.Data, options)

// Avoid - string-based
const result = await validate('Data', options)
```

### Handle Errors

```typescript
// Good - error handling
try {
  const result = await validate(data)
  return result
} catch (error) {
  console.error('Failed:', error)
  throw error
}
```

### Use TypeScript

```typescript
// Good - type-safe
import type { Options } from 'validate.do'
const options: Options = {}

// Avoid - untyped
const options = {}
```
