# Structured Output Generation

Generate structured JSON objects validated against Schema.org types.

## Basic Structured Output

Generate a Product with schema validation:

```typescript
import { $, ai } from 'sdk.do'

const product = await ai.generate('Create a laptop product listing', {
  schema: $.Product,
  structured: true,
})

console.log(product)
// {
//   $type: "Product",
//   $id: "generated-uuid",
//   name: "UltraBook Pro 15",
//   description: "High-performance laptop with 16GB RAM and 512GB SSD",
//   brand: {
//     $type: "Brand",
//     name: "TechCorp"
//   },
//   offers: {
//     $type: "Offer",
//     price: 1299.99,
//     priceCurrency: "USD",
//     availability: "InStock"
//   }
// }
```

## Batch Structured Generation

Generate multiple structured entities:

```typescript
import { $, ai } from 'sdk.do'

const categories = ['Electronics', 'Clothing', 'Home', 'Sports']

const products = await Promise.all(
  categories.map((category) =>
    ai.generate(`Create example product for ${category}`, {
      schema: $.Product,
      structured: true,
      context: { category },
    })
  )
)

console.log(products)
// [
//   { $type: "Product", name: "Smart Watch Pro", category: "Electronics", ... },
//   { $type: "Product", name: "Cotton T-Shirt", category: "Clothing", ... },
//   { $type: "Product", name: "Bed Sheets Set", category: "Home", ... },
//   { $type: "Product", name: "Running Shoes", category: "Sports", ... }
// ]
```

## Enrich Existing Data

Add missing fields to existing entities:

```typescript
import { $, ai, db } from 'sdk.do'

// Get product without description
const product = await db.get('product-123', $.Product)

if (!product.description) {
  // Generate description
  const enhanced = await ai.generate(`Generate product description for ${product.name}`, {
    schema: $.Product,
    structured: true,
    context: product,
  })

  // Update with generated description
  await db.update(product.$id, {
    description: enhanced.description,
  })
}
```

## Validation and Error Handling

Handle schema validation:

```typescript
import { $, ai } from 'sdk.do'

try {
  const product = await ai.generate('Create product', {
    schema: $.Product,
    structured: true,
  })

  // Validate required fields
  if (!product.name || !product.description) {
    throw new Error('Missing required product fields')
  }

  console.log('Valid product:', product)
} catch (error) {
  if (error.code === 'SCHEMA_VALIDATION_ERROR') {
    console.error('Schema validation failed:', error.details)
  } else {
    console.error('Generation failed:', error)
  }
}
```

## Available Schema Types

Common Schema.org types available:

- **Business**: Organization, LocalBusiness, Corporation
- **People**: Person, Employee, Customer
- **Products**: Product, Service, Offer
- **Content**: Article, BlogPost, HowTo, Recipe
- **Events**: Event, OnlineEvent, BusinessEvent
- **Places**: Place, PostalAddress, GeoCoordinates
- **Reviews**: Review, Rating, AggregateRating

See [schema.org.ai](https://schema.org.ai) for all 817 available types.
