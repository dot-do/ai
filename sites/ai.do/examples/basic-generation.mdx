# Basic Text Generation

Simple examples demonstrating text generation with ai.do.

## Simple Text Generation

Generate text from a prompt:

```typescript
import { ai } from 'sdk.do'

const haiku = await ai.generate('Write a haiku about programming')

console.log(haiku)
// Output:
// Code flows through the night
// Bugs emerge at dawn's first light
// Debug and make right
```

## Content Summarization

Summarize long text:

```typescript
import { ai } from 'sdk.do'

const longText = `
  Artificial intelligence (AI) is intelligence demonstrated by machines,
  in contrast to the natural intelligence displayed by humans and animals.
  Leading AI textbooks define the field as the study of "intelligent agents":
  any device that perceives its environment and takes actions that maximize
  its chance of successfully achieving its goals.
`

const summary = await ai.generate('Summarize this text in one sentence', {
  context: { text: longText },
})

console.log(summary)
// Output: "AI is machine intelligence that enables devices to perceive
// their environment and take actions to achieve goals."
```

## Question Answering

Answer questions based on context:

```typescript
import { ai } from 'sdk.do'

const context = {
  product: {
    name: 'UltraBook Pro',
    price: 1299,
    specs: {
      processor: 'Intel Core i7',
      ram: '16GB',
      storage: '512GB SSD',
    },
  },
}

const answer = await ai.generate('How much RAM does the UltraBook Pro have?', { context })

console.log(answer)
// Output: "The UltraBook Pro has 16GB of RAM."
```

## Content Categories

Classify content into categories:

```typescript
import { ai } from 'sdk.do'

const texts = [
  'Latest smartphone with AI camera',
  'Comfortable running shoes for athletes',
  'Organic coffee beans from Colombia',
  'Gaming laptop with RTX graphics',
]

for (const text of texts) {
  const category = await ai.generate('Classify into category: Electronics, Apparel, Food, or Other', { context: { text } })
  console.log(`"${text}" → ${category}`)
}

// Output:
// "Latest smartphone with AI camera" → Electronics
// "Comfortable running shoes for athletes" → Apparel
// "Organic coffee beans from Colombia" → Food
// "Gaming laptop with RTX graphics" → Electronics
```

## Sentiment Analysis

Analyze sentiment of text:

```typescript

const reviews = ['This product is amazing! Best purchase ever.', 'Okay product, nothing special.', 'Terrible quality. Would not recommend.']

for (const review of reviews) {

}

// "This product is amazing!..." → positive
// "Okay product, nothing special." → neutral
// "Terrible quality..." → negative
```

## Error Handling

Handle generation errors:

```typescript
import { ai } from 'sdk.do'

async function safeGenerate(prompt: string) {
  try {
    return await ai.generate(prompt)
  } catch (error) {
    if (error.code === 'RATE_LIMIT_EXCEEDED') {
      console.log('Rate limit exceeded, retrying in 1 second...')
      await new Promise((resolve) => setTimeout(resolve, 1000))
      return await ai.generate(prompt)
    }

    if (error.code === 'CONTENT_FILTER') {
      return 'Content filtered by safety system'
    }

    console.error('Generation failed:', error.message)
    return null
  }
}

const result = await safeGenerate('Write a story')
```
