# Basic Usage

Simple examples to get started with experiments.do.

## Example 1: Basic Operation

\`\`\`typescript
import \$, { experiments } from 'sdk.do'

const result = await experiments.run({
// Configuration
})

console.log('Result:', result)
\`\`\`

## Example 2: With Options

\`\`\`typescript
const result = await experiments.run({
// Configuration
options: {
timeout: 30000
}
})
\`\`\`
