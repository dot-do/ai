# Basic Usage Examples

Get started with Marketing AI Agent through practical, real-world examples.

## Prerequisites

```bash
pnpm add sdk.do
```

```typescript
import $, { agent } from 'sdk.do'

// Create agent
const marketingAgent = await agent.create({
  $role: $.MarketingAgent,
  model: 'claude-sonnet-4.5',
})
```

## Common Patterns

### Error Handling

```typescript
try {
  const result = await marketingAgent.execute(task)
  console.log('Success:', result)
} catch (error) {
  console.error('Error:', error.message)
}
```

### Batch Processing

```typescript
const results = await marketingAgent.batch({
  items: [...],
  parallel: 10
})
```
