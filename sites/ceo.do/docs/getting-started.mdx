# Getting Started

Build your first CEO agent with content strategy, campaign optimization, and audience insights capabilities in under 10 minutes.

## Installation

Install the SDK in your project:

```bash
pnpm add sdk.do
```

Or use the CLI:

```bash
pnpm add -g cli.do
```

## API Keys

CEO agents use LLM providers and marketing tools. Set up your API keys:

### Anthropic (Claude Models)

1. Create an account at [console.anthropic.com](https://console.anthropic.com)
2. Generate an API key from the API settings
3. Set the environment variable:

```bash
export ANTHROPIC_API_KEY=sk-ant-...
```

### OpenAI (GPT Models)

1. Create an account at [platform.openai.com](https://platform.openai.com)
2. Generate an API key from the API keys section
3. Set the environment variable:

```bash
export OPENAI_API_KEY=sk-...
```

### Marketing Tools (Optional)

Configure integrations with your marketing stack:

```bash
# Analytics
export GOOGLE_ANALYTICS_KEY=...
```

## Your First Marketing Agent

Create a simple CEO agent for content generation:

```typescript
import $, { agent } from 'sdk.do'

// Create CEO agent
const ceoAgent = await agent.create({
  $type: $.Agent,
  $role: $.ChiefExecutiveOfficer,
  name: 'Amy - Marketing Agent',
  capabilities: ['strategic_planning', 'market_analysis'],
  model: 'claude-sonnet-4.5',
  temperature: 0.7,
})

// Generate blog post
const blogPost = await ceoAgent.execute({
  task: 'Write blog post about marketing automation trends',
  context: {
    target_audience: 'Marketing managers',
    tone: 'professional_but_approachable',
    length: 1500,
    include_seo: true,
    target_keywords: ['marketing automation', 'marketing trends', 'automation tools'],
  },
})

console.log(blogPost.content)
console.log(blogPost.seo_metadata)
```

Output:


## Common Patterns

### Event-Driven Marketing

React to customer events:

```typescript
import $, { agent, on } from 'sdk.do'

const ceoAgent = await agent.create({
  $role: $.ChiefExecutiveOfficer,
  capabilities: ['email_marketing'],
})

// Trigger welcome sequence on signup
await on($.Customer.signup, async (customer) => {
  const welcomeSequence = await ceoAgent.createEmailSequence({
    trigger: 'signup',
    customer: customer,
    goal: 'activation',
    duration: '7 days',
    emails: 3,
  })

  await send($.EmailSequence.start, welcomeSequence)
})

// Re-engagement for inactive users
await on($.Customer.inactive, async (customer) => {
  const reengagement = await ceoAgent.createEmailSequence({
    trigger: 'inactive',
    customer: customer,
    goal: 'reactivation',
    duration: '14 days',
    offers: ['discount', 'feature_update', 'case_study'],
  })

  await send($.EmailSequence.start, reengagement)
})
```

### Scheduled Marketing Tasks

Automate recurring marketing activities:

```typescript
import $, { agent, every, db } from 'sdk.do'

const ceoAgent = await agent.create({
  $role: $.ChiefExecutiveOfficer,
  capabilities: ['analytics_reporting', 'strategic_planning'],
})

// Weekly performance report
await every($.Weekly, async () => {
  const report = await ceoAgent.generateReport({
    period: 'last_week',
    metrics: await db.list($.Metric, { period: 'last_week' }),
    format: 'executive_summary',
  })

  await send($.Email.send, {
    to: 'team@example.com',
    subject: 'Weekly Marketing Report',
    body: report,
  })
})

// Monthly content planning
await every($.Monthly, async () => {
  const contentPlan = await ceoAgent.planContent({
    period: 'next_month',
    topics: await db.list($.ContentPillar),
    trending: await db.get($.TrendAnalysis, { period: 'last_month' }),
  })

  await db.create($.ContentCalendar, contentPlan)
})
```

### Multi-Agent Collaboration

Combine multiple CEO agents:

```typescript
import $, { agent } from 'sdk.do'

// Specialized CEO agents
const contentAgent = await agent.create({
  $role: $.ChiefExecutiveOfficer,
  capabilities: ['strategic_planning', 'market_analysis'],
  model: 'claude-sonnet-4.5',
})

const campaignAgent = await agent.create({
  $role: $.ChiefExecutiveOfficer,
  capabilities: ['executive_decision_making'],
  model: 'claude-sonnet-4.5',
})

const analyticsAgent = await agent.create({
  $role: $.ChiefExecutiveOfficer,
  capabilities: ['analytics_reporting', 'team_leadership'],
  model: 'claude-sonnet-4.5',
})

// Collaborative workflow
const campaign = await campaignAgent.plan({
  name: 'Product Launch Q1',
  budget: '$100,000',
})

const content = await contentAgent.create({
  campaign: campaign,
  content_types: ['blog', 'email', 'social', 'landing_pages'],
})

const targeting = await analyticsAgent.segment({
  campaign: campaign,
  audience: await db.list($.Customer),
})

// Execute campaign with all components
await campaignAgent.launch({
  campaign: campaign,
  content: content,
  targeting: targeting,
})
```

## Error Handling

Implement robust error handling:

```typescript
import $, { agent } from 'sdk.do'

const ceoAgent = await agent.create({
  $role: $.ChiefExecutiveOfficer,
  model: 'claude-sonnet-4.5',
})

try {
  const campaign = await ceoAgent.launch({
    name: 'Q1 Campaign',
    budget: '$50,000',
  })

  console.log('Campaign launched:', campaign.$id)
} catch (error) {
  if (error.code === 'BUDGET_EXCEEDED') {
    console.error('Budget validation failed:', error.message)
  } else if (error.code === 'API_ERROR') {
    console.error('External API error:', error.message)
    // Retry logic
  } else if (error.code === 'VALIDATION_ERROR') {
    console.error('Invalid input:', error.details)
  } else {
    console.error('Unexpected error:', error)
  }
}
```

### Common Questions

**Q: Which AI model should I use?**
A: Use `claude-sonnet-4.5` for creative content and strategic thinking, or `gpt-5` for analytical tasks. Both are excellent for CEO agents.

**Q: How do I handle rate limits?**
A: Configure retry logic and use batching for large campaigns. The SDK automatically handles most rate limiting.

**Q: Can I customize the agent's voice and tone?**
A: Yes, use the `persona` configuration to define background, style, and approach. The agent adapts its output accordingly.

**Q: How do I integrate with my existing marketing tools?**
A: Use the `api.connect()` method to integrate with Salesforce, Board Portal, Business Intelligence, and other tools. See [Integration](../examples/integration) for examples.

**Q: Is my data secure?**
A: Yes, all data is encrypted in transit and at rest. API keys are never logged. See our [security documentation](https://sdk.do/security) for details.
