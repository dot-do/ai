# cli.do

**Command-line interface to SDK.do functions with semantic patterns**

cli.do provides a powerful command-line interface to execute all SDK.do functions directly from your terminal. Use semantic patterns like `do db list Business` or `do ai generate "..."` to interact with your Business-as-Code platform.

## Quick Start

```bash
# Install globally
pnpm add -g cli.do

# Or use with npx
npx cli.do db list Business
```

```bash
# List all businesses
do db list Business

# Generate content with AI
do ai generate "Write a haiku about code"

# Send semantic events
do send Order.created '{"id": "123", "total": 99.99}'

# Schedule workflows
do every Daily "backup database"

# Query relationships
do db related Business:acme owns Brand
```

## Features

- **Semantic Commands** - Use `$.Subject.predicate.Object` patterns in CLI
- **Full SDK Access** - Execute any SDK.do function from terminal
- **Type-Safe** - TypeScript-powered with intelligent autocomplete
- **JSON/YAML Support** - Input and output in multiple formats
- **Interactive Mode** - REPL-style interface for exploration
- **Script-Friendly** - Perfect for automation and CI/CD pipelines
- **Watch Mode** - Monitor events and databases in real-time
- **Config Management** - Store credentials and preferences securely

## Core Commands

### Database Operations

```bash
# List entities by type
do db list Business
do db list Person --where '{"jobTitle": "Engineer"}'

# Get entity by ID
do db get Business acme-corp
do db get Person john-doe

# Create entity
do db create Blog '{"$id": "engineering-blog", "name": "Engineering Blog"}'

# Update entity
do db update Person john-doe '{"jobTitle": "Senior Engineer"}'

# Delete entity
do db delete Person john-doe

# Query relationships
do db related Business:acme owns Brand
do db related Organization:acme employs Person
```

### AI Operations

```bash
# Generate text
do ai generate "Write a haiku about coding"

# Generate with model selection
do ai generate "Explain quantum computing" --model claude-sonnet-4.5

# Generate with schema
do ai generate "Create a blog post" --schema BlogPosting

# Generate structured data
do ai generate "Create a product" --schema Product --format json

# Create embeddings
do ai embed "Hello world"

# Batch processing
do ai batch --file prompts.json
```

### Event Operations

```bash
# Send events
do send Order.created '{"id": "123", "total": 99.99}'
do send Invoice.paid '{"id": "456", "amount": 99.99}'

# Listen for events
do on Order.created
do on Invoice.paid --handler ./handler.js

# Pattern matching
do on "*.created"
```

### Scheduled Workflows

```bash
# Schedule with semantic intervals
do every Daily "backup database"
do every Hourly "collect metrics"
do every Weekly "send newsletter"

# Custom cron expressions
do every "0 0 * * *" "full backup"

# List scheduled tasks
do every list

# Remove scheduled task
do every remove backup-database
```

### User & Session

```bash
# Get current user
do user current

# Get session info
do user session

# Check permissions
do user can edit Blog
do user can delete Blog engineering-blog
```

## Semantic Patterns

cli.do supports semantic patterns throughout:

```bash
# Database queries with semantic types
do db list $.Business
do db get $.Person john-doe
do db related $.Organization.owns.Brand

# Event patterns
do send $.Order.created '{"id": "123"}'
do on $.Invoice.paid

# Schedule patterns
do every $.Daily "task"
do every $.Hourly "job"
```

## Integration with SDK.do

cli.do is a command-line interface to [sdk.do](https://sdk.do). Every CLI command maps to an SDK function:

| CLI Command             | SDK Function                           |
| ----------------------- | -------------------------------------- |
| `do db list Business`   | `await db.list($.Business)`            |
| `do db get Person id`   | `await db.get($.Person, 'id')`         |
| `do ai generate "..."`  | `await ai.generate({ prompt: '...' })` |
| `do send Order.created` | `await send($.Order.created, data)`    |
| `do on Invoice.paid`    | `on($.Invoice.paid, handler)`          |
| `do every Daily`        | `every($.Daily, handler)`              |


```bash
# Table output

# Pretty output

# Raw output (for piping)
do db list Business --format raw
```

## Scripting

```bash
# Generate backup report
report=$(do ai generate "Summarize: $businesses" --format json)

do send Backup.completed "{\"timestamp\": \"$(date -u +%Y-%m-%dT%H:%M:%SZ)\"}"
```

## Use Cases

### DevOps & Automation

```bash
# CI/CD integration
do db create Deployment "{\"version\": \"$VERSION\", \"status\": \"pending\"}"
do send Deployment.started "{\"version\": \"$VERSION\"}"

# Health checks
do db list Service --where '{"status": "unhealthy"}'

# Monitoring
do on Service.unhealthy --exec "./alert.sh"
```

### Data Management

```bash
# Bulk operations
cat entities.json | do db create Business --batch

# Data migration
do db list Person --format json | do db create Contact --batch

# Data export
do db list Business > businesses.json
```

### AI Workflows

```bash
# Content generation
do ai generate "Write blog post about $TOPIC" --schema BlogPosting > post.json

# Batch content generation
cat topics.txt | xargs -I {} do ai generate "Write about {}" --schema Article

# Content enrichment
do db list Product | jq -c '.[]' | xargs -I {} do ai generate "Enhance: {}" --schema Product
```
