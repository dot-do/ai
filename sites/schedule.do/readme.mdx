# schedule.do

**Schedule tasks, jobs, and workflows with cron expressions and intervals**

schedule.do provides powerful utilities for task scheduling in your Business-as-Code applications. Built with semantic patterns, type safety, and integration with the entire `.do` platform ecosystem.

## Quick Start

```bash
pnpm add schedule.do
```

```typescript
import $, { schedule } from 'schedule.do'

// Basic usage
const result = await schedule({
  // Your data
  data: 'example',
  // Options
  options: {},
})

console.log('Result:', result)
```

## Key Features

### Semantic Patterns

Use `$.Subject.predicate.Object` patterns for type-safe operations:

```typescript
import $ from 'schedule.do'

// Semantic patterns
const result = await schedule($.Data, {
  // Configuration
})
```

### Type Safety

Full TypeScript support with comprehensive type definitions:

```typescript
import type { ScheduleOptions, ScheduleResult } from 'schedule.do'

const options: ScheduleOptions = {
  // Type-safe configuration
}

const result: ScheduleResult = await schedule(options)
```

### Integration Ready

Seamlessly integrates with other `.do` platform services:

```typescript
import { schedule } from 'schedule.do'
import { db, ai } from 'sdk.do'

// Integrate with database
const data = await db.list($.Data)
const result = await schedule(data)

// Integrate with AI
const processed = await ai.generate('process', { data: result })
```

## Best Practices

### Use Semantic Patterns

```typescript
// Good - semantic patterns
const result = await schedule($.Data, options)

// Avoid - string-based
const result = await schedule('Data', options)
```

### Handle Errors

```typescript
// Good - error handling
try {
  const result = await schedule(data)
  return result
} catch (error) {
  console.error('Failed:', error)
  throw error
}
```

### Use TypeScript

```typescript
// Good - type-safe
import type { Options } from 'schedule.do'
const options: Options = {}

// Avoid - untyped
const options = {}
```
