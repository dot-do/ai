# Best Practices

## Content Structure

Use semantic types:

```typescript
const article = await cms.create($.Article, {
  headline: 'Title',
  articleBody: 'Content...',
  author: { $type: 'Person', name: 'Author' },
})
```

## Media Optimization

```typescript
await cms.media.upload(file, {
  optimize: true,
  formats: ['webp', 'jpg'],
  generateThumbnails: true,
})
```
