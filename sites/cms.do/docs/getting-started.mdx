# Getting Started with cms.do

## Installation

```bash
pnpm add cms.do
```

## Configuration

```bash
CMS_DATABASE_URL=postgres://...
CMS_MEDIA_STORAGE=s3://bucket
CMS_WORKFLOW_ENABLED=true
```

## First Content

```typescript
import { cms } from 'cms.do'

const article = await cms.create($.Article, {
  title: 'My First Article',
  body: 'Content here...',
  status: 'draft',
})

await cms.publish(article)
```
