# Real-World Use Case

## Blog Platform

```typescript
import { cms } from 'cms.do'

class BlogCMS {
  async initialize() {
    await this.setupWorkflows()
    await this.configureChannels()
  }

  async setupWorkflows() {
    await cms.workflows.create({
      name: 'Blog Publishing',
      stages: [{ name: 'draft' }, { name: 'review' }, { name: 'published' }],
    })
  }

  async publishPost(post) {
    await cms.publish(post, {
      channels: ['web', 'rss', 'social'],
    })
  }
}
```
