# Advanced Patterns

## Custom Workflows

```typescript
const workflow = await cms.workflows.create({
  name: 'Custom Publishing',
  stages: [
    { name: 'draft', next: ['review'] },
    { name: 'review', next: ['published', 'draft'] },
    { name: 'published', final: true },
  ],
})
```

## Content Scheduling

```typescript
await cms.schedule(article, {
  publishAt: new Date('2025-02-01T09:00:00Z'),
  unpublishAt: new Date('2025-02-28T23:59:59Z'),
})
```
