# Integration

## API Integration

```typescript
import express from 'express'
import { cms } from 'cms.do'

const app = express()

app.get('/api/articles', async (req, res) => {
  const articles = await cms.list($.Article, {
    where: { status: 'published' },
  })
  res.json({ articles })
})
```
