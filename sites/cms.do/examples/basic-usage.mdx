# Basic Usage

## Creating Content

```typescript
import { cms } from 'cms.do'

const article = await cms.create($.Article, {
  title: 'New Article',
  body: 'Content here...',
  author: user.$id,
  status: 'draft',
})
```

## Publishing

```typescript
await cms.publish(article, {
  channels: ['web', 'api'],
})
```
