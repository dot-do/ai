# sandbox.do

**Secure isolated code execution environment for running untrusted code safely**

sandbox.do provides powerful utilities for code sandbox in your Business-as-Code applications. Built with semantic patterns, type safety, and integration with the entire `.do` platform ecosystem.

## Quick Start

```bash
pnpm add sandbox.do
```

```typescript
import $, { sandbox } from 'sandbox.do'

// Basic usage
const result = await sandbox({
  // Your data
  data: 'example',
  // Options
  options: {},
})

console.log('Result:', result)
```

## Key Features

### Semantic Patterns

Use `$.Subject.predicate.Object` patterns for type-safe operations:

```typescript
import $ from 'sandbox.do'

// Semantic patterns
const result = await sandbox($.Data, {
  // Configuration
})
```

### Type Safety

Full TypeScript support with comprehensive type definitions:

```typescript
import type { SandboxOptions, SandboxResult } from 'sandbox.do'

const options: SandboxOptions = {
  // Type-safe configuration
}

const result: SandboxResult = await sandbox(options)
```

### Integration Ready

Seamlessly integrates with other `.do` platform services:

```typescript
import { sandbox } from 'sandbox.do'
import { db, ai } from 'sdk.do'

// Integrate with database
const data = await db.list($.Data)
const result = await sandbox(data)

// Integrate with AI
const processed = await ai.generate('process', { data: result })
```

## TypeScript Support

sandbox.do is built with TypeScript and provides full type safety:

```typescript
import type { SandboxOptions, SandboxResult, SandboxError } from 'sandbox.do'

// Type-safe operations
const options: SandboxOptions = {
  // Configuration
}

try {
  const result: SandboxResult = await sandbox(options)
  console.log('Success:', result)
} catch (error: SandboxError) {
  console.error('Error:', error.message)
}
```

## Best Practices

### Use Semantic Patterns

```typescript
// Good - semantic patterns
const result = await sandbox($.Data, options)

// Avoid - string-based
const result = await sandbox('Data', options)
```

### Handle Errors

```typescript
// Good - error handling
try {
  const result = await sandbox(data)
  return result
} catch (error) {
  console.error('Failed:', error)
  throw error
}
```

### Use TypeScript

```typescript
// Good - type-safe
import type { Options } from 'sandbox.do'
const options: Options = {}

// Avoid - untyped
const options = {}
```
