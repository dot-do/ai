# CTO Agent API Reference

Complete API documentation for CTO agents.

## Agent Creation

### `agent.create(config)`

Create a new CTO agent instance.

```typescript
const cto = await agent.create({
  $type: 'Agent',
  $role: $.ChiefTechnologyOfficer,
  name: string,
  persona?: AgentPersona,
  capabilities: string[],
  tools: Record<string, Tool>,
  model: string,
  temperature?: number
})
```

**Parameters:**

- `name` (string): Agent name
- `persona` (AgentPersona): Optional persona definition
- `capabilities` (string[]): List of agent capabilities
- `tools` (Record): Tools the agent can use
- `model` (string): AI model (e.g., 'claude-sonnet-4.5')
- `temperature` (number): 0-1, default 0.7

**Returns:** CTO Agent instance

## Execution

### `cto.execute(task)`

Execute a CTO task.

```typescript
const result = await cto.execute({
  task: string,
  context?: Context,
  model?: string,
  temperature?: number
})
```

**Parameters:**

- `task` (string): Task description
- `context` (Context): Optional execution context
- `model` (string): Override default model
- `temperature` (number): Override default temperature

**Returns:** ExecutionResult

## Configuration

### `cto.configure(config)`

Configure agent behavior.

```typescript
await cto.configure({
  model?: string,
  temperature?: number,
  maxTokens?: number,
  autonomy?: 'autonomous' | 'supervised' | 'manual',
  decision_framework?: DecisionFramework,
  logging?: LoggingConfig
})
```

## Monitoring

### `cto.getMetrics(options)`

Get agent performance metrics.

```typescript
const metrics = await cto.getMetrics({
  timeframe?: string,
  categories?: string[]
})
```

**Returns:**

```typescript
{
  decisions_made: number,
  avg_confidence: number,
  success_rate: number,
  categories: Record<string, number>
}
```

## Events

### `cto.on(event, handler)`

Listen to agent events.

```typescript
cto.on('decision_made', (event) => {
  console.log(event.decision, event.confidence)
})

cto.on('approval_required', async (decision) => {
  const approved = await requestApproval(decision)
  return approved
})
```

**Events:**

- `decision_made`: When agent makes a decision
- `approval_required`: When agent needs approval
- `error`: When an error occurs
- `task_start`: When task execution starts
- `task_complete`: When task completes
