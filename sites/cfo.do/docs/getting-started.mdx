# Getting Started with CFO Agents

Build your first CFO agent with financial planning & analysis, forecasting, budgeting, and reporting capabilities in under 10 minutes.

## Installation

Install the SDK in your project:

```bash
pnpm add sdk.do
```

Or use the CLI globally:

```bash
pnpm add -g cli.do
```

## API Keys

CFO agents use AI models and integrate with financial systems. Set up your API keys:

### AI Models

**OpenAI (GPT-5 - Recommended)**

1. Create account at [platform.openai.com](https://platform.openai.com)
2. Generate API key from API keys section
3. Set environment variable:

```bash
export OPENAI_API_KEY=sk-...
```

**Anthropic (Claude Sonnet 4.5)**

1. Create account at [console.anthropic.com](https://console.anthropic.com)
2. Generate API key from API settings
3. Set environment variable:

```bash
export ANTHROPIC_API_KEY=sk-ant-...
```

## Your First CFO Agent

Create a simple CFO agent for financial analysis:

```typescript
import $, { cfo } from 'sdk.do'

// Create CFO agent
const jordan = await cfo.create({
  $type: $.ChiefFinancialOfficer,
  name: 'Jordan Taylor',
  persona: {
    background: 'CPA, MBA Finance, 15 years in SaaS finance leadership',
    style: 'analytical, strategic, risk-aware, data-driven',
    expertise: ['financial_planning', 'forecasting', 'investor_relations'],
  },
  capabilities: ['financial_planning', 'forecasting', 'reporting'],
  model: 'gpt-5',
  temperature: 0.3, // Lower temperature for financial accuracy
})

// Analyze financial performance
const analysis = await jordan.analyzePerformance({
  period: 'Q3 2025',
  metrics: ['revenue', 'gross_margin', 'burn_rate', 'runway'],
})

console.log(analysis.summary)
console.log(analysis.insights)
console.log(analysis.recommendations)
```

## Common Patterns

### Event-Driven Finance

React to financial events:

```typescript
import $, { cfo, on, send } from 'sdk.do'

const jordan = await cfo.create({
  $type: $.ChiefFinancialOfficer,
  capabilities: ['financial_reporting', 'cash_management'],
  model: 'gpt-5',
})

// Alert on significant transactions
await on($.Transaction.recorded, async (transaction) => {
  if (transaction.amount > 100000) {
    const analysis = await jordan.analyzeTransaction(transaction)
    await send($.Alert.finance, {
      transaction: transaction,
      analysis: analysis,
      requires_approval: transaction.amount > 500000,
    })
  }
})

// Automated monthly close
await on($.Month.ended, async (month) => {
  const closePackage = await jordan.performMonthEndClose({
    period: month.period,
    reconcile: true,
    generate_reports: true,
    distribute: true,
  })

  await db.create($.FinancialClose, closePackage)
})
```

### Scheduled Financial Tasks

Automate recurring financial activities:

```typescript
import $, { cfo, every, db } from 'sdk.do'

const jordan = await cfo.create({
  $type: $.ChiefFinancialOfficer,
  capabilities: ['financial_reporting', 'kpi_tracking'],
  model: 'gpt-5',
})

// Daily cash monitoring
await every($.Daily, async () => {
  const cash = await jordan.checkCashPosition()
  await db.create($.CashSnapshot, { date: new Date(), data: cash })

  if (cash.alerts.length > 0) {
    await send($.Alert.finance, cash.alerts)
  }
})

// Weekly financial summary
await every($.Weekly, async () => {
  const summary = await jordan.generateWeeklySummary({
    metrics: ['revenue', 'burn', 'cash', 'pipeline'],
    format: 'email',
  })

  await send($.Email.send, {
    to: 'leadership@company.com',
    subject: 'Weekly Financial Summary',
    body: summary,
  })
})

// Monthly financial package
await every($.Monthly, async () => {
  const package = await jordan.generateMonthlyPackage({
    period: 'current_month',
    distribute: true,
  })

  await db.create($.FinancialPackage, package)
})

// Quarterly board deck
await every($.Quarterly, async () => {
  const boardDeck = await jordan.createBoardDeck({
    quarter: 'current_quarter',
    style: 'comprehensive',
  })

  await send($.Email.send, {
    to: 'board@company.com',
    subject: 'Q${quarter} Board Deck',
    attachments: [boardDeck.pdf],
  })
})
```

## Error Handling

Implement robust error handling:

```typescript
import $, { cfo } from 'sdk.do'

const jordan = await cfo.create({
  $type: $.ChiefFinancialOfficer,
  model: 'gpt-5',
})

try {
  const forecast = await jordan.createForecast({
    horizon: 18,
    methodology: 'driver_based',
  })

  console.log('Forecast created:', forecast.$id)
} catch (error) {
  if (error.code === 'INSUFFICIENT_DATA') {
    console.error('Need more historical data:', error.message)
    // Request manual data input
  } else if (error.code === 'VALIDATION_ERROR') {
    console.error('Invalid assumptions:', error.details)
    // Review and correct assumptions
  } else if (error.code === 'MODEL_ERROR') {
    console.error('AI model error:', error.message)
    // Retry with different model or parameters
  } else if (error.code === 'API_ERROR') {
    console.error('External system error:', error.message)
    // Check integration configuration
  } else {
    console.error('Unexpected error:', error)
    // Log for investigation
  }
}
```

### Common Questions

**Q: Which AI model should I use for CFO agents?**
A: Use `gpt-5` for most financial tasks. It provides excellent analytical capabilities and accuracy. Use `claude-sonnet-4.5` for narrative reports and board communications.

**Q: How accurate are the financial forecasts?**
A: Accuracy depends on data quality and historical patterns. CFO agents typically achieve 85-95% accuracy for 3-month forecasts, 70-85% for 12-month forecasts. Always review and validate forecasts.

**Q: Can CFO agents handle multi-currency operations?**
A: Yes, CFO agents support multi-currency operations with automatic FX conversion and hedging analysis.

**Q: How do I ensure compliance?**
A: Enable compliance features in the agent configuration and implement proper audit trails, access controls, and data encryption.

**Q: Can I customize financial reports?**
A: Yes, use custom templates and specify exact metrics, formats, and styling for all reports.
