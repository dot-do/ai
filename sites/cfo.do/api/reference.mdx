# API Reference

Complete API documentation for cfo.do.

## Functions

### create()

Create a cfo agent instance.

```typescript
function create(config: AgentConfig): Promise<Agent>
```

**Parameters:**

- `config.name` (string) - Agent name
- `config.role` (string) - Agent role
- `config.model` (string) - AI model

**Returns:** Agent instance
