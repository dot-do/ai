# Basic Usage

Basic examples for worker fleet management.

## Example 1: Create Fleet

```typescript
import { $, workers } from 'sdk.do'

const fleet = await workers.createFleet({
  name: 'Sales Development Fleet',
  workerTemplate: {
    role: $.SalesDevelopmentRepresentative,
    model: 'claude-sonnet-4.5',
  },
  scaling: {
    min: 5,
    max: 50,
  },
})
```

## Example 2: Manual Scaling

```typescript
// Scale up
await fleet.scale({ count: 20 })

// Scale down
await fleet.scale({ count: 10 })

// Get current size
const size = await fleet.getSize()
console.log(`Fleet size: ${size}`)
```

## Example 3: Distribute Work

```typescript
await fleet.assignWork({
  tasks: leadList,
  distribution: 'load_balanced',
})
```

## Example 4: Monitor Fleet

```typescript
const metrics = await fleet.getMetrics()

console.log(`Active workers: ${metrics.activeWorkers}`)
console.log(`Queue depth: ${metrics.queueDepth}`)
console.log(`Throughput: ${metrics.tasksPerHour}/hour`)
```

## Example 5: Auto-Scaling

```typescript
await fleet.enableAutoScaling({
  metric: 'queue_depth',
  scaleUp: { threshold: 150, increment: 5 },
  scaleDown: { threshold: 50, decrement: 2 },
  cooldown: 300, // seconds
})
```
