# API Reference

Complete API reference for worker fleet management.

## workers.createFleet()

Create a new worker fleet.

```typescript
workers.createFleet(options: FleetOptions): Promise<WorkerFleet>
```

**Parameters:**

- `options.name` (string): Fleet name
- `options.workerTemplate` (object): Template for creating workers
- `options.scaling` (object): Scaling configuration

## fleet.scale()

Manually scale fleet size.

```typescript
fleet.scale(options: ScaleOptions): Promise<void>
```

**Parameters:**

- `options.count` (number): Target fleet size
- `options.delta` (number): Relative change (+/-)

## fleet.assignWork()

Assign work to fleet.

```typescript
fleet.assignWork(options: WorkOptions): Promise<void>
```

**Parameters:**

- `options.tasks` (Task[]): Tasks to assign
- `options.distribution` (string): Distribution strategy
- `options.filters` (object): Worker filters

## fleet.enableAutoScaling()

Enable automatic scaling.

```typescript
fleet.enableAutoScaling(options: AutoScaleOptions): Promise<void>
```

**Parameters:**

- `options.metric` (string): Scaling metric
- `options.scaleUp` (object): Scale-up configuration
- `options.scaleDown` (object): Scale-down configuration
- `options.cooldown` (number): Cooldown period in seconds

## fleet.getMetrics()

Get fleet performance metrics.

```typescript
fleet.getMetrics(): Promise<FleetMetrics>
```

**Returns:**

- `activeWorkers` (number): Currently active workers
- `queueDepth` (number): Pending tasks
- `tasksPerHour` (number): Throughput
- `avgCompletionTime` (number): Average task completion time
