# Getting Started

Learn how to create your first individual worker in minutes.

## Installation

```bash
pnpm add sdk.do
```

## Your First worker.do

```typescript
import { worker } from 'sdk.do'

const myWorker = await worker.create({
  name: 'Example Worker',
  role: 'individual worker',
  model: 'claude-sonnet-4.5',
})

const result = await myWorker.work({
  task: 'example_task',
})
```

## Core Concepts

### Identity

Workers have names, roles, and personas.

### Goals

Workers work toward measurable objectives.

### Autonomy

Workers make decisions within their domain.
