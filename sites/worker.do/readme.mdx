# worker.do

**Individual Worker Agent Framework**

worker.do enables you to create, configure, and manage individual worker agents - autonomous digital workers with specific roles, capabilities, and responsibilities. Unlike general bots or multi-agent systems, worker.do focuses on the lifecycle and management of single worker instances.

## What is a Worker Agent?

A worker agent is an autonomous digital entity that:

- **Has Identity**: Unique persona, role, and responsibilities
- **Performs Work**: Executes tasks within their domain of expertise
- **Learns**: Improves performance through experience
- **Reports**: Provides visibility into work performed
- **Collaborates**: Works with other workers when needed

## Quick Start

```typescript
import $, { worker } from 'sdk.do'

// Create a worker agent
const salesWorker = await worker.create({
  $type: 'Worker',
  name: 'Sarah Chen',
  role: $.SalesDevelopmentRepresentative,
  persona: {
    background: 'Former tech recruiter, 5 years in B2B SaaS',
    expertise: ['lead_qualification', 'outreach', 'discovery_calls'],
    style: 'consultative, data-driven',
  },
  capabilities: ['research', 'outreach', 'qualification'],
  goals: {
    daily: ['Qualify 4 leads', 'Book 2 meetings'],
    weekly: ['Qualify 20 leads', 'Book 10 meetings'],
    monthly: ['100 qualified leads', '50 booked meetings'],
  },
  model: 'claude-sonnet-4.5',
})

// Assign work
const result = await salesWorker.work({
  task: 'qualify_lead',
  lead: {
    name: 'John Doe',
    company: 'Acme Corp',
    email: 'john@acme.com',
  },
})
```

## Key Features

### Worker Identity

Define rich worker personas:

```typescript
const worker = await worker.create({
  $type: 'Worker',
  name: 'Alex Rodriguez',
  role: $.CustomerSuccessManager,
  persona: {
    $type: 'Person',
    name: 'Alex Rodriguez',
    jobTitle: 'Customer Success Manager',
    background: 'Psychology degree, 7 years in SaaS customer success',
    expertise: ['customer_onboarding', 'relationship_building', 'churn_prevention', 'upsell_identification'],
    personalityTraits: ['empathetic', 'proactive', 'detail-oriented', 'problem-solver'],
    communicationStyle: 'warm, professional, solution-focused',
    workingHours: '9am-5pm PST',
    languages: ['English', 'Spanish'],
  },
  skills: [
    { name: 'Customer Onboarding', level: 'expert' },
    { name: 'Product Knowledge', level: 'advanced' },
    { name: 'Data Analysis', level: 'intermediate' },
  ],
})
```

### Work Assignment

Assign tasks to workers:

```typescript
// Assign single task
const task = await worker.assign({
  $type: 'WorkAction',
  action: 'onboard_customer',
  priority: 'high',
  dueDate: new Date('2025-10-15'),
  customer: {
    name: 'Acme Corp',
    contact: 'jane@acme.com',
    plan: 'enterprise',
  },
})

// Assign recurring work
await worker.assignRecurring({
  action: 'check_health_scores',
  schedule: '$.Daily',
  time: '09:00',
})

// Assign project
await worker.assignProject({
  name: 'Q4 Customer Success Initiative',
  tasks: [
    { action: 'review_at_risk_accounts', dueDate: '2025-10-20' },
    { action: 'create_success_plans', dueDate: '2025-10-25' },
    { action: 'schedule_qbrs', dueDate: '2025-10-30' },
  ],
})
```

### Worker Lifecycle

Manage worker lifecycle:

```typescript
// Hire (create) worker
const worker = await worker.hire({
  role: $.DataAnalyst,
  name: 'Morgan Lee',
  startDate: new Date(),
})

// Onboard worker
await worker.onboard({
  training: ['product_knowledge', 'tools_training', 'process_training'],
  access: ['database', 'bi_tools', 'slack'],
  mentor: 'senior-analyst-id',
})

// Activate worker
await worker.activate()

// Monitor performance
const performance = await worker.getPerformance({
  period: 'last_30_days',
})

// Provide feedback
await worker.provideFeedback({
  type: 'performance_review',
  rating: 4.5,
  strengths: ['Data visualization', 'Attention to detail'],
  improvements: ['Speed up report generation'],
  goals: ['Learn advanced SQL', 'Master Python pandas'],
})

// Promote/Update role
await worker.updateRole({
  newRole: $.SeniorDataAnalyst,
  effectiveDate: new Date(),
})

// Offboard worker
await worker.offboard({
  reason: 'role_change',
  transition: {
    transferWorkTo: 'other-worker-id',
    documentationComplete: true,
  },
})
```

### Worker Communication

Communicate with workers:

```typescript
// Send message to worker
await worker.message({
  from: 'manager-id',
  subject: 'New Priority Task',
  body: 'Please prioritize the Acme Corp analysis',
  priority: 'high',
})

// Get worker status
const status = await worker.getStatus()
// {
//   state: 'working' | 'available' | 'busy' | 'offline',
//   currentTask: {...},
//   availability: '2 hours',
//   location: 'remote'
// }

// Request work update
const update = await worker.requestUpdate({
  taskId: 'task-123',
})

// Schedule meeting with worker
await worker.scheduleMeeting({
  type: 'one_on_one',
  duration: 30,
  agenda: ['Goal progress', 'Blockers', 'Feedback'],
})
```

## Integration with .do Platform

```typescript
import $, { worker, db, on, send, every } from 'sdk.do'

// Worker with database access
const worker = await worker.create({
  name: 'Data Worker',
  tools: { database: db },
})

// Worker responds to events
await on($.Lead.created, async (lead) => {
  await sdrWorker.work({
    task: 'qualify_lead',
    lead,
  })
})

// Worker publishes events
await worker.onTaskComplete(async (task, result) => {
  await send($.Task.completed, {
    workerId: worker.id,
    task,
    result,
  })
})

// Worker scheduled work
await every($.Daily, async () => {
  await worker.work({
    task: 'daily_standup_report',
  })
})
```
