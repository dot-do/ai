# Basic Worker Usage

Simple examples for creating and using individual workers.

## Sales Development Representative

\`\`\`typescript
import { worker } from 'sdk.do'

const sarah = await worker.create({
name: 'Sarah Chen',
role: 'SDR',
skills: ['prospecting', 'outreach', 'qualification'],
goals: [
{ metric: 'qualified_leads', target: 20, period: '$.Weekly' }
]
})

await sarah.work({ task: 'qualify_leads' })
\`\`\`
