# Basic Usage

Simple examples to get started with benchmarks.do.

## Example 1: Basic Operation

\`\`\`typescript
import \$, { benchmarks } from 'sdk.do'

const result = await benchmarks.run({
// Configuration
})

console.log('Result:', result)
\`\`\`

## Example 2: With Options

\`\`\`typescript
const result = await benchmarks.run({
// Configuration
options: {
timeout: 30000
}
})
\`\`\`
