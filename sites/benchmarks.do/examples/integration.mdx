# Integration

Integrate benchmarks.do with the .do platform.

## Database Integration

\`\`\`typescript
import \$, { benchmarks, db } from 'sdk.do'

const data = await db.list($.Data)
const result = await benchmarks.run({ data })
\`\`\`

## Event Integration

\`\`\`typescript
import \$, { benchmarks, on } from 'sdk.do'

on($.Event.occurred, async (event) => {
await benchmarks.run({ event })
})
\`\`\`
