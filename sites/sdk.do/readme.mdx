# sdk.do

**TypeScript SDK for building autonomous Business-as-Code with the .do platform**

The `sdk.do` package provides a comprehensive, type-safe TypeScript SDK for building autonomous business applications using semantic patterns and AI-native primitives. It enables developers to write code that understands business intent and workflows through the `$` semantic proxy.

## Installation

\`\`\`bash
npm install sdk.do

# or

pnpm add sdk.do

# or

yarn add sdk.do
\`\`\`

## Quick Start

\`\`\`typescript
import { $ } from 'sdk.do'

// Database operations
const user = await $.db.get('users', 'user_123')
await $.db.create('orders', {
userId: user.id,
total: 99.99,
items: [...]
})

// AI generation
const content = await $.ai.generate({
prompt: 'Write a product description for...',
model: 'gpt-5'
})

// Event publishing
await $.send('$.Order.created', {
orderId: 'order_456',
userId: 'user_123'
})

// Event listeners
$.on('$.User.created', async (event) => {
await $.ai.generate({
prompt: `Welcome email for ${event.payload.name}`
})
})

// Scheduled tasks
$.every('0 9 \* \* \*', async () => {
// Runs daily at 9am
await $.workflows.start('daily-report')
})
\`\`\`

## Core Services

### Database (\`$.db\`)

Complete CRUD operations with relationships and transactions.

### AI Services (\`$.ai\`)

Multi-provider AI access with structured outputs.

### Event Publishing (\`$.send\`)

Publish events to semantic queues.

### Event Listeners (\`$.on\`)

Subscribe to events with pattern matching.

### Scheduled Tasks (\`$.every\`)

Cron-based and interval scheduling.

See full documentation at https://docs.do/sdk
