# Integration Examples

Examples of integrating SDR agents with other services.

## CRM Integration

```typescript
import { $, sdr, db } from 'sdk.do'

// Sync leads from CRM
on($.Lead.created, async (event) => {
  await sarah.assignLeads({ leads: [event.lead] })
})

// Update CRM with qualification results
on($.Lead.qualified, async (event) => {
  await db.update(event.leadId, {
    status: 'qualified',
    qualificationScore: event.score,
  })
})
```

## Calendar Integration

```typescript
const meeting = await sarah.bookMeeting({
  leadId: 'lead-123',
  type: 'discovery_call',
  duration: 30,
  calendar: {
    provider: 'google',
    calendarId: 'sales@company.com',
  },
})
```

## Email Automation

```typescript
import { $, sdr, send } from 'sdk.do'

await sarah.sendEmail({
  to: 'john@acme.com',
  subject: 'Quick question about {{company}}',
  template: 'cold_outreach',
  variables: { company: 'Acme Corp' },
})
```
