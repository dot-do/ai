# Advanced Patterns

Advanced patterns for SDR agents.

## Multi-Channel Outreach

```typescript
import { $, sdr } from 'sdk.do'

const sequence = await sarah.createOutreachSequence({
  leadId: 'lead-123',
  channels: [
    { day: 1, channel: 'email', template: 'intro' },
    { day: 3, channel: 'linkedin', template: 'connection' },
    { day: 5, channel: 'email', template: 'value_prop' },
    { day: 7, channel: 'phone', template: 'cold_call' },
    { day: 10, channel: 'email', template: 'case_study' },
  ],
})
```

## Automated Lead Scoring

```typescript
import { $, sdr, llm } from 'sdk.do'

const score = await llm.generate(`Score this lead based on fit and intent`, {
  schema: $.LeadScore,
  context: leadData,
  model: 'gpt-5',
})

if (score.value > 75) {
  await sarah.prioritizeLead(leadId)
}
```

## Territory Management

```typescript
const territories = await sdr.createTerritories({
  rules: [
    { name: 'Enterprise', criteria: { companySize: { min: 1000 } }, assignTo: 'sarah' },
    { name: 'Mid-Market', criteria: { companySize: { min: 100, max: 999 } }, assignTo: 'mike' },
    { name: 'SMB', criteria: { companySize: { max: 99 } }, assignTo: 'alex' },
  ],
})
```
