# Basic Usage

Basic examples for working with SDR agents.

## Example 1: Create SDR Agent

```typescript
import { $, sdr } from 'sdk.do'

const sarah = await sdr.create({
  name: 'Sarah Chen',
  territory: {
    industries: ['Technology', 'SaaS'],
    companySize: { min: 50, max: 5000 },
  },
  quota: {
    leadsQualified: 100,
    meetingsBooked: 50,
  },
  model: 'claude-sonnet-4.5',
})
```

## Example 2: Research Leads

```typescript
const research = await sarah.research({
  lead: {
    name: 'John Doe',
    company: 'Acme Corp',
    email: 'john@acme.com',
  },
})

console.log(research.company.painPoints)
console.log(research.person.interests)
```

## Example 3: Qualify Lead

```typescript
const qualification = await sarah.qualifyLead({
  leadId: 'lead-123',
  criteria: {
    budget: { min: 10000 },
    authority: 'decision_maker',
    need: 'high',
    timing: '3_months',
  },
})

if (qualification.qualified) {
  console.log('Lead is qualified!')
}
```

## Example 4: Book Meeting

```typescript
const meeting = await sarah.bookMeeting({
  leadId: 'lead-123',
  type: 'discovery_call',
  duration: 30,
})

console.log(`Meeting booked for ${meeting.scheduledTime}`)
```

## Example 5: Track Performance

```typescript
const performance = await sarah.getPerformance({
  period: 'last_30_days',
})

console.log(`Leads qualified: ${performance.leadsQualified}`)
console.log(`Meetings booked: ${performance.meetingsBooked}`)
console.log(`Show rate: ${performance.showRate}%`)
```
