# sdr.do

**Sales Development Representative Agent**

sdr.do provides autonomous SDR (Sales Development Representative) agents that handle lead qualification, outbound prospecting, meeting scheduling, and pipeline management. Build AI-powered sales development teams that work 24/7.

## What is an SDR Agent?

An SDR agent is a specialized sales worker that:

- **Researches**: Gathers intel on leads and accounts
- **Prospects**: Identifies and reaches out to potential customers
- **Qualifies**: Evaluates lead fit and intent
- **Books**: Schedules meetings with qualified prospects
- **Nurtures**: Maintains engagement until ready to buy

## Quick Start

```typescript
import $, { sdr } from 'sdk.do'

// Create an SDR agent
const sarah = await sdr.create({
  $type: 'SalesDevelopmentRepresentative',
  name: 'Sarah Chen',
  persona: {
    background: 'Former tech recruiter, 5 years in B2B SaaS sales',
    style: 'consultative, data-driven, relationship-focused',
    expertise: ['lead_qualification', 'cold_outreach', 'discovery_calls'],
  },
  territory: {
    industries: ['Technology', 'SaaS', 'FinTech'],
    companySize: { min: 50, max: 5000 },
    regions: ['North America'],
  },
  quota: {
    leadsQualified: 100, // per month
    meetingsBooked: 50,
    pipelineValue: 500000,
    showRate: 0.5,
  },
  tools: ['linkedin', 'email', 'crm', 'calendar'],
  model: 'claude-sonnet-4.5',
})

// Assign leads
await sarah.assignLeads({
  leads: leadList,
  priority: 'high',
})

// Check performance
const performance = await sarah.getPerformance()
```

## SDR Workflows

### New Lead Workflow

```typescript
// Automated new lead workflow
await sarah.createWorkflow({
  trigger: $.Lead.created,
  steps: [
    {
      action: 'research',
      data: ['company', 'person', 'intent_signals'],
      timeout: 300,
    },
    {
      action: 'qualify',
      framework: 'BANT',
      threshold: 7.0,
    },
    {
      action: 'personalize_outreach',
      channels: ['email', 'linkedin'],
      template: 'value_prop',
    },
    {
      action: 'schedule_followup',
      delay: 3,
      type: 'check_engagement',
    },
  ],
  onQualified: async (lead) => {
    await sarah.scheduleMeeting({ lead, type: 'discovery' })
  },
  onNotQualified: async (lead) => {
    await sarah.addToNurture({ lead, campaign: 'long_term' })
  },
})
```

## Integration Examples

```typescript
import $, { sdr, db, on, send } from 'sdk.do'

// Assign new leads automatically
await on($.Lead.created, async (lead) => {
  await sarah.assignLead(lead)
})

// Update CRM on qualification
await sarah.onLeadQualified(async (lead, qualification) => {
  await db.update($.Lead, lead.id, {
    status: 'qualified',
    bant: qualification,
    qualifiedBy: sarah.id,
    qualifiedAt: new Date(),
  })

  // Notify sales team
  await send($.Lead.qualified, { lead, qualification })
})

// Track meeting outcomes
await sarah.onMeetingCompleted(async (meeting, outcome) => {
  await db.create($.Meeting, {
    leadId: meeting.leadId,
    outcome: outcome.result,
    nextSteps: outcome.nextSteps,
    opportunityCreated: outcome.qualified,
  })
})
```
