# API Reference

Complete API reference for SDR (Sales Development Representative) agents.

## sdr.create()

Create a new SDR agent.

```typescript
sdr.create(options: SDROptions): Promise<SDRAgent>
```

**Parameters:**

- `options.name` (string): Agent name
- `options.territory` (object): Territory assignment
- `options.quota` (object): Performance quotas
- `options.model` (string): LLM model

## research()

Research a lead and their company.

```typescript
agent.research(options: ResearchOptions): Promise<LeadResearch>
```

## qualifyLead()

Qualify a lead using BANT or custom criteria.

```typescript
agent.qualifyLead(options: QualificationOptions): Promise<Qualification>
```

## bookMeeting()

Book a meeting with a qualified lead.

```typescript
agent.bookMeeting(options: MeetingOptions): Promise<Meeting>
```

## getPerformance()

Get SDR performance metrics.

```typescript
agent.getPerformance(options: PerformanceOptions): Promise<SDRMetrics>
```
