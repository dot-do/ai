# Getting Started

Get started with SDR (Sales Development Representative) agents in the `.do` platform and build autonomous sales development teams.

## Installation

```bash
pnpm add sdk.do
```

## Environment Setup

```bash
# LLM Provider
OPENAI_API_KEY=sk-...
ANTHROPIC_API_KEY=sk-ant-...

# CRM Integration
SALESFORCE_API_KEY=...
HUBSPOT_API_KEY=...

# Communication Tools
LINKEDIN_API_KEY=...
EMAIL_SMTP_HOST=smtp.gmail.com
EMAIL_SMTP_USER=...
EMAIL_SMTP_PASS=...
```

## Creating Your First SDR Agent

```typescript
import { $, sdr } from 'sdk.do'

const sarah = await sdr.create({
  $type: 'SalesDevelopmentRepresentative',
  name: 'Sarah Chen',
  territory: {
    industries: ['Technology', 'SaaS'],
    companySize: { min: 50, max: 5000 },
    regions: ['North America'],
  },
  quota: {
    leadsQualified: 100,
    meetingsBooked: 50,
    pipelineValue: 500000,
  },
  model: 'claude-sonnet-4.5',
})
```

## Research Leads

```typescript
const research = await sarah.research({
  lead: {
    name: 'John Doe',
    title: 'VP of Engineering',
    company: 'Acme Corp',
    email: 'john@acme.com',
  },
})
```

## Qualify Leads

```typescript
const qualification = await sarah.qualifyLead({
  leadId: 'lead-123',
  criteria: {
    budget: { min: 10000 },
    authority: 'decision_maker',
    need: 'high',
    timing: '3_months',
  },
})
```

## Book Meetings

```typescript
const meeting = await sarah.bookMeeting({
  leadId: 'lead-123',
  type: 'discovery_call',
  duration: 30,
  availability: sarah.calendar.getAvailability(),
})
```
