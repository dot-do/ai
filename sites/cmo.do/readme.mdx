# cmo.do

**Chief Marketing Officer Agent**

cmo.do provides autonomous CMO (Chief Marketing Officer) agents that handle marketing strategy, campaign management, content creation, analytics, and brand building. Build AI-powered marketing teams that drive growth and optimize marketing ROI.

## What is a CMO Agent?

A CMO agent is a specialized marketing worker that:

- **Strategizes**: Develops marketing strategies and go-to-market plans
- **Executes**: Manages campaigns across multiple channels
- **Creates**: Generates marketing content and creative assets
- **Analyzes**: Tracks metrics and optimizes performance
- **Builds**: Develops and maintains brand identity

## Quick Start

```typescript
import $, { cmo } from 'sdk.do'

// Create a CMO agent
const taylor = await cmo.create({
  $type: 'ChiefMarketingOfficer',
  name: 'Taylor Morgan',
  persona: {
    background: 'MBA Marketing, 12 years in B2B SaaS marketing',
    style: 'data-driven, creative, growth-focused',
    expertise: ['growth_marketing', 'content_strategy', 'analytics', 'brand_building'],
  },
  responsibilities: ['Marketing strategy', 'Campaign management', 'Content creation', 'Marketing analytics', 'Brand development'],
  channels: ['content', 'email', 'social', 'paid', 'events', 'seo'],
  tools: ['hubspot', 'google_analytics', 'semrush', 'canva'],
  model: 'claude-sonnet-4.5',
})

// Create marketing campaign
const campaign = await taylor.createCampaign({
  name: 'Q4 Product Launch',
  goal: 'generate_leads',
  target: 500,
  channels: ['content', 'email', 'social', 'paid'],
})

// Analyze performance
const analytics = await taylor.analyzePerformance({
  period: 'last_30_days',
})
```

## Integration Examples

```typescript
import $, { cmo, db, on, every } from 'sdk.do'

// Auto-generate weekly reports
await every($.Weekly, async () => {
  const report = await taylor.generateWeeklyReport()
  await db.create($.MarketingReport, report)
  await taylor.distribute(report, ['leadership', 'marketing_team'])
})

// Track campaign performance
await on($.Campaign.launched, async (campaign) => {
  const tracking = await taylor.trackCampaign(campaign.id)
  await db.create($.CampaignTracking, tracking)
})

// Optimize based on performance
await every($.Daily, async () => {
  const campaigns = await taylor.getActiveCampaigns()

  for (const campaign of campaigns) {
    const performance = await taylor.analyze(campaign.id)

    if (performance.underperforming) {
      await taylor.optimize(campaign.id, {
        adjustBid: true,
        refreshCreative: true,
        refinetargeting: true,
      })
    }
  }
})
```
