# basic usage

Practical examples for cmo.do.

## Example Implementation

```typescript
import { cmo } from 'sdk.do'

const agent = await cmo.create({
  name: 'Example Agent',
  role: 'cmo Specialist',
  model: 'claude-sonnet-4.5',
})

await agent.work({ task: 'example_task' })
```
