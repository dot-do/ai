# Troubleshooting

## Authentication Errors

### "Unauthorized" (401)

```typescript
// Check authentication
await gateway.get(url, {
  auth: {
    type: 'bearer',
    token: process.env.API_TOKEN, // Verify token is valid
  },
})
```

### "Forbidden" (403)

Check API permissions and rate limits.

## Rate Limiting

### "Too Many Requests" (429)

```typescript
// Implement retry with backoff
await gateway.get(url, {
  retry: {
    max: 5,
    delay: 1000,
    backoff: 'exponential',
  },
})
```

## Timeout Errors

### "Request Timeout"

```typescript
// Increase timeout
await gateway.get(url, {
  timeout: 60000, // 60 seconds
})
```

## Cache Issues

### Cache Not Working

```typescript
// Verify cache configuration
gateway.config({
  cache: { enabled: true, ttl: 3600 },
})

// Check cache key
const data = await gateway.get(url, {
  cache: { enabled: true, key: 'custom-key' },
})
```

## Connection Errors

### "Network Error"

```typescript
// Check URL and network connectivity
// Implement retry logic
await gateway.get(url, {
  retry: { max: 3 },
})
```
