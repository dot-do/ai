# verbs.do API Reference

Complete API reference for semantic verbs and operations.

## CRUD Operations

### $.create()

Creates a new entity.

```typescript
function create<T>(type: TypeReference, data: Partial<T>): Promise<T>
```

**Example**:

```typescript
const person = await $.create($.Person, {
  $type: 'Person',
  name: 'Jane Smith',
  email: 'jane@example.com',
})
```

### $.get()

Retrieves an entity by ID.

```typescript
function get<T>(type: TypeReference, id: string): Promise<T>
```

### $.list()

Lists entities of a type.

```typescript
function list<T>(type: TypeReference, options?: QueryOptions): Promise<T[]>

interface QueryOptions {
  where?: Record<string, any>
  limit?: number
  offset?: number
  orderBy?: Record<string, 'asc' | 'desc'>
}
```

### $.update()

Updates an entity.

```typescript
function update<T>(entity: T, data: Partial<T>): Promise<T>
```

### $.delete()

Deletes an entity.

```typescript
function delete<T>(entity: T | string): Promise<void>
```

## GraphDL Predicates

### $.worksFor

Person works for Organization.

```typescript
await db.relate(person, $.worksFor, organization)
const employer = await db.related(person, $.worksFor, $.Organization)
```

### $.employs

Organization employs Person.

```typescript
await db.relate(organization, $.employs, person)
const employees = await db.related(organization, $.employs, $.Person)
```

### $.owns

Entity owns another entity.

```typescript
await db.relate(organization, $.owns, brand)
const brands = await db.related(organization, $.owns, $.Brand)
```

### $.manages

Person manages entity.

```typescript
await db.relate(manager, $.manages, team)
const reports = await db.related(manager, $.manages, $.Person)
```

### $.produces

Organization produces product/service.

```typescript
await db.relate(organization, $.produces, service)
const services = await db.related(organization, $.produces, $.Service)
```

### $.manufactures

Brand manufactures product.

```typescript
await db.relate(brand, $.manufactures, product)
const products = await db.related(brand, $.manufactures, $.Product)
```

### $.sells

Business sells product.

```typescript
await db.relate(business, $.sells, product)
const offerings = await db.related(business, $.sells, $.Product)
```

## Business Operations

### $.process()

Processes an entity (order, payment, etc.).

```typescript
function process<T>(entity: T): Promise<T>
```

### $.approve()

Approves an entity.

```typescript
function approve<T>(entity: T, approver: Person): Promise<T>
```

### $.fulfill()

Fulfills an order or request.

```typescript
function fulfill<T>(entity: T): Promise<T>
```

### $.ship()

Ships an order.

```typescript
function ship(order: Order, details: ShippingDetails): Promise<Order>
```

### $.notify()

Sends a notification.

```typescript
function notify(recipient: Person, notification: Notification): Promise<void>
```
