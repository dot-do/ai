# verbs.do Troubleshooting

Common issues and solutions when working with semantic verbs.

## Operation Errors

### Verb Not Found

**Problem**: `Unknown verb: customVerb`

**Solution**:

```typescript
import { registry } from 'verbs.do/core'

// Register custom verb
registry.registerVerb({
  name: 'customVerb',
  implementation: async (params) => {
    // Implementation
  },
})
```

### Invalid Parameters

**Problem**: `Invalid parameters for verb create`

**Solution**:

```typescript
// Ensure correct parameter types
await $.create($.Person, {
  $type: 'Person', // Required
  name: 'John', // Required properties
})
```

## Relationship Errors

### Invalid Predicate

**Problem**: `Invalid predicate: invalidPredicate`

**Solution**:

```typescript
// Use valid GraphDL predicates
await db.relate(person, $.worksFor, organization) // ✓ Valid
await db.relate(person, 'works-for', organization) // ✗ Invalid
```

### Domain/Range Mismatch

**Problem**: `Type mismatch: Person cannot worksFor Person`

**Solution**:

```typescript
// Ensure correct domain/range
await db.relate(person, $.worksFor, organization) // ✓ Valid
await db.relate(person, $.worksFor, otherPerson) // ✗ Invalid
```

## Performance Issues

### Slow Operations

**Solution**:

```typescript
// Use batch operations
const people = await $.createMany($.Person, [{ name: 'Alice' }, { name: 'Bob' }, { name: 'Carol' }])

// Instead of:
for (const data of peopleData) {
  await $.create($.Person, data) // Slow
}
```
