# Basic Usage Examples

## CRUD Operations

```typescript
import $, { db } from 'sdk.do'

// Create
const person = await $.create($.Person, {
  $type: 'Person',
  name: 'Jane Smith',
  email: 'jane@example.com',
})

// Read
const retrieved = await $.get($.Person, person.$id)
const all = await $.list($.Person)

// Update
await $.update(person, { jobTitle: 'CTO' })

// Delete
await $.delete(person)
```

## Relationships

```typescript
// Create entities
const person = await $.create($.Person, { name: 'John' })
const org = await $.create($.Organization, { name: 'Acme' })

// Create relationship
await db.relate(person, $.worksFor, org)

// Query relationship
const employer = await db.related(person, $.worksFor, $.Organization)
```

## Business Operations

```typescript
// Process order
await $.process(order)
await $.fulfill(order)
await $.ship(order)

// Notify customer
await $.notify(customer, {
  message: 'Order shipped',
})
```
