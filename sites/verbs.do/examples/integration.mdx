# Integration Examples

## With nouns.do

```typescript
// Nouns define types
const person = await $.get($.Person, 'person-123')

// Verbs operate on entities
await $.update(person, { jobTitle: 'CTO' })
await $.notify(person, { message: 'Promoted!' })
```

## With resources.do

```typescript
// Verbs create resources
const person = await $.create($.Person, { name: 'Jane' })

// Verbs operate on resources
await $.update(person, { email: 'jane@newco.com' })
await $.delete(person)
```

## With events.do

```typescript
import { on, send } from 'sdk.do'

// Listen for verb events
on($.Order.created, async (order) => {
  await $.process(order)
})

// Verbs trigger events
await $.create($.Order, orderData) // Triggers event
```

## With graphdl.do

```typescript
import { graph } from 'graphdl.do'

const g = graph`
  ${person: $.Person} -[${$.worksFor}]-> ${org: $.Organization}
`

const results = await g.query({ person: { name: 'John' } })
```
