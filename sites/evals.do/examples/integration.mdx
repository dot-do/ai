# Integration

Integrate evals.do with the .do platform.

## Database Integration

\`\`\`typescript
import \$, { evals, db } from 'sdk.do'

const data = await db.list($.Data)
const result = await evals.run({ data })
\`\`\`

## Event Integration

\`\`\`typescript
import \$, { evals, on } from 'sdk.do'

on($.Event.occurred, async (event) => {
await evals.run({ event })
})
\`\`\`
