# Blazemeter Integration

BlazeMeter is a continuous testing platform that enables users to create, run, and analyze performance and functional tests for web and mobile applications.

## Quick Start

```typescript
import { BlazemeterClient } from '@dotdo/sdk.do/integrations/blazemeter'

const blazemeter = new BlazemeterClient({
  apiKey: process.env.BLAZEMETER_API_KEY,
})

const result = await blazemeter.action.execute({
  action: 'example_action',
  parameters: {},
})
```
