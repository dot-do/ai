# Memberstack Integration

Memberstack is a platform that enables developers to add user authentication, payments, and member management to their websites without writing backend code.

## Quick Start

```typescript
import { MemberstackClient } from '@dotdo/sdk.do/integrations/memberstack'

const memberstack = new MemberstackClient({
  apiKey: process.env.MEMBERSTACK_API_KEY,
})

const result = await memberstack.action.execute({
  action: 'example_action',
  parameters: {},
})
```
