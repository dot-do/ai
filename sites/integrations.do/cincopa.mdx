# Cincopa Integration

Cincopa is a comprehensive media platform offering tools for uploading, managing, and customizing multimedia content, including videos, images, and audio, with robust APIs for seamless integration.

## Quick Start

```typescript
import { CincopaClient } from '@dotdo/sdk.do/integrations/cincopa'

const cincopa = new CincopaClient({
  apiKey: process.env.CINCOPA_API_KEY,
})

const result = await cincopa.action.execute({
  action: 'example_action',
  parameters: {},
})
```
