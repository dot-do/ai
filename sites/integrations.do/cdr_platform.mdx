# Cdr platform Integration

CDR Platform provides an API for purchasing carbon dioxide removal services.

## Quick Start

```typescript
import { CdrplatformClient } from '@dotdo/sdk.do/integrations/cdr_platform'

const cdr_platform = new CdrplatformClient({
  apiKey: process.env.CDR_PLATFORM_API_KEY,
})

const result = await cdr_platform.action.execute({
  action: 'example_action',
  parameters: {},
})
```
