# Eventbrite Integration

Eventbrite enables organizers to plan, promote, and manage events, selling tickets and providing attendee tools for conferences, concerts, and gatherings

## Quick Start

```typescript
import { EventbriteClient } from '@dotdo/sdk.do/integrations/eventbrite'

const eventbrite = new EventbriteClient({
  apiKey: process.env.EVENTBRITE_API_KEY,
})

const result = await eventbrite.action.execute({
  action: 'example_action',
  parameters: {},
})
```
