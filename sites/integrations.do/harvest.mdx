# Harvest Integration

Harvest is a time-tracking and invoicing tool designed for teams and freelancers, helping them log billable hours, manage projects, and streamline payments

## Quick Start

```typescript
import { HarvestClient } from '@dotdo/sdk.do/integrations/harvest'

const harvest = new HarvestClient({
  apiKey: process.env.HARVEST_API_KEY,
})

const result = await harvest.action.execute({
  action: 'example_action',
  parameters: {},
})
```
