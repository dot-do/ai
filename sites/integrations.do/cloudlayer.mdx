# Cloudlayer Integration

cloudlayer.io is a document and asset generation service that enables users to dynamically create PDFs and images through a REST-based API, SDKs, or no-code integrations.

## Quick Start

```typescript
import { CloudlayerClient } from '@dotdo/sdk.do/integrations/cloudlayer'

const cloudlayer = new CloudlayerClient({
  apiKey: process.env.CLOUDLAYER_API_KEY,
})

const result = await cloudlayer.action.execute({
  action: 'example_action',
  parameters: {},
})
```
