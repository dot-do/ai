# Google Classroom Integration

Google Classroom is a free web service developed by Google for schools that aims to simplify creating, distributing, and grading assignments

## Quick Start

```typescript
import { GoogleClassroomClient } from '@dotdo/sdk.do/integrations/google_classroom'

const google_classroom = new GoogleClassroomClient({
  apiKey: process.env.GOOGLE_CLASSROOM_API_KEY,
})

const result = await google_classroom.action.execute({
  action: 'example_action',
  parameters: {},
})
```
