# Share point Integration

SharePoint is a Microsoft platform for document management and intranets, enabling teams to collaborate, store, and organize content securely and effectively

## Quick Start

```typescript
import { SharepointClient } from '@dotdo/sdk.do/integrations/share_point'

const share_point = new SharepointClient({
  apiKey: process.env.SHARE_POINT_API_KEY,
})

const result = await share_point.action.execute({
  action: 'example_action',
  parameters: {},
})
```
