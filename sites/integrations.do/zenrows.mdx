# Zenrows Integration

ZenRows is a web scraping API allowing developers to bypass CAPTCHAs and blocks, gather structured data from dynamic websites, and quickly integrate results into applications

## Quick Start

```typescript
import { ZenrowsClient } from '@dotdo/sdk.do/integrations/zenrows'

const zenrows = new ZenrowsClient({
  apiKey: process.env.ZENROWS_API_KEY,
})

const result = await zenrows.action.execute({
  action: 'example_action',
  parameters: {},
})
```
