# Census bureau Integration

The Census Bureau Data API provides developers with access to a wide range of statistical data collected by the U.S. Census Bureau, facilitating integration into applications and data visualizations.

## Quick Start

```typescript
import { CensusbureauClient } from '@dotdo/sdk.do/integrations/census_bureau'

const census_bureau = new CensusbureauClient({
  apiKey: process.env.CENSUS_BUREAU_API_KEY,
})

const result = await census_bureau.action.execute({
  action: 'example_action',
  parameters: {},
})
```
