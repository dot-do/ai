# Ticktick Integration

TickTick is a cross-platform task management and to-do list application designed to help users organize their tasks and schedules efficiently.

## Quick Start

```typescript
import { TicktickClient } from '@dotdo/sdk.do/integrations/ticktick'

const ticktick = new TicktickClient({
  apiKey: process.env.TICKTICK_API_KEY,
})

const result = await ticktick.action.execute({
  action: 'example_action',
  parameters: {},
})
```
