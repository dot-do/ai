# Remote retrieval Integration

Remote Retrieval automates the logistics of managing equipment returns, providing efficient tracking and management of laptop and monitor returns.

## Quick Start

```typescript
import { RemoteretrievalClient } from '@dotdo/sdk.do/integrations/remote_retrieval'

const remote_retrieval = new RemoteretrievalClient({
  apiKey: process.env.REMOTE_RETRIEVAL_API_KEY,
})

const result = await remote_retrieval.action.execute({
  action: 'example_action',
  parameters: {},
})
```
