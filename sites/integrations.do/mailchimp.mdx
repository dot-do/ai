# Mailchimp Integration

Mailchimp is an email marketing and automation platform providing campaign templates, audience segmentation, and performance analytics to drive engagement and conversions

## Quick Start

```typescript
import { MailchimpClient } from '@dotdo/sdk.do/integrations/mailchimp'

const mailchimp = new MailchimpClient({
  apiKey: process.env.MAILCHIMP_API_KEY,
})

const result = await mailchimp.action.execute({
  action: 'example_action',
  parameters: {},
})
```
