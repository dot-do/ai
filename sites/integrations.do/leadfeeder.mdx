# Leadfeeder Integration

Leadfeeder identifies companies visiting your website and converts them into high-value leads.

## Quick Start

```typescript
import { LeadfeederClient } from '@dotdo/sdk.do/integrations/leadfeeder'

const leadfeeder = new LeadfeederClient({
  apiKey: process.env.LEADFEEDER_API_KEY,
})

const result = await leadfeeder.action.execute({
  action: 'example_action',
  parameters: {},
})
```
