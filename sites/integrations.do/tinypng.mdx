# Tinypng Integration

TinyPNG uses smart lossy compression techniques to reduce the file size of your WebP, JPEG, and PNG files.

## Quick Start

```typescript
import { TinypngClient } from '@dotdo/sdk.do/integrations/tinypng'

const tinypng = new TinypngClient({
  apiKey: process.env.TINYPNG_API_KEY,
})

const result = await tinypng.action.execute({
  action: 'example_action',
  parameters: {},
})
```
