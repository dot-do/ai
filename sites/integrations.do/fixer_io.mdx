# Fixer io Integration

Fixer.io is a simple and lightweight API for current and historical foreign exchange rates and currency conversion.

## Quick Start

```typescript
import { FixerioClient } from '@dotdo/sdk.do/integrations/fixer_io'

const fixer_io = new FixerioClient({
  apiKey: process.env.FIXER_IO_API_KEY,
})

const result = await fixer_io.action.execute({
  action: 'example_action',
  parameters: {},
})
```
