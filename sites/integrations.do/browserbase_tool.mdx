# Browserbase tool Integration

Browserbase is a serverless platform that enables developers to run, manage, and monitor headless browsers at scale, offering seamless integration with tools like Playwright, Puppeteer, and Selenium.

## Quick Start

```typescript
import { BrowserbasetoolClient } from '@dotdo/sdk.do/integrations/browserbase_tool'

const browserbase_tool = new BrowserbasetoolClient({
  apiKey: process.env.BROWSERBASE_TOOL_API_KEY,
})

const result = await browserbase_tool.action.execute({
  action: 'example_action',
  parameters: {},
})
```
