# SurveyMonkey Integration

Online survey creation and analysis platform.

## Quick Start

```typescript
import { SurveyMonkeyClient } from '@dotdo/sdk.do/integrations/surveymonkey'

const surveymonkey = new SurveyMonkeyClient({
  clientId: process.env.SURVEYMONKEY_CLIENT_ID,
  clientSecret: process.env.SURVEYMONKEY_CLIENT_SECRET,
  accessToken: process.env.SURVEYMONKEY_ACCESS_TOKEN,
})

const survey = await surveymonkey.survey.create({
  title: 'Customer Satisfaction Survey',
})
```
