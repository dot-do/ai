# Ocrspace Integration

OCR.space provides a free and paid OCR API for extracting text from images and PDFs, returning results in JSON format.

## Quick Start

```typescript
import { OcrspaceClient } from '@dotdo/sdk.do/integrations/ocrspace'

const ocrspace = new OcrspaceClient({
  apiKey: process.env.OCRSPACE_API_KEY,
})

const result = await ocrspace.action.execute({
  action: 'example_action',
  parameters: {},
})
```

Ocrspace provides 1 actions through the Composio API.
