# Scrapingbee Integration

ScrapingBee is a web scraping API that handles headless browsers and proxy rotation, allowing developers to extract HTML from any website in a single API call.

## Quick Start

```typescript
import { ScrapingbeeClient } from '@dotdo/sdk.do/integrations/scrapingbee'

const scrapingbee = new ScrapingbeeClient({
  apiKey: process.env.SCRAPINGBEE_API_KEY,
})

const result = await scrapingbee.action.execute({
  action: 'example_action',
  parameters: {},
})
```
