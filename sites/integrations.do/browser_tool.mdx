# Browser tool Integration

Composio enables AI Agents and LLMs to authenticate and integrate with various tools via function calling.

## Quick Start

```typescript
import { BrowsertoolClient } from '@dotdo/sdk.do/integrations/browser_tool'

const browser_tool = new BrowsertoolClient({
  apiKey: process.env.BROWSER_TOOL_API_KEY,
})

const result = await browser_tool.action.execute({
  action: 'example_action',
  parameters: {},
})
```
