# Asana Integration

Tool to help teams organize, track, and manage their work.

## Quick Start

```typescript
import { AsanaClient } from '@dotdo/sdk.do/integrations/asana'

const asana = new AsanaClient({
  apiKey: process.env.ASANA_API_KEY,
})

const result = await asana.action.execute({
  action: 'example_action',
  parameters: {},
})
```
