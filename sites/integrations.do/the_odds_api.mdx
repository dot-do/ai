# The odds api Integration

The Odds API provides real-time sports betting odds data from various bookmakers worldwide, covering multiple sports and betting markets.

## Quick Start

```typescript
import { TheoddsapiClient } from '@dotdo/sdk.do/integrations/the_odds_api'

const the_odds_api = new TheoddsapiClient({
  apiKey: process.env.THE_ODDS_API_API_KEY,
})

const result = await the_odds_api.action.execute({
  action: 'example_action',
  parameters: {},
})
```
