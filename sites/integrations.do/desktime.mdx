# Desktime Integration

DeskTime is an automatic time tracking software that helps teams and freelancers monitor productivity, manage projects, and analyze work habits.

## Quick Start

```typescript
import { DesktimeClient } from '@dotdo/sdk.do/integrations/desktime'

const desktime = new DesktimeClient({
  apiKey: process.env.DESKTIME_API_KEY,
})

const result = await desktime.action.execute({
  action: 'example_action',
  parameters: {},
})
```
