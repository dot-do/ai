# Google BigQuery Integration

Google BigQuery is a fully managed data warehouse for large-scale data analytics, offering fast SQL queries and machine learning capabilities on massive datasets

## Quick Start

```typescript
import { GoogleBigQueryClient } from '@dotdo/sdk.do/integrations/googlebigquery'

const googlebigquery = new GoogleBigQueryClient({
  apiKey: process.env.GOOGLEBIGQUERY_API_KEY,
})

const result = await googlebigquery.action.execute({
  action: 'example_action',
  parameters: {},
})
```
