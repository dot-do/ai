# Fireflies Integration

Fireflies.ai helps your team transcribe, summarize, search, and analyze voice conversations.

## Quick Start

```typescript
import { FirefliesClient } from '@dotdo/sdk.do/integrations/fireflies'

const fireflies = new FirefliesClient({
  apiKey: process.env.FIREFLIES_API_KEY,
})

const result = await fireflies.action.execute({
  action: 'example_action',
  parameters: {},
})
```

## Actions

Fireflies provides 10 actions through the Composio API.

## Webhooks

Fireflies supports 1 webhook triggers for real-time event notifications.
