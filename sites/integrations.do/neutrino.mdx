# Neutrino Integration

Neutrino API provides a suite of general-purpose APIs for various tasks, including data validation, geolocation, and security.

## Quick Start

```typescript
import { NeutrinoClient } from '@dotdo/sdk.do/integrations/neutrino'

const neutrino = new NeutrinoClient({
  apiKey: process.env.NEUTRINO_API_KEY,
})

const result = await neutrino.action.execute({
  action: 'example_action',
  parameters: {},
})
```
