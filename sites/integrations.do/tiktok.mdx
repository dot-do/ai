# Tiktok Integration

## Quick Start

```typescript
import { TiktokClient } from '@dotdo/sdk.do/integrations/tiktok'

const tiktok = new TiktokClient({
  apiKey: process.env.TIKTOK_API_KEY,
})

const result = await tiktok.action.execute({
  action: 'example_action',
  parameters: {},
})
```
