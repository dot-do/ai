# Abyssale Integration

Abyssale is a creative automation platform that enables users to generate images, videos, GIFs, PDFs, and HTML5 content programmatically, streamlining visual content production.

## Quick Start

```typescript
import { AbyssaleClient } from '@dotdo/sdk.do/integrations/abyssale'

const abyssale = new AbyssaleClient({
  apiKey: process.env.ABYSSALE_API_KEY,
})

const result = await abyssale.action.execute({
  action: 'example_action',
  parameters: {},
})
```
