# Bestbuy Integration

Best Buy offers a suite of APIs providing access to product, store, category, and recommendation data.

## Quick Start

```typescript
import { BestbuyClient } from '@dotdo/sdk.do/integrations/bestbuy'

const bestbuy = new BestbuyClient({
  apiKey: process.env.BESTBUY_API_KEY,
})

const result = await bestbuy.action.execute({
  action: 'example_action',
  parameters: {},
})
```
