# Bitbucket Integration

Bitbucket is a Git-based code hosting and collaboration platform supporting private and public repositories, enabling teams to manage and review code through pull requests and integrations

## Quick Start

```typescript
import { BitbucketClient } from '@dotdo/sdk.do/integrations/bitbucket'

const bitbucket = new BitbucketClient({
  apiKey: process.env.BITBUCKET_API_KEY,
})

const result = await bitbucket.action.execute({
  action: 'example_action',
  parameters: {},
})
```
