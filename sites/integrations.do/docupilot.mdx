# Docupilot Integration

Docupilot is a document automation tool that enables users to generate PDFs, DOCX files, contracts, invoices, and more by integrating with various online services.

## Quick Start

```typescript
import { DocupilotClient } from '@dotdo/sdk.do/integrations/docupilot'

const docupilot = new DocupilotClient({
  apiKey: process.env.DOCUPILOT_API_KEY,
})

const result = await docupilot.action.execute({
  action: 'example_action',
  parameters: {},
})
```
