# Chatbotkit Integration

ChatBotKit is a platform that enables developers to build and manage AI-powered chatbots, offering comprehensive APIs and SDKs for seamless integration into applications.

## Quick Start

```typescript
import { ChatbotkitClient } from '@dotdo/sdk.do/integrations/chatbotkit'

const chatbotkit = new ChatbotkitClient({
  apiKey: process.env.CHATBOTKIT_API_KEY,
})

const result = await chatbotkit.action.execute({
  action: 'example_action',
  parameters: {},
})
```
