# Botpress Integration

Botpress is an open-source platform for building, deploying, and managing chatbots.

## Quick Start

```typescript
import { BotpressClient } from '@dotdo/sdk.do/integrations/botpress'

const botpress = new BotpressClient({
  apiKey: process.env.BOTPRESS_API_KEY,
})

const result = await botpress.action.execute({
  action: 'example_action',
  parameters: {},
})
```
