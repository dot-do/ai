# Instacart Integration

Instacart Developer Platform APIs to create shoppable lists/recipes and discover nearby retailers

## Quick Start

```typescript
import { InstacartClient } from '@dotdo/sdk.do/integrations/instacart'

const instacart = new InstacartClient({
  apiKey: process.env.INSTACART_API_KEY,
})

const result = await instacart.action.execute({
  action: 'example_action',
  parameters: {},
})
```
