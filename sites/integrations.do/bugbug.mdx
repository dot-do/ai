# Bugbug Integration

Bugbug is a platform for testing and monitoring your code.

## Quick Start

```typescript
import { BugbugClient } from '@dotdo/sdk.do/integrations/bugbug'

const bugbug = new BugbugClient({
  apiKey: process.env.BUGBUG_API_KEY,
})

const result = await bugbug.action.execute({
  action: 'example_action',
  parameters: {},
})
```
