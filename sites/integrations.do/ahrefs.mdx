# Ahrefs Integration

Ahrefs is an SEO and marketing platform offering site audits, keyword research, content analysis, and competitive insights to improve search rankings and drive organic traffic

## Quick Start

```typescript
import { AhrefsClient } from '@dotdo/sdk.do/integrations/ahrefs'

const ahrefs = new AhrefsClient({
  apiKey: process.env.AHREFS_API_KEY,
})

const result = await ahrefs.action.execute({
  action: 'example_action',
  parameters: {},
})
```
