# Shortpixel Integration

ShortPixel is an image optimization and compression service that offers APIs and plugins to enhance website performance by reducing image sizes without compromising quality.

## Quick Start

```typescript
import { ShortpixelClient } from '@dotdo/sdk.do/integrations/shortpixel'

const shortpixel = new ShortpixelClient({
  apiKey: process.env.SHORTPIXEL_API_KEY,
})

const result = await shortpixel.action.execute({
  action: 'example_action',
  parameters: {},
})
```
