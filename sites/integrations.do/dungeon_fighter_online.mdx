# Dungeon fighter online Integration

Dungeon Fighter Online (DFO) is an arcade-style, side-scrolling action game with RPG elements, offering players a dynamic combat experience.

## Quick Start

```typescript
import { DungeonfighteronlineClient } from '@dotdo/sdk.do/integrations/dungeon_fighter_online'

const dungeon_fighter_online = new DungeonfighteronlineClient({
  apiKey: process.env.DUNGEON_FIGHTER_ONLINE_API_KEY,
})

const result = await dungeon_fighter_online.action.execute({
  action: 'example_action',
  parameters: {},
})
```
