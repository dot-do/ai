# Freshdesk Integration

Freshdesk provides customer support software with ticketing, knowledge base, and automation features for efficient helpdesk operations and better customer experiences

## Quick Start

```typescript
import { FreshdeskClient } from '@dotdo/sdk.do/integrations/freshdesk'

const freshdesk = new FreshdeskClient({
  apiKey: process.env.FRESHDESK_API_KEY,
})

const result = await freshdesk.action.execute({
  action: 'example_action',
  parameters: {},
})
```
