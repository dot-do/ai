# Beeminder Integration

Beeminder is an online goal-tracking service that uses financial incentives—pledge money on your objectives and forfeit it if you don’t keep up—to help you achieve your goals.

## Quick Start

```typescript
import { BeeminderClient } from '@dotdo/sdk.do/integrations/beeminder'

const beeminder = new BeeminderClient({
  apiKey: process.env.BEEMINDER_API_KEY,
})

const result = await beeminder.action.execute({
  action: 'example_action',
  parameters: {},
})
```
