# Opengraph io Integration

OpenGraph.io provides a simple API to retrieve Open Graph data from websites, even those without properly defined Open Graph tags.

## Quick Start

```typescript
import { OpengraphioClient } from '@dotdo/sdk.do/integrations/opengraph_io'

const opengraph_io = new OpengraphioClient({
  apiKey: process.env.OPENGRAPH_IO_API_KEY,
})

const result = await opengraph_io.action.execute({
  action: 'example_action',
  parameters: {},
})
```
