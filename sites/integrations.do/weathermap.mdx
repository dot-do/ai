# Weathermap Integration

WeatherMap provides visual weather data, forecasts, and mappings, helping users understand climate patterns or track severe weather conditions

## Quick Start

```typescript
import { WeathermapClient } from '@dotdo/sdk.do/integrations/weathermap'

const weathermap = new WeathermapClient({
  apiKey: process.env.WEATHERMAP_API_KEY,
})

const result = await weathermap.action.execute({
  action: 'example_action',
  parameters: {},
})
```

## Actions

Weathermap provides 1 actions through the Composio API.
