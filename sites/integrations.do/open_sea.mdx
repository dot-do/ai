# Open sea Integration

OpenSea is the world's first and largest NFT marketplace for NFTs and crypto collectibles.

## Quick Start

```typescript
import { OpenseaClient } from '@dotdo/sdk.do/integrations/open_sea'

const open_sea = new OpenseaClient({
  apiKey: process.env.OPEN_SEA_API_KEY,
})

const result = await open_sea.action.execute({
  action: 'example_action',
  parameters: {},
})
```
