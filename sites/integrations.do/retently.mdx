# Retently Integration

Retently is a customer experience management platform that enables businesses to collect, analyze, and act on customer feedback through surveys and integrations.

## Quick Start

```typescript
import { RetentlyClient } from '@dotdo/sdk.do/integrations/retently'

const retently = new RetentlyClient({
  apiKey: process.env.RETENTLY_API_KEY,
})

const result = await retently.action.execute({
  action: 'example_action',
  parameters: {},
})
```
