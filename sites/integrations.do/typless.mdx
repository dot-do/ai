# Typless Integration

Typless is an AI-powered document extraction platform that automates manual data entry from various documents using a simple API.

## Quick Start

```typescript
import { TyplessClient } from '@dotdo/sdk.do/integrations/typless'

const typless = new TyplessClient({
  apiKey: process.env.TYPLESS_API_KEY,
})

const result = await typless.action.execute({
  action: 'example_action',
  parameters: {},
})
```
