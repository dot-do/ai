# Whatsapp Integration

Enables interaction with customers through the WhatsApp Business API for messaging and automation. Only supports WhatsApp Business accounts, not WhatsApp Personal accounts.

## Quick Start

```typescript
import { WhatsappClient } from '@dotdo/sdk.do/integrations/whatsapp'

const whatsapp = new WhatsappClient({
  apiKey: process.env.WHATSAPP_API_KEY,
})

const result = await whatsapp.action.execute({
  action: 'example_action',
  parameters: {},
})
```

## Actions

Whatsapp provides 19 actions through the Composio API.
