# Affinda Integration

Affinda provides an AI-powered document processing platform that automates data extraction from various document types.

## Quick Start

```typescript
import { AffindaClient } from '@dotdo/sdk.do/integrations/affinda'

const affinda = new AffindaClient({
  apiKey: process.env.AFFINDA_API_KEY,
})

const result = await affinda.action.execute({
  action: 'example_action',
  parameters: {},
})
```
