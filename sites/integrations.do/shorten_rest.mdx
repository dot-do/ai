# Shorten rest Integration

Shorten.REST is a link shortening and management app that allows you to generate unique, branded URLs with advanced tracking capabilities.

## Quick Start

```typescript
import { ShortenrestClient } from '@dotdo/sdk.do/integrations/shorten_rest'

const shorten_rest = new ShortenrestClient({
  apiKey: process.env.SHORTEN_REST_API_KEY,
})

const result = await shorten_rest.action.execute({
  action: 'example_action',
  parameters: {},
})
```
