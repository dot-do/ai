# Draftable Integration

Draftable provides an enterprise-grade document comparison API that allows developers to compare popular document types such as PDFs, Word, and PowerPoint files.

## Quick Start

```typescript
import { DraftableClient } from '@dotdo/sdk.do/integrations/draftable'

const draftable = new DraftableClient({
  apiKey: process.env.DRAFTABLE_API_KEY,
})

const result = await draftable.action.execute({
  action: 'example_action',
  parameters: {},
})
```

Draftable provides 1 actions through the Composio API.
