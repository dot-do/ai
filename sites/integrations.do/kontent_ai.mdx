# Kontent ai Integration

Kontent.ai is a headless CMS that delivers content via API, allowing developers to build websites and applications using their preferred frameworks, languages, or libraries.

## Quick Start

```typescript
import { KontentaiClient } from '@dotdo/sdk.do/integrations/kontent_ai'

const kontent_ai = new KontentaiClient({
  apiKey: process.env.KONTENT_AI_API_KEY,
})

const result = await kontent_ai.action.execute({
  action: 'example_action',
  parameters: {},
})
```
