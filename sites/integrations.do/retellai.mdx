# Retellai Integration

RetellAI captures calls and transcripts, enabling businesses to analyze conversations, extract insights, and enhance customer interactions in one centralized platform

## Quick Start

```typescript
import { RetellaiClient } from '@dotdo/sdk.do/integrations/retellai'

const retellai = new RetellaiClient({
  apiKey: process.env.RETELLAI_API_KEY,
})

const result = await retellai.action.execute({
  action: 'example_action',
  parameters: {},
})
```
