# Twelve data Integration

Twelve Data provides a comprehensive financial data API offering real-time and historical market data for stocks, forex, cryptocurrencies, ETFs, and indices.

## Quick Start

```typescript
import { TwelvedataClient } from '@dotdo/sdk.do/integrations/twelve_data'

const twelve_data = new TwelvedataClient({
  apiKey: process.env.TWELVE_DATA_API_KEY,
})

const result = await twelve_data.action.execute({
  action: 'example_action',
  parameters: {},
})
```
