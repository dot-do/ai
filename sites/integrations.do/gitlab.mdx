# Gitlab Integration

A web-based DevOps lifecycle tool that provides a Git repository manager providing wiki, issue-tracking, and CI/CD pipeline features.

## Quick Start

```typescript
import { GitlabClient } from '@dotdo/sdk.do/integrations/gitlab'

const gitlab = new GitlabClient({
  apiKey: process.env.GITLAB_API_KEY,
})

const result = await gitlab.action.execute({
  action: 'example_action',
  parameters: {},
})
```

Gitlab provides 54 actions through the Composio API.
