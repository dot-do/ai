# Cutt ly Integration

Cutt.ly is a URL shortening service that allows users to shorten, manage, and analyze URLs.

## Quick Start

```typescript
import { CuttlyClient } from '@dotdo/sdk.do/integrations/cutt_ly'

const cutt_ly = new CuttlyClient({
  apiKey: process.env.CUTT_LY_API_KEY,
})

const result = await cutt_ly.action.execute({
  action: 'example_action',
  parameters: {},
})
```
