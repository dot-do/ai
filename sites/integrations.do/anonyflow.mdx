# Anonyflow Integration

AnonyFlow offers a simple and powerful service for encryption-based data anonymization and community sharing, enabling GDPR, CCPA, and HIPAA data privacy protection compliance.

## Quick Start

```typescript
import { AnonyflowClient } from '@dotdo/sdk.do/integrations/anonyflow'

const anonyflow = new AnonyflowClient({
  apiKey: process.env.ANONYFLOW_API_KEY,
})

const result = await anonyflow.action.execute({
  action: 'example_action',
  parameters: {},
})
```
