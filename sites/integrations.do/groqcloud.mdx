# Groqcloud Integration

GroqCloud provides high-performance AI inference services, enabling developers to deploy and manage AI models efficiently.

## Quick Start

```typescript
import { GroqcloudClient } from '@dotdo/sdk.do/integrations/groqcloud'

const groqcloud = new GroqcloudClient({
  apiKey: process.env.GROQCLOUD_API_KEY,
})

const result = await groqcloud.action.execute({
  action: 'example_action',
  parameters: {},
})
```
