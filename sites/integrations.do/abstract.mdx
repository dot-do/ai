# Abstract Integration

Abstract API provides a suite of APIs for developers to automate various tasks, including data validation, enrichment, and more.

## Quick Start

```typescript
import { AbstractClient } from '@dotdo/sdk.do/integrations/abstract'

const abstract = new AbstractClient({
  apiKey: process.env.ABSTRACT_API_KEY,
})

const result = await abstract.action.execute({
  action: 'example_action',
  parameters: {},
})
```
