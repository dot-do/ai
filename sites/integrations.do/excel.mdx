# Excel Integration

Microsoft Excel is a powerful spreadsheet application for data analysis, calculations, and visualization, enabling users to organize and process data with formulas, charts, and pivot tables

## Quick Start

```typescript
import { ExcelClient } from '@dotdo/sdk.do/integrations/excel'

const excel = new ExcelClient({
  apiKey: process.env.EXCEL_API_KEY,
})

const result = await excel.action.execute({
  action: 'example_action',
  parameters: {},
})
```
