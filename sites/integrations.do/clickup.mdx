# Clickup Integration

ClickUp unifies tasks, docs, goals, and chat in a single platform, allowing teams to plan, organize, and collaborate across projects with customizable workflows

## Quick Start

```typescript
import { ClickupClient } from '@dotdo/sdk.do/integrations/clickup'

const clickup = new ClickupClient({
  apiKey: process.env.CLICKUP_API_KEY,
})

const result = await clickup.action.execute({
  action: 'example_action',
  parameters: {},
})
```
