# Pipedrive Integration

Pipedrive is a sales management tool built around pipeline visualization, lead tracking, activity reminders, and automation to keep deals progressing

## Quick Start

```typescript
import { PipedriveClient } from '@dotdo/sdk.do/integrations/pipedrive'

const pipedrive = new PipedriveClient({
  apiKey: process.env.PIPEDRIVE_API_KEY,
})

const result = await pipedrive.action.execute({
  action: 'example_action',
  parameters: {},
})
```
