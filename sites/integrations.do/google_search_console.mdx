# Google search console Integration

Google Search Console provides tools to monitor, maintain, and troubleshoot your site's presence in Google Search results.

## Quick Start

```typescript
import { GooglesearchconsoleClient } from '@dotdo/sdk.do/integrations/google_search_console'

const google_search_console = new GooglesearchconsoleClient({
  apiKey: process.env.GOOGLE_SEARCH_CONSOLE_API_KEY,
})

const result = await google_search_console.action.execute({
  action: 'example_action',
  parameters: {},
})
```
