# Apiflash Integration

ApiFlash is a website screenshot API that allows users to capture high-quality screenshots of web pages programmatically.

## Quick Start

```typescript
import { ApiflashClient } from '@dotdo/sdk.do/integrations/apiflash'

const apiflash = new ApiflashClient({
  apiKey: process.env.APIFLASH_API_KEY,
})

const result = await apiflash.action.execute({
  action: 'example_action',
  parameters: {},
})
```
