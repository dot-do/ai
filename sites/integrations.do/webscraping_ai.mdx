# Webscraping ai Integration

WebScraping.AI provides an API for web scraping with features like Chrome JS rendering, rotating proxies, and HTML parsing.

## Quick Start

```typescript
import { WebscrapingaiClient } from '@dotdo/sdk.do/integrations/webscraping_ai'

const webscraping_ai = new WebscrapingaiClient({
  apiKey: process.env.WEBSCRAPING_AI_API_KEY,
})

const result = await webscraping_ai.action.execute({
  action: 'example_action',
  parameters: {},
})
```
