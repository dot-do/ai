# Hookdeck Integration

Hookdeck is a platform that enables developers to manage, monitor, and secure webhooks and API events.

## Quick Start

```typescript
import { HookdeckClient } from '@dotdo/sdk.do/integrations/hookdeck'

const hookdeck = new HookdeckClient({
  apiKey: process.env.HOOKDECK_API_KEY,
})

const result = await hookdeck.action.execute({
  action: 'example_action',
  parameters: {},
})
```
