# Payhip Integration

Payhip is an e-commerce platform that enables individuals and businesses to sell digital products, memberships, and physical goods directly to their audience.

## Quick Start

```typescript
import { PayhipClient } from '@dotdo/sdk.do/integrations/payhip'

const payhip = new PayhipClient({
  apiKey: process.env.PAYHIP_API_KEY,
})

const result = await payhip.action.execute({
  action: 'example_action',
  parameters: {},
})
```
