# Google Tasks Integration

Google Tasks provides a simple to-do list and task management system integrated into Gmail and Google Calendar for quick and easy tracking

## Quick Start

```typescript
import { GoogleTasksClient } from '@dotdo/sdk.do/integrations/googletasks'

const googletasks = new GoogleTasksClient({
  apiKey: process.env.GOOGLETASKS_API_KEY,
})

const result = await googletasks.action.execute({
  action: 'example_action',
  parameters: {},
})
```
