# Sourcegraph Integration

Sourcegraph is a code intelligence platform that enables developers to search, understand, and manage code across large codebases.

## Quick Start

```typescript
import { SourcegraphClient } from '@dotdo/sdk.do/integrations/sourcegraph'

const sourcegraph = new SourcegraphClient({
  apiKey: process.env.SOURCEGRAPH_API_KEY,
})

const result = await sourcegraph.action.execute({
  action: 'example_action',
  parameters: {},
})
```
