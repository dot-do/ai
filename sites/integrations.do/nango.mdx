# Nango Integration

Nango provides a unified API to integrate with over 250 external APIs, offering pre-built and customizable integrations for various categories such as CRM, HR, and accounting systems.

## Quick Start

```typescript
import { NangoClient } from '@dotdo/sdk.do/integrations/nango'

const nango = new NangoClient({
  apiKey: process.env.NANGO_API_KEY,
})

const result = await nango.action.execute({
  action: 'example_action',
  parameters: {},
})
```

Nango provides 7 actions through the Composio API.
