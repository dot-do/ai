# Shotstack Integration

Shotstack is a cloud-based platform that enables developers, marketers, and designers to automate the generation of videos, images, and audio at scale using a RESTful API.

## Quick Start

```typescript
import { ShotstackClient } from '@dotdo/sdk.do/integrations/shotstack'

const shotstack = new ShotstackClient({
  apiKey: process.env.SHOTSTACK_API_KEY,
})

const result = await shotstack.action.execute({
  action: 'example_action',
  parameters: {},
})
```
