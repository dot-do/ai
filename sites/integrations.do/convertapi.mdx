# Convertapi Integration

ConvertAPI is a file conversion service that allows developers to convert various file formats, such as documents, images, and spreadsheets, into different formats programmatically.

## Quick Start

```typescript
import { ConvertapiClient } from '@dotdo/sdk.do/integrations/convertapi'

const convertapi = new ConvertapiClient({
  apiKey: process.env.CONVERTAPI_API_KEY,
})

const result = await convertapi.action.execute({
  action: 'example_action',
  parameters: {},
})
```

## Actions

Convertapi provides 11 actions through the Composio API.
