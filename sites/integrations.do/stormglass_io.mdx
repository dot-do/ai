# Stormglass io Integration

stormglass.io provides a global weather API offering high-resolution forecasts and historical data from trusted meteorological institutions.

## Quick Start

```typescript
import { StormglassioClient } from '@dotdo/sdk.do/integrations/stormglass_io'

const stormglass_io = new StormglassioClient({
  apiKey: process.env.STORMGLASS_IO_API_KEY,
})

const result = await stormglass_io.action.execute({
  action: 'example_action',
  parameters: {},
})
```
