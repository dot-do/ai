# Bitquery Integration

Bitquery provides historical and real-time indexed data for over 40 blockchains through GraphQL APIs, Websockets, SQL, and cloud providers.

## Quick Start

```typescript
import { BitqueryClient } from '@dotdo/sdk.do/integrations/bitquery'

const bitquery = new BitqueryClient({
  apiKey: process.env.BITQUERY_API_KEY,
})

const result = await bitquery.action.execute({
  action: 'example_action',
  parameters: {},
})
```
