# Parsehub Integration

ParseHub is a web scraping tool that allows users to extract data from websites without coding.

## Quick Start

```typescript
import { ParsehubClient } from '@dotdo/sdk.do/integrations/parsehub'

const parsehub = new ParsehubClient({
  apiKey: process.env.PARSEHUB_API_KEY,
})

const result = await parsehub.action.execute({
  action: 'example_action',
  parameters: {},
})
```

## Actions

Parsehub provides 1 actions through the Composio API.
