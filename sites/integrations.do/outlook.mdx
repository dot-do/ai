# Outlook Integration

Outlook is Microsoft’s email and calendaring platform integrating contacts, tasks, and scheduling, enabling users to manage communications and events in a unified workspace

## Quick Start

```typescript
import { OutlookClient } from '@dotdo/sdk.do/integrations/outlook'

const outlook = new OutlookClient({
  apiKey: process.env.OUTLOOK_API_KEY,
})

const result = await outlook.action.execute({
  action: 'example_action',
  parameters: {},
})
```

## Actions

Outlook provides 50 actions through the Composio API.

Outlook supports 5 webhook triggers for real-time event notifications.
