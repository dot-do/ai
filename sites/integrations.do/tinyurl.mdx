# Tinyurl Integration

TinyURL shortens lengthy URLs, generating concise links for easier sharing and managing, often used in social media and marketing campaigns

## Quick Start

```typescript
import { TinyurlClient } from '@dotdo/sdk.do/integrations/tinyurl'

const tinyurl = new TinyurlClient({
  apiKey: process.env.TINYURL_API_KEY,
})

const result = await tinyurl.action.execute({
  action: 'example_action',
  parameters: {},
})
```

Tinyurl provides 1 actions through the Composio API.
