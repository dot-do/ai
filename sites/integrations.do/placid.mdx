# Placid Integration

Placid is a creative automation toolkit that enables users to generate images, PDFs, and videos from custom templates via REST or URL APIs.

## Quick Start

```typescript
import { PlacidClient } from '@dotdo/sdk.do/integrations/placid'

const placid = new PlacidClient({
  apiKey: process.env.PLACID_API_KEY,
})

const result = await placid.action.execute({
  action: 'example_action',
  parameters: {},
})
```
