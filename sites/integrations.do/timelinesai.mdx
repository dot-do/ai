# Timelinesai Integration

TimelinesAI enables teams to manage and automate WhatsApp communications, integrating with various CRMs and tools to streamline workflows.

## Quick Start

```typescript
import { TimelinesaiClient } from '@dotdo/sdk.do/integrations/timelinesai'

const timelinesai = new TimelinesaiClient({
  apiKey: process.env.TIMELINESAI_API_KEY,
})

const result = await timelinesai.action.execute({
  action: 'example_action',
  parameters: {},
})
```
