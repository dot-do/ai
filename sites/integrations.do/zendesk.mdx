# Zendesk Integration

Zendesk provides customer support software with ticketing, live chat, and knowledge base features, enabling efficient helpdesk operations and customer engagement

## Quick Start

```typescript
import { ZendeskClient } from '@dotdo/sdk.do/integrations/zendesk'

const zendesk = new ZendeskClient({
  apiKey: process.env.ZENDESK_API_KEY,
})

const result = await zendesk.action.execute({
  action: 'example_action',
  parameters: {},
})
```
