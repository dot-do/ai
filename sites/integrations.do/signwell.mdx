# Signwell Integration

SignWell is an electronic signature tool for legally binding e-signatures and faster document signing.

## Quick Start

```typescript
import { SignwellClient } from '@dotdo/sdk.do/integrations/signwell'

const signwell = new SignwellClient({
  apiKey: process.env.SIGNWELL_API_KEY,
})

const result = await signwell.action.execute({
  action: 'example_action',
  parameters: {},
})
```
