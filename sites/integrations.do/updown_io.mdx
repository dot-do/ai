# Updown io Integration

updown.io is a simple and inexpensive website monitoring service that checks the availability and performance of your websites and APIs.

## Quick Start

```typescript
import { UpdownioClient } from '@dotdo/sdk.do/integrations/updown_io'

const updown_io = new UpdownioClient({
  apiKey: process.env.UPDOWN_IO_API_KEY,
})

const result = await updown_io.action.execute({
  action: 'example_action',
  parameters: {},
})
```
