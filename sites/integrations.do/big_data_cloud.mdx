# Big data cloud Integration

BigDataCloud offers a suite of APIs providing geolocation, reverse geocoding, and data validation services.

## Quick Start

```typescript
import { BigdatacloudClient } from '@dotdo/sdk.do/integrations/big_data_cloud'

const big_data_cloud = new BigdatacloudClient({
  apiKey: process.env.BIG_DATA_CLOUD_API_KEY,
})

const result = await big_data_cloud.action.execute({
  action: 'example_action',
  parameters: {},
})
```
