# Google Analytics Integration

Web analytics and reporting platform.

## Quick Start

```typescript
import { GoogleAnalyticsClient } from '@dotdo/sdk.do/integrations/googleanalytics'

const analytics = new GoogleAnalyticsClient({
  clientId: process.env.GA_CLIENT_ID,
  clientSecret: process.env.GA_CLIENT_SECRET,
  accessToken: process.env.GA_ACCESS_TOKEN,
})

const report = await analytics.report.get({
  viewId: '12345678',
  dateRanges: [{ startDate: '7daysAgo', endDate: 'today' }],
  metrics: [{ expression: 'ga:sessions' }],
  dimensions: [{ name: 'ga:country' }],
})
```
