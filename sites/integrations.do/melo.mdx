# Melo Integration

Melo provides a comprehensive API for accessing real-time, deduplicated real estate listings and market analytics across France.

## Quick Start

```typescript
import { MeloClient } from '@dotdo/sdk.do/integrations/melo'

const melo = new MeloClient({
  apiKey: process.env.MELO_API_KEY,
})

const result = await melo.action.execute({
  action: 'example_action',
  parameters: {},
})
```
