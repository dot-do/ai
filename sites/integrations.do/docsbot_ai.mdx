# Docsbot ai Integration

DocsBot AI enables the creation of custom chatbots trained on your documentation, facilitating automated customer support and content generation.

## Quick Start

```typescript
import { DocsbotaiClient } from '@dotdo/sdk.do/integrations/docsbot_ai'

const docsbot_ai = new DocsbotaiClient({
  apiKey: process.env.DOCSBOT_AI_API_KEY,
})

const result = await docsbot_ai.action.execute({
  action: 'example_action',
  parameters: {},
})
```
