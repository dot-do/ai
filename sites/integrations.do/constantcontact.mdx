# Constant Contact Integration

Email marketing and campaign management platform.

## Quick Start

```typescript
import { ConstantContactClient } from '@dotdo/sdk.do/integrations/constantcontact'

const constantcontact = new ConstantContactClient({
  clientId: process.env.CONSTANTCONTACT_CLIENT_ID,
  clientSecret: process.env.CONSTANTCONTACT_CLIENT_SECRET,
  accessToken: process.env.CONSTANTCONTACT_ACCESS_TOKEN,
})

const contact = await constantcontact.contact.create({
  email_address: 'customer@example.com',
  first_name: 'John',
  last_name: 'Doe',
})
```
