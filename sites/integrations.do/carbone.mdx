# Carbone Integration

Carbone is a fast and simple report generator that converts JSON data into various document formats such as PDF, DOCX, XLSX, ODT, and more, using templates.

## Quick Start

```typescript
import { CarboneClient } from '@dotdo/sdk.do/integrations/carbone'

const carbone = new CarboneClient({
  apiKey: process.env.CARBONE_API_KEY,
})

const result = await carbone.action.execute({
  action: 'example_action',
  parameters: {},
})
```
