# Flexisign Integration

FlexiSign is a privacy and security-focused e-signature tool that simplifies how you sign, send, and manage documents.

## Quick Start

```typescript
import { FlexisignClient } from '@dotdo/sdk.do/integrations/flexisign'

const flexisign = new FlexisignClient({
  apiKey: process.env.FLEXISIGN_API_KEY,
})

const result = await flexisign.action.execute({
  action: 'example_action',
  parameters: {},
})
```
