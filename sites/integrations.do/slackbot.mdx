# Slackbot Integration

Slackbot automates responses and reminders within Slack, assisting with tasks like onboarding, FAQs, and notifications to streamline team productivity

## Quick Start

```typescript
import { SlackbotClient } from '@dotdo/sdk.do/integrations/slackbot'

const slackbot = new SlackbotClient({
  apiKey: process.env.SLACKBOT_API_KEY,
})

const result = await slackbot.action.execute({
  action: 'example_action',
  parameters: {},
})
```

Slackbot provides 133 actions through the Composio API.
