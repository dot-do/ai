# Apitemplate io Integration

APITemplate.io provides APIs for generating PDFs and images from reusable templates using JSON data.

## Quick Start

```typescript
import { ApitemplateioClient } from '@dotdo/sdk.do/integrations/apitemplate_io'

const apitemplate_io = new ApitemplateioClient({
  apiKey: process.env.APITEMPLATE_IO_API_KEY,
})

const result = await apitemplate_io.action.execute({
  action: 'example_action',
  parameters: {},
})
```
