# Linkhut Integration

LinkHut manages bookmarked links in a minimalistic, shareable interface, helping teams organize URLs and track references in one place

## Quick Start

```typescript
import { LinkhutClient } from '@dotdo/sdk.do/integrations/linkhut'

const linkhut = new LinkhutClient({
  apiKey: process.env.LINKHUT_API_KEY,
})

const result = await linkhut.action.execute({
  action: 'example_action',
  parameters: {},
})
```
