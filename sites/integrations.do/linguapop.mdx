# Linguapop Integration

Linguapop is a web-based platform designed for schools and organizations to administer language placement tests in English, German, Spanish, Italian, and French.

## Quick Start

```typescript
import { LinguapopClient } from '@dotdo/sdk.do/integrations/linguapop'

const linguapop = new LinguapopClient({
  apiKey: process.env.LINGUAPOP_API_KEY,
})

const result = await linguapop.action.execute({
  action: 'example_action',
  parameters: {},
})
```
