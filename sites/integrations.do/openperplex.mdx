# Openperplex Integration

Openperplex API provides powerful, global search capabilities and web content analysis for AI applications.

## Quick Start

```typescript
import { OpenperplexClient } from '@dotdo/sdk.do/integrations/openperplex'

const openperplex = new OpenperplexClient({
  apiKey: process.env.OPENPERPLEX_API_KEY,
})

const result = await openperplex.action.execute({
  action: 'example_action',
  parameters: {},
})
```
