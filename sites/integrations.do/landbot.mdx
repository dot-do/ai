# Landbot Integration

Landbot is a no-code chatbot builder that enables businesses to create conversational experiences for customer engagement, lead generation, and support across various messaging channels.

## Quick Start

```typescript
import { LandbotClient } from '@dotdo/sdk.do/integrations/landbot'

const landbot = new LandbotClient({
  apiKey: process.env.LANDBOT_API_KEY,
})

const result = await landbot.action.execute({
  action: 'example_action',
  parameters: {},
})
```

## Actions

Landbot provides 7 actions through the Composio API.
