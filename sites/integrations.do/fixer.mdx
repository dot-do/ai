# Fixer Integration

Fixer is a currency data API providing real-time and historical exchange rates for 170 world currencies.

## Quick Start

```typescript
import { FixerClient } from '@dotdo/sdk.do/integrations/fixer'

const fixer = new FixerClient({
  apiKey: process.env.FIXER_API_KEY,
})

const result = await fixer.action.execute({
  action: 'example_action',
  parameters: {},
})
```
