# Remarkety Integration

Remarkety is an AI-powered marketing automation platform designed for eCommerce, enabling personalized email, SMS, and social campaigns based on customer behavior.

## Quick Start

```typescript
import { RemarketyClient } from '@dotdo/sdk.do/integrations/remarkety'

const remarkety = new RemarketyClient({
  apiKey: process.env.REMARKETY_API_KEY,
})

const result = await remarkety.action.execute({
  action: 'example_action',
  parameters: {},
})
```
