# Perigon Integration

Perigon provides an HTTP REST API for retrieving news and web content data, offering structured data suitable for various applications.

## Quick Start

```typescript
import { PerigonClient } from '@dotdo/sdk.do/integrations/perigon'

const perigon = new PerigonClient({
  apiKey: process.env.PERIGON_API_KEY,
})

const result = await perigon.action.execute({
  action: 'example_action',
  parameters: {},
})
```
