# Dreamstudio Integration

DreamStudio is Stability AI's user interface for generative AI, allowing users to create and edit generated images.

## Quick Start

```typescript
import { DreamstudioClient } from '@dotdo/sdk.do/integrations/dreamstudio'

const dreamstudio = new DreamstudioClient({
  apiKey: process.env.DREAMSTUDIO_API_KEY,
})

const result = await dreamstudio.action.execute({
  action: 'example_action',
  parameters: {},
})
```
