# Mapulus Integration

Mapulus is a platform that enables users to create, manage, and analyze interactive maps, integrating with various applications to automate tasks and leverage location insights.

## Quick Start

```typescript
import { MapulusClient } from '@dotdo/sdk.do/integrations/mapulus'

const mapulus = new MapulusClient({
  apiKey: process.env.MAPULUS_API_KEY,
})

const result = await mapulus.action.execute({
  action: 'example_action',
  parameters: {},
})
```
