# Omnisend Integration

Omnisend is a marketing automation platform for ecommerce businesses, focusing on email and SMS marketing.

## Quick Start

```typescript
import { OmnisendClient } from '@dotdo/sdk.do/integrations/omnisend'

const omnisend = new OmnisendClient({
  apiKey: process.env.OMNISEND_API_KEY,
})

const result = await omnisend.action.execute({
  action: 'example_action',
  parameters: {},
})
```
