# Serply Integration

Serply is a Python SDK for the Serply API, providing a simple interface to access various search engine results, including web, video, image, maps, news, SERP, crawl, product, and job searches.

## Quick Start

```typescript
import { SerplyClient } from '@dotdo/sdk.do/integrations/serply'

const serply = new SerplyClient({
  apiKey: process.env.SERPLY_API_KEY,
})

const result = await serply.action.execute({
  action: 'example_action',
  parameters: {},
})
```

## Actions

Serply provides 3 actions through the Composio API.
