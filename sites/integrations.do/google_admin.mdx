# Google Admin Integration

Google Admin Console for managing Google Workspace users, groups, and organizational units.

## Quick Start

```typescript
import { GoogleAdminClient } from '@dotdo/sdk.do/integrations/google_admin'

const google_admin = new GoogleAdminClient({
  apiKey: process.env.GOOGLE_ADMIN_API_KEY,
})

const result = await google_admin.action.execute({
  action: 'example_action',
  parameters: {},
})
```

## Actions

Google Admin provides 14 actions through the Composio API.
