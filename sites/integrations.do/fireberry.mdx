# Fireberry Integration

Fireberry is a CRM platform that offers integrations with various tools and applications to streamline business processes.

## Quick Start

```typescript
import { FireberryClient } from '@dotdo/sdk.do/integrations/fireberry'

const fireberry = new FireberryClient({
  apiKey: process.env.FIREBERRY_API_KEY,
})

const result = await fireberry.action.execute({
  action: 'example_action',
  parameters: {},
})
```
