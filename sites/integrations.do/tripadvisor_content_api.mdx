# Tripadvisor content api Integration

The Tripadvisor Content API provides developers with access to Tripadvisor’s extensive data set, including over 7.5 million locations, 1 billion reviews and opinions, and support for 29 languages.

## Quick Start

```typescript
import { TripadvisorcontentapiClient } from '@dotdo/sdk.do/integrations/tripadvisor_content_api'

const tripadvisor_content_api = new TripadvisorcontentapiClient({
  apiKey: process.env.TRIPADVISOR_CONTENT_API_API_KEY,
})

const result = await tripadvisor_content_api.action.execute({
  action: 'example_action',
  parameters: {},
})
```

Tripadvisor content api provides 2 actions through the Composio API.
