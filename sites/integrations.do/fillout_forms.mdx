# Fillout forms Integration

Fillout is a platform that allows users to create and manage forms, offering a REST API for programmatic access to form data.

## Quick Start

```typescript
import { FilloutformsClient } from '@dotdo/sdk.do/integrations/fillout_forms'

const fillout_forms = new FilloutformsClient({
  apiKey: process.env.FILLOUT_FORMS_API_KEY,
})

const result = await fillout_forms.action.execute({
  action: 'example_action',
  parameters: {},
})
```
