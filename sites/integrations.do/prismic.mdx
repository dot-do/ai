# Prismic Integration

Prismic is a headless CMS that enables developers and content teams to manage and deliver content seamlessly across various platforms.

## Quick Start

```typescript
import { PrismicClient } from '@dotdo/sdk.do/integrations/prismic'

const prismic = new PrismicClient({
  apiKey: process.env.PRISMIC_API_KEY,
})

const result = await prismic.action.execute({
  action: 'example_action',
  parameters: {},
})
```
