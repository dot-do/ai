# D2lbrightspace Integration

D2L Brightspace is a learning management system that provides a comprehensive suite of tools for educators to create, manage, and deliver online courses and learning experiences.

## Quick Start

```typescript
import { D2lbrightspaceClient } from '@dotdo/sdk.do/integrations/d2lbrightspace'

const d2lbrightspace = new D2lbrightspaceClient({
  apiKey: process.env.D2LBRIGHTSPACE_API_KEY,
})

const result = await d2lbrightspace.action.execute({
  action: 'example_action',
  parameters: {},
})
```
