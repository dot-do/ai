# Proxiedmail Integration

ProxiedMail offers privacy-focused email services, including automatic creation of proxy emails and receiving emails via webhooks.

## Quick Start

```typescript
import { ProxiedmailClient } from '@dotdo/sdk.do/integrations/proxiedmail'

const proxiedmail = new ProxiedmailClient({
  apiKey: process.env.PROXIEDMAIL_API_KEY,
})

const result = await proxiedmail.action.execute({
  action: 'example_action',
  parameters: {},
})
```
