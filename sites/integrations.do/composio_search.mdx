# Composio search Integration

Composio Search provides comprehensive web search across travel (flights, hotels, events), e-commerce (Amazon, Walmart, shopping), financial markets, news, academic research, images, and location services.

## Quick Start

```typescript
import { ComposiosearchClient } from '@dotdo/sdk.do/integrations/composio_search'

const composio_search = new ComposiosearchClient({
  apiKey: process.env.COMPOSIO_SEARCH_API_KEY,
})

const result = await composio_search.action.execute({
  action: 'example_action',
  parameters: {},
})
```
