# Api sports Integration

API-Sports provides comprehensive sports data APIs covering over 2,000 competitions with more than 15 years of historical data, offering real-time updates and easy integration for developers.

## Quick Start

```typescript
import { ApisportsClient } from '@dotdo/sdk.do/integrations/api_sports'

const api_sports = new ApisportsClient({
  apiKey: process.env.API_SPORTS_API_KEY,
})

const result = await api_sports.action.execute({
  action: 'example_action',
  parameters: {},
})
```
