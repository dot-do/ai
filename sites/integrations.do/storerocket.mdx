# Storerocket Integration

StoreRocket provides a customizable and easy-to-install store locator solution for websites, enabling businesses to display their physical locations interactively.

## Quick Start

```typescript
import { StorerocketClient } from '@dotdo/sdk.do/integrations/storerocket'

const storerocket = new StorerocketClient({
  apiKey: process.env.STOREROCKET_API_KEY,
})

const result = await storerocket.action.execute({
  action: 'example_action',
  parameters: {},
})
```
