# Peopledatalabs Integration

PeopleDataLabs provides B2B data enrichment and identity resolution, empowering organizations to build enriched user profiles and validate customer information

## Quick Start

```typescript
import { PeopledatalabsClient } from '@dotdo/sdk.do/integrations/peopledatalabs'

const peopledatalabs = new PeopledatalabsClient({
  apiKey: process.env.PEOPLEDATALABS_API_KEY,
})

const result = await peopledatalabs.action.execute({
  action: 'example_action',
  parameters: {},
})
```
