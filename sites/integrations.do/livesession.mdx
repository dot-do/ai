# Livesession Integration

LiveSession is a session replay and user behavior analytics tool that helps you understand how users interact with your website.

## Quick Start

```typescript
import { LivesessionClient } from '@dotdo/sdk.do/integrations/livesession'

const livesession = new LivesessionClient({
  apiKey: process.env.LIVESESSION_API_KEY,
})

const result = await livesession.action.execute({
  action: 'example_action',
  parameters: {},
})
```
