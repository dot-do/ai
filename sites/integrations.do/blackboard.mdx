# Blackboard Integration

Anthology Adopt powered by Pendo allows institutions to gain insights on Blackboard Learn usage and take action through in-app messages, digital walkthrough guides, and tooltips.

## Quick Start

```typescript
import { BlackboardClient } from '@dotdo/sdk.do/integrations/blackboard'

const blackboard = new BlackboardClient({
  apiKey: process.env.BLACKBOARD_API_KEY,
})

const result = await blackboard.action.execute({
  action: 'example_action',
  parameters: {},
})
```

## Actions

Blackboard provides 314 actions through the Composio API.
