# Gemini Integration

Comprehensive Gemini integration supporting Veo 3 video generation, Gemini Flash text generation (Nano Banana), chat completions, and multimodal AI capabilities via the Google Gemini API.

## Quick Start

```typescript
import { GeminiClient } from '@dotdo/sdk.do/integrations/gemini'

const gemini = new GeminiClient({
  apiKey: process.env.GEMINI_API_KEY,
})

const result = await gemini.action.execute({
  action: 'example_action',
  parameters: {},
})
```

## Actions

Gemini provides 8 actions through the Composio API.
