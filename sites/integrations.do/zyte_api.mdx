# Zyte api Integration

Zyte API is a web scraping solution that automates data extraction from websites, handling challenges like bans and dynamic content.

## Quick Start

```typescript
import { ZyteapiClient } from '@dotdo/sdk.do/integrations/zyte_api'

const zyte_api = new ZyteapiClient({
  apiKey: process.env.ZYTE_API_API_KEY,
})

const result = await zyte_api.action.execute({
  action: 'example_action',
  parameters: {},
})
```
