# Parallel Integration

Parallel offers a Task API for automated, structured web research, transforming natural language queries into precise, schema-compliant outputs.

## Quick Start

```typescript
import { ParallelClient } from '@dotdo/sdk.do/integrations/parallel'

const parallel = new ParallelClient({
  apiKey: process.env.PARALLEL_API_KEY,
})

const result = await parallel.action.execute({
  action: 'example_action',
  parameters: {},
})
```
