# Dialpad Integration

Dialpad is a cloud-based business phone system and contact center platform that enables voice, video, messages and meetings across your existing devices

## Quick Start

```typescript
import { DialpadClient } from '@dotdo/sdk.do/integrations/dialpad'

const dialpad = new DialpadClient({
  apiKey: process.env.DIALPAD_API_KEY,
})

const result = await dialpad.action.execute({
  action: 'example_action',
  parameters: {},
})
```
