# Beamer Integration

Beamer is a platform that enables companies to announce news, updates, and features directly within their applications or websites, enhancing user engagement and communication.

## Quick Start

```typescript
import { BeamerClient } from '@dotdo/sdk.do/integrations/beamer'

const beamer = new BeamerClient({
  apiKey: process.env.BEAMER_API_KEY,
})

const result = await beamer.action.execute({
  action: 'example_action',
  parameters: {},
})
```
