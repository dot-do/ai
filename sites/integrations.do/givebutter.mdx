# Givebutter Integration

Givebutter is a fundraising platform that offers a free, open, and public API for developers to manage campaigns, track donations, and engage with supporters.

## Quick Start

```typescript
import { GivebutterClient } from '@dotdo/sdk.do/integrations/givebutter'

const givebutter = new GivebutterClient({
  apiKey: process.env.GIVEBUTTER_API_KEY,
})

const result = await givebutter.action.execute({
  action: 'example_action',
  parameters: {},
})
```
