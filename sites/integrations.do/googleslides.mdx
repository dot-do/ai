# Google Slides Integration

Google Slides is a cloud-based presentation editor with real-time collaboration, template gallery, and integration with other Google Workspace apps

## Quick Start

```typescript
import { GoogleSlidesClient } from '@dotdo/sdk.do/integrations/googleslides'

const googleslides = new GoogleSlidesClient({
  apiKey: process.env.GOOGLESLIDES_API_KEY,
})

const result = await googleslides.action.execute({
  action: 'example_action',
  parameters: {},
})
```
