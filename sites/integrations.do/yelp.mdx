# Yelp Integration

Yelp Fusion API provides access to business search, reviews, ratings, and local business information with rich data for location-based services

## Quick Start

```typescript
import { YelpClient } from '@dotdo/sdk.do/integrations/yelp'

const yelp = new YelpClient({
  apiKey: process.env.YELP_API_KEY,
})

const result = await yelp.action.execute({
  action: 'example_action',
  parameters: {},
})
```
