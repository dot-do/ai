# Google Sheets Integration

Google Sheets is a cloud-based spreadsheet tool enabling real-time collaboration, data analysis, and integration with other Google Workspace apps

## Quick Start

```typescript
import { GoogleSheetsClient } from '@dotdo/sdk.do/integrations/googlesheets'

const googlesheets = new GoogleSheetsClient({
  apiKey: process.env.GOOGLESHEETS_API_KEY,
})

const result = await googlesheets.action.execute({
  action: 'example_action',
  parameters: {},
})
```
