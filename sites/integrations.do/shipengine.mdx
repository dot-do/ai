# Shipengine Integration

ShipEngine is a REST API that simplifies shipping processes by integrating with multiple carriers, allowing users to manage shipping labels, track packages, and validate addresses efficiently.

## Quick Start

```typescript
import { ShipengineClient } from '@dotdo/sdk.do/integrations/shipengine'

const shipengine = new ShipengineClient({
  apiKey: process.env.SHIPENGINE_API_KEY,
})

const result = await shipengine.action.execute({
  action: 'example_action',
  parameters: {},
})
```

Shipengine provides 14 actions through the Composio API.
