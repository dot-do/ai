# Geocodio Integration

Geocodio provides straightforward and easy-to-use geocoding, reverse geocoding, and data matching services for US and Canadian addresses.

## Quick Start

```typescript
import { GeocodioClient } from '@dotdo/sdk.do/integrations/geocodio'

const geocodio = new GeocodioClient({
  apiKey: process.env.GEOCODIO_API_KEY,
})

const result = await geocodio.action.execute({
  action: 'example_action',
  parameters: {},
})
```
