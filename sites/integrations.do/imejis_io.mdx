# Imejis io Integration

Imejis.io is an API-based image generation platform that enables users to create and customize images seamlessly through a user-friendly interface and a rich library of templates.

## Quick Start

```typescript
import { ImejisioClient } from '@dotdo/sdk.do/integrations/imejis_io'

const imejis_io = new ImejisioClient({
  apiKey: process.env.IMEJIS_IO_API_KEY,
})

const result = await imejis_io.action.execute({
  action: 'example_action',
  parameters: {},
})
```
