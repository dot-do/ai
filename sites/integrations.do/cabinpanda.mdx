# Cabinpanda Integration

## Quick Start

```typescript

const cabinpanda = new CabinpandaClient({
  apiKey: process.env.CABINPANDA_API_KEY,
})

const result = await cabinpanda.action.execute({
  action: 'example_action',
  parameters: {},
})
```
