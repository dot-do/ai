Dropbox is a cloud storage service offering file syncing, sharing, and collaboration across devices with version control and robust integrations

## Quick Start

```typescript
import { DropboxClient } from '@dotdo/sdk.do/integrations/dropbox'

const dropbox = new DropboxClient({
  apiKey: process.env.DROPBOX_API_KEY,
})

const result = await dropbox.action.execute({
  action: 'example_action',
  parameters: {},
})
```
