# Cloudconvert Integration

CloudConvert is a file conversion service supporting over 200 formats, including audio, video, document, ebook, archive, image, spreadsheet, and presentation formats.

## Quick Start

```typescript
import { CloudconvertClient } from '@dotdo/sdk.do/integrations/cloudconvert'

const cloudconvert = new CloudconvertClient({
  apiKey: process.env.CLOUDCONVERT_API_KEY,
})

const result = await cloudconvert.action.execute({
  action: 'example_action',
  parameters: {},
})
```

## Actions

Cloudconvert provides 8 actions through the Composio API.
