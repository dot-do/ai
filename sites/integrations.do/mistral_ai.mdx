# Mistral ai Integration

Mistral AI is a research lab building state-of-the-art open-source language models and providing APIs for developers and enterprises to integrate these models into their applications.

## Quick Start

```typescript
import { MistralaiClient } from '@dotdo/sdk.do/integrations/mistral_ai'

const mistral_ai = new MistralaiClient({
  apiKey: process.env.MISTRAL_AI_API_KEY,
})

const result = await mistral_ai.action.execute({
  action: 'example_action',
  parameters: {},
})
```
