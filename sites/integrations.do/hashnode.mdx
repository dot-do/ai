# Hashnode Integration

Hashnode is a blogging platform tailored for developers, offering tools to create, manage, and publish content seamlessly.

## Quick Start

```typescript
import { HashnodeClient } from '@dotdo/sdk.do/integrations/hashnode'

const hashnode = new HashnodeClient({
  apiKey: process.env.HASHNODE_API_KEY,
})

const result = await hashnode.action.execute({
  action: 'example_action',
  parameters: {},
})
```
