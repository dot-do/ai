# Zenserp Integration

Zenserp is a real-time SERP API that extracts Google Search results in JSON format.

## Quick Start

```typescript
import { ZenserpClient } from '@dotdo/sdk.do/integrations/zenserp'

const zenserp = new ZenserpClient({
  apiKey: process.env.ZENSERP_API_KEY,
})

const result = await zenserp.action.execute({
  action: 'example_action',
  parameters: {},
})
```
