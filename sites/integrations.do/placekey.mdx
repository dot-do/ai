# Placekey Integration

Placekey standardizes location data by assigning unique IDs to physical addresses, simplifying address matching and enabling data sharing across platforms

## Quick Start

```typescript
import { PlacekeyClient } from '@dotdo/sdk.do/integrations/placekey'

const placekey = new PlacekeyClient({
  apiKey: process.env.PLACEKEY_API_KEY,
})

const result = await placekey.action.execute({
  action: 'example_action',
  parameters: {},
})
```
