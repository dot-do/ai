# Sslmate cert spotter api Integration

Cert Spotter API by SSLMate monitors Certificate Transparency logs to alert users about SSL/TLS certificates issued for their domains, helping detect unauthorized certificates and potential security issues.

## Quick Start

```typescript
import { SslmatecertspotterapiClient } from '@dotdo/sdk.do/integrations/sslmate_cert_spotter_api'

const sslmate_cert_spotter_api = new SslmatecertspotterapiClient({
  apiKey: process.env.SSLMATE_CERT_SPOTTER_API_API_KEY,
})

const result = await sslmate_cert_spotter_api.action.execute({
  action: 'example_action',
  parameters: {},
})
```
