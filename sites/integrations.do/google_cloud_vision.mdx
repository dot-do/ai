# Google cloud vision Integration

Google Cloud Vision API enables developers to integrate vision detection features into applications, including image labeling, face and landmark detection, optical character recognition (OCR), and explicit content tagging.

## Quick Start

```typescript
import { GooglecloudvisionClient } from '@dotdo/sdk.do/integrations/google_cloud_vision'

const google_cloud_vision = new GooglecloudvisionClient({
  apiKey: process.env.GOOGLE_CLOUD_VISION_API_KEY,
})

const result = await google_cloud_vision.action.execute({
  action: 'example_action',
  parameters: {},
})
```
