# Serpdog Integration

Serpdog provides APIs for scraping Google search results and related services, enabling developers to access real-time data from Google Search, Maps, Scholar, and more.

## Quick Start

```typescript
import { SerpdogClient } from '@dotdo/sdk.do/integrations/serpdog'

const serpdog = new SerpdogClient({
  apiKey: process.env.SERPDOG_API_KEY,
})

const result = await serpdog.action.execute({
  action: 'example_action',
  parameters: {},
})
```
