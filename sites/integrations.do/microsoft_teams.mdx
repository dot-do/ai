# Microsoft teams Integration

Microsoft Teams integrates chat, video meetings, and file storage within Microsoft 365, providing virtual collaboration and communication for distributed teams

## Quick Start

```typescript
import { MicrosoftteamsClient } from '@dotdo/sdk.do/integrations/microsoft_teams'

const microsoft_teams = new MicrosoftteamsClient({
  apiKey: process.env.MICROSOFT_TEAMS_API_KEY,
})

const result = await microsoft_teams.action.execute({
  action: 'example_action',
  parameters: {},
})
```
