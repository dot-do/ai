# Nasa Integration

NASA offers a suite of APIs providing access to a wide range of data, including Earth science, planetary imagery, and technology transfer information.

## Quick Start

```typescript
import { NasaClient } from '@dotdo/sdk.do/integrations/nasa'

const nasa = new NasaClient({
  apiKey: process.env.NASA_API_KEY,
})

const result = await nasa.action.execute({
  action: 'example_action',
  parameters: {},
})
```
