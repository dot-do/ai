# Composio Integration

Composio enables AI Agents and LLMs to authenticate and integrate with various tools via function calling.

## Quick Start

```typescript
import { ComposioClient } from '@dotdo/sdk.do/integrations/composio'

const composio = new ComposioClient({
  apiKey: process.env.COMPOSIO_API_KEY,
})

const result = await composio.action.execute({
  action: 'example_action',
  parameters: {},
})
```
