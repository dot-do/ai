# Starton Integration

Starton is an all-in-one Web3 API platform that enables developers to deploy and interact with smart contracts, store files on IPFS, and monitor blockchain events through a unified API and frontend interface.

## Quick Start

```typescript
import { StartonClient } from '@dotdo/sdk.do/integrations/starton'

const starton = new StartonClient({
  apiKey: process.env.STARTON_API_KEY,
})

const result = await starton.action.execute({
  action: 'example_action',
  parameters: {},
})
```
