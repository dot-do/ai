# Pexels Integration

Pexels offers a free API that provides access to a vast library of high-quality photos and videos, enabling seamless integration into applications and websites.

## Quick Start

```typescript
import { PexelsClient } from '@dotdo/sdk.do/integrations/pexels'

const pexels = new PexelsClient({
  apiKey: process.env.PEXELS_API_KEY,
})

const result = await pexels.action.execute({
  action: 'example_action',
  parameters: {},
})
```
