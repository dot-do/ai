# Felt Integration

Felt is a modern GIS platform that allows users to create, modify, and share interactive maps, integrating powerful mapping capabilities into various workflows and applications.

## Quick Start

```typescript
import { FeltClient } from '@dotdo/sdk.do/integrations/felt'

const felt = new FeltClient({
  apiKey: process.env.FELT_API_KEY,
})

const result = await felt.action.execute({
  action: 'example_action',
  parameters: {},
})
```
