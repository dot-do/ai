# Plisio Integration

Plisio is a cryptocurrency payment gateway that enables businesses to accept payments in over 15 cryptocurrencies, including Bitcoin, Ethereum, and Litecoin.

## Quick Start

```typescript
import { PlisioClient } from '@dotdo/sdk.do/integrations/plisio'

const plisio = new PlisioClient({
  apiKey: process.env.PLISIO_API_KEY,
})

const result = await plisio.action.execute({
  action: 'example_action',
  parameters: {},
})
```
