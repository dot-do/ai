# Foursquare Integration

Search for places and place recommendations from the Foursquare Places database

## Quick Start

```typescript
import { FoursquareClient } from '@dotdo/sdk.do/integrations/foursquare'

const foursquare = new FoursquareClient({
  apiKey: process.env.FOURSQUARE_API_KEY,
})

const result = await foursquare.action.execute({
  action: 'example_action',
  parameters: {},
})
```
