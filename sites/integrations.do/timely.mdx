# Timely Integration

Timely is an automatic time-tracking platform capturing activity across applications, calendars, and devices, creating detailed timesheets for billing or productivity insights

## Quick Start

```typescript
import { TimelyClient } from '@dotdo/sdk.do/integrations/timely'

const timely = new TimelyClient({
  apiKey: process.env.TIMELY_API_KEY,
})

const result = await timely.action.execute({
  action: 'example_action',
  parameters: {},
})
```
