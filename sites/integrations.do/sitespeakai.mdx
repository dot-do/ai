# Sitespeakai Integration

SiteSpeakAI enables businesses to automate customer support by creating custom-trained, embeddable GPT chatbots that provide real-time answers about products and services, reducing support tickets.

## Quick Start

```typescript
import { SitespeakaiClient } from '@dotdo/sdk.do/integrations/sitespeakai'

const sitespeakai = new SitespeakaiClient({
  apiKey: process.env.SITESPEAKAI_API_KEY,
})

const result = await sitespeakai.action.execute({
  action: 'example_action',
  parameters: {},
})
```
