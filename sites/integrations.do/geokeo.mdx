# Geokeo Integration

Geokeo provides geocoding services, converting addresses to coordinates and vice versa, with a free tier offering 2,500 daily API requests.

## Quick Start

```typescript
import { GeokeoClient } from '@dotdo/sdk.do/integrations/geokeo'

const geokeo = new GeokeoClient({
  apiKey: process.env.GEOKEO_API_KEY,
})

const result = await geokeo.action.execute({
  action: 'example_action',
  parameters: {},
})
```

## Actions

Geokeo provides 2 actions through the Composio API.
