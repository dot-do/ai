# Linkedin Integration

LinkedIn is a professional networking platform enabling job seekers, companies, and thought leaders to connect, share content, and discover business opportunities

## Quick Start

```typescript
import { LinkedinClient } from '@dotdo/sdk.do/integrations/linkedin'

const linkedin = new LinkedinClient({
  apiKey: process.env.LINKEDIN_API_KEY,
})

const result = await linkedin.action.execute({
  action: 'example_action',
  parameters: {},
})
```
