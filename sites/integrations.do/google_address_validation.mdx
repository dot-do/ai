# Google address validation Integration

The Address Validation API allows developers to verify the accuracy of addresses by validating and standardizing them, and determining their geocode.

## Quick Start

```typescript
import { GoogleaddressvalidationClient } from '@dotdo/sdk.do/integrations/google_address_validation'

const google_address_validation = new GoogleaddressvalidationClient({
  apiKey: process.env.GOOGLE_ADDRESS_VALIDATION_API_KEY,
})

const result = await google_address_validation.action.execute({
  action: 'example_action',
  parameters: {},
})
```
