# Reply Integration

Reply.io is a sales engagement platform that automates multichannel outreach, enabling users to create and manage email campaigns, track replies, and monitor performance directly within their platform.

## Quick Start

```typescript
import { ReplyClient } from '@dotdo/sdk.do/integrations/reply'

const reply = new ReplyClient({
  apiKey: process.env.REPLY_API_KEY,
})

const result = await reply.action.execute({
  action: 'example_action',
  parameters: {},
})
```
