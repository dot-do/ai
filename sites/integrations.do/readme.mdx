# integrations.do

**Pre-built external service integrations for Business-as-Code**

integrations.do provides a unified platform for connecting your Business-as-Code applications with external services. Access pre-built integrations for CRM systems (Salesforce, HubSpot), payment processors (Stripe, PayPal), communication platforms (Slack, SendGrid), and storage services (S3, Dropbox) through semantic `$.Subject.predicate.Object` patterns.

## Quick Start

```typescript
import { $, integrations } from 'sdk.do'

// Initialize Salesforce integration
const salesforce = await integrations.use($.Salesforce, {
  clientId: process.env.SALESFORCE_CLIENT_ID,
  clientSecret: process.env.SALESFORCE_CLIENT_SECRET,
  instanceUrl: process.env.SALESFORCE_INSTANCE_URL,
})

// Create a lead in Salesforce
const lead = await salesforce.create($.Lead, {
  $type: 'Lead',
  firstName: 'John',
  lastName: 'Smith',
  company: 'Acme Corp',
  email: 'john@acme.com',
  status: 'Open',
})

// Use Stripe for payments
const stripe = await integrations.use($.Stripe, {
  apiKey: process.env.STRIPE_API_KEY,
})

// Create payment
const payment = await stripe.create($.PaymentIntent, {
  amount: 5000, // $50.00
  currency: 'usd',
  customer: customerId,
})
```

### Bi-Directional Sync

Keep data in sync between your database and external services:

```typescript
import { $, integrations, db, on } from 'sdk.do'

const salesforce = await integrations.use($.Salesforce)

// Sync to external service
await integrations.sync({
  from: db,
  to: salesforce,
  type: $.Organization,
  mapping: {
    name: 'Name',
    url: 'Website',
    industry: 'Industry',
  },
})

// Sync from external service
await integrations.sync({
  from: salesforce,
  to: db,
  type: $.Account,
  mapping: {
    Name: 'name',
    Website: 'url',
    Industry: 'industry',
  },
})

// Real-time sync with webhooks
on($.Account.updated, async (event) => {
  // Salesforce webhook received
  const account = event.data

  await db.update(account.externalId, {
    name: account.Name,
    url: account.Website,
  })
})
```

## Use Cases

### CRM Data Sync

Automatically sync customers and organizations to your CRM:

```typescript
import { $, integrations, db, on } from 'sdk.do'

const salesforce = await integrations.use($.Salesforce)

// Sync new organizations to Salesforce
on($.Organization.created, async (event) => {
  const org = event.data

  const account = await salesforce.create($.Account, {
    $type: 'Account',
    name: org.name,
    website: org.url,
    industry: org.industry,
    externalId: org.$id,
  })

  // Store Salesforce ID
  await db.update(org.$id, {
    salesforceAccountId: account.id,
  })
})

// Sync updates back to database
on($.Account.updated, async (event) => {
  const account = event.data

  const org = await db.get($.Organization, {
    where: { salesforceAccountId: account.id },
  })

  await db.update(org.$id, {
    name: account.Name,
    url: account.Website,
    industry: account.Industry,
  })
})
```

### Payment Processing

Process payments with Stripe integration:

```typescript
import { $, integrations, db, on } from 'sdk.do'

const stripe = await integrations.use($.Stripe)

// Process order payment
on($.Order.created, async (event) => {
  const order = event.data

  // Create Stripe customer if needed
  let customer = order.customer.stripeCustomerId
  if (!customer) {
    const stripeCustomer = await stripe.create($.Customer, {
      email: order.customer.email,
      name: order.customer.name,
      metadata: {
        customerId: order.customer.$id,
      },
    })

    customer = stripeCustomer.id

    await db.update(order.customer.$id, {
      stripeCustomerId: customer,
    })
  }

  // Create payment intent
  const paymentIntent = await stripe.create($.PaymentIntent, {
    amount: order.totalPrice * 100, // cents
    currency: 'usd',
    customer,
    metadata: {
      orderId: order.$id,
    },
  })

  // Update order with payment intent
  await db.update(order.$id, {
    stripePaymentIntentId: paymentIntent.id,
    paymentStatus: 'pending',
  })
})

// Handle payment success
on($.PaymentIntent.succeeded, async (event) => {
  const paymentIntent = event.data

  const order = await db.get($.Order, {
    where: { stripePaymentIntentId: paymentIntent.id },
  })

  await db.update(order.$id, {
    paymentStatus: 'paid',
    paidDate: new Date(),
  })
})
```

## Best Practices

### 1. Store Credentials Securely

Always use environment variables for API keys and secrets:

```typescript
// Good: Environment variables
const stripe = await integrations.use($.Stripe, {
  apiKey: process.env.STRIPE_API_KEY,
})

// Avoid: Hardcoded credentials
const stripe = await integrations.use($.Stripe, {
  apiKey: 'sk_live_hardcoded_key',
})
```

### 2. Handle OAuth Flows Properly

Implement proper OAuth flows with state validation:

```typescript
import { integrations } from 'sdk.do'

// Generate state token
const state = crypto.randomUUID()

// Store state in session
session.oauthState = state

// Generate auth URL with state
const authUrl = await integrations.oauth($.Salesforce, {
  clientId: process.env.SALESFORCE_CLIENT_ID,
  redirectUri: 'https://example.com/oauth/callback',
  state,
})

// In callback, validate state
if (request.query.state !== session.oauthState) {
  throw new Error('Invalid OAuth state')
}
```

### 3. Implement Error Handling

Always handle integration errors gracefully:

```typescript
try {
  const account = await salesforce.create($.Account, accountData)
} catch (error) {
  if (error.code === 'DUPLICATE_VALUE') {
    // Handle duplicate
  } else if (error.code === 'REQUIRED_FIELD_MISSING') {
    // Handle missing field
  } else {
    // Log and handle unknown error
    console.error('Salesforce error:', error)
  }
}
```

### 4. Use Schema Mapping

Map data between Schema.org types and service-specific types:

```typescript
const salesforce = await integrations.use($.Salesforce)

// Define mapping
await salesforce.configure({
  mapping: {
    [$.Person]: 'Contact',
    [$.Organization]: 'Account',
    [$.Opportunity]: 'Opportunity',
  },
  fieldMapping: {
    [$.Person]: {
      name: 'Name',
      email: 'Email',
      telephone: 'Phone',
    },
  },
})
```

### 5. Implement Retry Logic

Use retry logic for transient failures:

```typescript
const result = await integrations.execute(operation, {
  retry: {
    attempts: 3,
    delay: 1000,
    backoff: 'exponential',
  },
})
```
