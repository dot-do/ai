# Codeinterpreter Integration

CodeInterpreter extends Python-based coding environments with integrated data analysis, enabling developers to run scripts, visualize results, and prototype solutions inside supported platforms

## Quick Start

```typescript
import { CodeinterpreterClient } from '@dotdo/sdk.do/integrations/codeinterpreter'

const codeinterpreter = new CodeinterpreterClient({
  apiKey: process.env.CODEINTERPRETER_API_KEY,
})

const result = await codeinterpreter.action.execute({
  action: 'example_action',
  parameters: {},
})
```
