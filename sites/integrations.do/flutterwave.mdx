# Flutterwave Integration

Flutterwave is a payments technology company that provides payment infrastructure and APIs enabling businesses to accept and send payments in Africa and globally.

## Quick Start

```typescript
import { FlutterwaveClient } from '@dotdo/sdk.do/integrations/flutterwave'

const flutterwave = new FlutterwaveClient({
  apiKey: process.env.FLUTTERWAVE_API_KEY,
})

const result = await flutterwave.action.execute({
  action: 'example_action',
  parameters: {},
})
```
