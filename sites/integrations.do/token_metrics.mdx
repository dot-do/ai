# Token metrics Integration

Token Metrics provides an API offering real-time, AI-powered cryptocurrency data and insights for developers to build trading bots, dashboards, and portfolio tools.

## Quick Start

```typescript
import { TokenmetricsClient } from '@dotdo/sdk.do/integrations/token_metrics'

const token_metrics = new TokenmetricsClient({
  apiKey: process.env.TOKEN_METRICS_API_KEY,
})

const result = await token_metrics.action.execute({
  action: 'example_action',
  parameters: {},
})
```
