# ActiveCampaign Integration

Marketing automation and CRM platform for email marketing and customer engagement.

## Quick Start

```typescript
import { ActiveCampaignClient } from '@dotdo/sdk.do/integrations/activecampaign'

const activecampaign = new ActiveCampaignClient({
  apiKey: process.env.ACTIVECAMPAIGN_API_KEY,
  accountName: 'youraccountname',
})

const contact = await activecampaign.contact.create({
  email: 'customer@example.com',
  firstName: 'John',
  lastName: 'Doe',
})
```
