# Postgrid verify Integration

PostGrid Verify is an API that allows you to autocomplete, verify, and standardize addresses in real-time, supporting both individual and batch address verification.

## Quick Start

```typescript
import { PostgridverifyClient } from '@dotdo/sdk.do/integrations/postgrid_verify'

const postgrid_verify = new PostgridverifyClient({
  apiKey: process.env.POSTGRID_VERIFY_API_KEY,
})

const result = await postgrid_verify.action.execute({
  action: 'example_action',
  parameters: {},
})
```

## Actions

Postgrid verify provides 2 actions through the Composio API.
