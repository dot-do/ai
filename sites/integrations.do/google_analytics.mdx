# Google Analytics Integration

Google Analytics tracks and reports website traffic, user behavior, and conversion data, enabling marketers to optimize online performance and customer journeys

## Quick Start

```typescript
import { GoogleAnalyticsClient } from '@dotdo/sdk.do/integrations/google_analytics'

const google_analytics = new GoogleAnalyticsClient({
  apiKey: process.env.GOOGLE_ANALYTICS_API_KEY,
})

const result = await google_analytics.action.execute({
  action: 'example_action',
  parameters: {},
})
```
