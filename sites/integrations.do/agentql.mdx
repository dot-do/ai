# Agentql Integration

AgentQL is a suite of tools designed to connect AI agents to the web, enabling web interaction and structured data extraction through a specialized query language.

## Quick Start

```typescript
import { AgentqlClient } from '@dotdo/sdk.do/integrations/agentql'

const agentql = new AgentqlClient({
  apiKey: process.env.AGENTQL_API_KEY,
})

const result = await agentql.action.execute({
  action: 'example_action',
  parameters: {},
})
```
