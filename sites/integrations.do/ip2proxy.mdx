# Ip2proxy Integration

IP2Proxy is a proxy detection web service that identifies anonymous proxies, VPNs, TOR exit nodes, search engine robots, and residential proxies by IP address.

## Quick Start

```typescript
import { Ip2proxyClient } from '@dotdo/sdk.do/integrations/ip2proxy'

const ip2proxy = new Ip2proxyClient({
  apiKey: process.env.IP2PROXY_API_KEY,
})

const result = await ip2proxy.action.execute({
  action: 'example_action',
  parameters: {},
})
```

Ip2proxy provides 1 actions through the Composio API.
