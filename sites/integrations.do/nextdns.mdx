# Nextdns Integration

NextDNS is a DNS service that enhances internet security and privacy by blocking malicious websites, tracking scripts, and ads before they reach your device.

## Quick Start

```typescript

const nextdns = new NextdnsClient({
  apiKey: process.env.NEXTDNS_API_KEY,
})

const result = await nextdns.action.execute({
  action: 'example_action',
  parameters: {},
})
```
