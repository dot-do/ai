# Vestaboard Integration

Vestaboard is a smart messaging display that allows users to send and receive messages through a mechanical split-flap display.

## Quick Start

```typescript
import { VestaboardClient } from '@dotdo/sdk.do/integrations/vestaboard'

const vestaboard = new VestaboardClient({
  apiKey: process.env.VESTABOARD_API_KEY,
})

const result = await vestaboard.action.execute({
  action: 'example_action',
  parameters: {},
})
```
