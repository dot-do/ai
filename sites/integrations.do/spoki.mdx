# Spoki Integration

Spoki is an Italian platform that integrates WhatsApp's official APIs to automate communication for marketing, sales, customer support, and payments.

## Quick Start

```typescript
import { SpokiClient } from '@dotdo/sdk.do/integrations/spoki'

const spoki = new SpokiClient({
  apiKey: process.env.SPOKI_API_KEY,
})

const result = await spoki.action.execute({
  action: 'example_action',
  parameters: {},
})
```
