# Ayrshare Integration

Ayrshare provides a Social Media API that enables developers to programmatically manage and automate social media posts, analytics, and interactions across multiple platforms.

## Quick Start

```typescript
import { AyrshareClient } from '@dotdo/sdk.do/integrations/ayrshare'

const ayrshare = new AyrshareClient({
  apiKey: process.env.AYRSHARE_API_KEY,
})

const result = await ayrshare.action.execute({
  action: 'example_action',
  parameters: {},
})
```
