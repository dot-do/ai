# Giphy Integration

GIPHY is the first and largest GIF search engine, offering an extensive library of GIFs and Stickers for integration into applications.

## Quick Start

```typescript
import { GiphyClient } from '@dotdo/sdk.do/integrations/giphy'

const giphy = new GiphyClient({
  apiKey: process.env.GIPHY_API_KEY,
})

const result = await giphy.action.execute({
  action: 'example_action',
  parameters: {},
})
```
