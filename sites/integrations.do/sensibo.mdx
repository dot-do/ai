# Sensibo Integration

Sensibo offers smart controllers that connect air conditioners and heat pumps to the internet, enabling remote control and automation.

## Quick Start

```typescript
import { SensiboClient } from '@dotdo/sdk.do/integrations/sensibo'

const sensibo = new SensiboClient({
  apiKey: process.env.SENSIBO_API_KEY,
})

const result = await sensibo.action.execute({
  action: 'example_action',
  parameters: {},
})
```
