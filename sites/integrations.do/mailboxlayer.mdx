# Mailboxlayer Integration

A simple REST-based JSON API for email address validation and verification.

## Quick Start

```typescript
import { MailboxlayerClient } from '@dotdo/sdk.do/integrations/mailboxlayer'

const mailboxlayer = new MailboxlayerClient({
  apiKey: process.env.MAILBOXLAYER_API_KEY,
})

const result = await mailboxlayer.action.execute({
  action: 'example_action',
  parameters: {},
})
```
