# Algolia Integration

Algolia is a hosted search API that provides developers with tools to build fast and relevant search experiences for their applications.

## Quick Start

```typescript
import { AlgoliaClient } from '@dotdo/sdk.do/integrations/algolia'

const algolia = new AlgoliaClient({
  apiKey: process.env.ALGOLIA_API_KEY,
})

const result = await algolia.action.execute({
  action: 'example_action',
  parameters: {},
})
```

Algolia provides 30 actions through the Composio API.
