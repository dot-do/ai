# Metaphor Integration

Metaphor is a neural search engine that enables developers to perform contextually relevant searches by understanding how people describe and share content online.

## Quick Start

```typescript
import { MetaphorClient } from '@dotdo/sdk.do/integrations/metaphor'

const metaphor = new MetaphorClient({
  apiKey: process.env.METAPHOR_API_KEY,
})

const result = await metaphor.action.execute({
  action: 'example_action',
  parameters: {},
})
```
