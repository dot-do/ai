# Bart Integration

BART (Bay Area Rapid Transit) provides public transportation services in the San Francisco Bay Area.

## Quick Start

```typescript
import { BartClient } from '@dotdo/sdk.do/integrations/bart'

const bart = new BartClient({
  apiKey: process.env.BART_API_KEY,
})

const result = await bart.action.execute({
  action: 'example_action',
  parameters: {},
})
```
