# Customerio Integration

Customer.io is a customer engagement platform that enables businesses to send targeted messages to their customers through various channels, including email, SMS, and push notifications.

## Quick Start

```typescript
import { CustomerioClient } from '@dotdo/sdk.do/integrations/customerio'

const customerio = new CustomerioClient({
  apiKey: process.env.CUSTOMERIO_API_KEY,
})

const result = await customerio.action.execute({
  action: 'example_action',
  parameters: {},
})
```
