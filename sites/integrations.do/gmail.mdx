# Gmail Integration

Gmail is Google’s email service, featuring spam protection, search functions, and seamless integration with other G Suite apps for productivity

## Quick Start

```typescript
import { GmailClient } from '@dotdo/sdk.do/integrations/gmail'

const gmail = new GmailClient({
  apiKey: process.env.GMAIL_API_KEY,
})

const result = await gmail.action.execute({
  action: 'example_action',
  parameters: {},
})
```
