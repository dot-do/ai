# Freshbooks Integration

FreshBooks is a cloud-based accounting software service designed for small and medium-sized businesses, offering features like invoicing, expense tracking, and time management.

## Quick Start

```typescript
import { FreshbooksClient } from '@dotdo/sdk.do/integrations/freshbooks'

const freshbooks = new FreshbooksClient({
  apiKey: process.env.FRESHBOOKS_API_KEY,
})

const result = await freshbooks.action.execute({
  action: 'example_action',
  parameters: {},
})
```
