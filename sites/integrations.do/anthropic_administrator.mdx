# Anthropic administrator Integration

The Anthropic Admin API allows programmatic management of organizational resources, including members, workspaces, and API keys.

## Quick Start

```typescript
import { AnthropicadministratorClient } from '@dotdo/sdk.do/integrations/anthropic_administrator'

const anthropic_administrator = new AnthropicadministratorClient({
  apiKey: process.env.ANTHROPIC_ADMINISTRATOR_API_KEY,
})

const result = await anthropic_administrator.action.execute({
  action: 'example_action',
  parameters: {},
})
```
