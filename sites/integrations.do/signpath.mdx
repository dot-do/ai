# Signpath Integration

SignPath is a code signing service that integrates with your build system to automate the signing of your software artifacts, ensuring their authenticity and integrity.

## Quick Start

```typescript
import { SignpathClient } from '@dotdo/sdk.do/integrations/signpath'

const signpath = new SignpathClient({
  apiKey: process.env.SIGNPATH_API_KEY,
})

const result = await signpath.action.execute({
  action: 'example_action',
  parameters: {},
})
```

Signpath provides 4 actions through the Composio API.
