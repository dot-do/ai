# Serpapi Integration

SerpApi provides a real-time API for structured search engine results, allowing developers to scrape, parse, and analyze SERP data for SEO and research

## Quick Start

```typescript
import { SerpapiClient } from '@dotdo/sdk.do/integrations/serpapi'

const serpapi = new SerpapiClient({
  apiKey: process.env.SERPAPI_API_KEY,
})

const result = await serpapi.action.execute({
  action: 'example_action',
  parameters: {},
})
```
