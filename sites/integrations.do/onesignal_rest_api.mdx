# Onesignal rest api Integration

The OneSignal REST API enables developers to programmatically send push notifications, emails, and SMS, manage users and subscriptions, and configure apps.

## Quick Start

```typescript
import { OnesignalrestapiClient } from '@dotdo/sdk.do/integrations/onesignal_rest_api'

const onesignal_rest_api = new OnesignalrestapiClient({
  apiKey: process.env.ONESIGNAL_REST_API_API_KEY,
})

const result = await onesignal_rest_api.action.execute({
  action: 'example_action',
  parameters: {},
})
```
