# Klaviyo Integration

Klaviyo is a data-driven email and SMS marketing platform that allows e-commerce brands to deliver targeted messages, track conversions, and scale customer relationships

## Quick Start

```typescript
import { KlaviyoClient } from '@dotdo/sdk.do/integrations/klaviyo'

const klaviyo = new KlaviyoClient({
  apiKey: process.env.KLAVIYO_API_KEY,
})

const result = await klaviyo.action.execute({
  action: 'example_action',
  parameters: {},
})
```
