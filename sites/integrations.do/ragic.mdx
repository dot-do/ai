# Ragic Integration

Ragic is a web-based database builder that allows users to create and manage databases as easily as creating a spreadsheet.

## Quick Start

```typescript
import { RagicClient } from '@dotdo/sdk.do/integrations/ragic'

const ragic = new RagicClient({
  apiKey: process.env.RAGIC_API_KEY,
})

const result = await ragic.action.execute({
  action: 'example_action',
  parameters: {},
})
```
