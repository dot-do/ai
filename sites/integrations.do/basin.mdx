# Basin Integration

Basin is a no-code form backend that enables users to set up powerful, reliable forms quickly without writing server-side code.

## Quick Start

```typescript
import { BasinClient } from '@dotdo/sdk.do/integrations/basin'

const basin = new BasinClient({
  apiKey: process.env.BASIN_API_KEY,
})

const result = await basin.action.execute({
  action: 'example_action',
  parameters: {},
})
```
