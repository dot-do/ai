# Agenty Integration

Agenty is a web scraping and automation platform that enables users to extract data, monitor changes, and automate browser tasks without coding.

## Quick Start

```typescript
import { AgentyClient } from '@dotdo/sdk.do/integrations/agenty'

const agenty = new AgentyClient({
  apiKey: process.env.AGENTY_API_KEY,
})

const result = await agenty.action.execute({
  action: 'example_action',
  parameters: {},
})
```
