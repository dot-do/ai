# Taggun Integration

Taggun provides a real-time, high-accuracy receipt OCR API for extracting structured data from receipt images.

## Quick Start

```typescript
import { TaggunClient } from '@dotdo/sdk.do/integrations/taggun'

const taggun = new TaggunClient({
  apiKey: process.env.TAGGUN_API_KEY,
})

const result = await taggun.action.execute({
  action: 'example_action',
  parameters: {},
})
```
