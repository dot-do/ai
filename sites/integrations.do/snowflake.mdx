# Snowflake Integration

Snowflake is a cloud-based data warehouse offering elastic scaling, secure data sharing, and SQL analytics across multiple cloud environments

## Quick Start

```typescript
import { SnowflakeClient } from '@dotdo/sdk.do/integrations/snowflake'

const snowflake = new SnowflakeClient({
  apiKey: process.env.SNOWFLAKE_API_KEY,
})

const result = await snowflake.action.execute({
  action: 'example_action',
  parameters: {},
})
```
