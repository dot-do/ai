# Typeform Integration

Online form builder for surveys, quizzes, and data collection.

## Features

- **Form Management**: Create, update, delete, and customize forms
- **Response Retrieval**: Access and manage form submissions
- **Webhooks**: Real-time notifications for new responses
- **Themes**: Customize form appearance with themes
- **API Key Authentication**: Simple personal access token authentication
- **Automatic Pagination**: SDK handles pagination automatically

## Quick Start

```typescript
import { TypeformClient } from '@dotdo/sdk.do/integrations/typeform'

const typeform = new TypeformClient({
  apiKey: process.env.TYPEFORM_API_KEY,
})

// Create a form
const form = await typeform.form.create({
  title: 'Customer Satisfaction Survey',
  fields: [
    {
      type: 'short_text',
      title: 'What is your name?',
      required: true,
    },
    {
      type: 'rating',
      title: 'How satisfied are you?',
      properties: { steps: 5 },
    },
  ],
})

// Get responses
const responses = await typeform.response.list({
  form_id: form.id,
  page_size: 100,
})
```

## Resources

### Forms

Create and manage forms:

```typescript
// Create form
const form = await typeform.form.create({
  title: 'My Survey',
  fields: [
    {
      type: 'short_text',
      title: 'Your name?',
      required: true,
    },
  ],
  settings: {
    is_public: true,
    show_progress_bar: true,
  },
})

// Get form
const form = await typeform.form.get({
  form_id: 'abc123',
})

// Update form
await typeform.form.update({
  form_id: 'abc123',
  title: 'Updated Survey Title',
})

// List forms
const forms = await typeform.form.list({
  page: 1,
  page_size: 20,
})

// Delete form
await typeform.form.delete({
  form_id: 'abc123',
})
```

### Responses

Retrieve form submissions:

```typescript
// List responses
const responses = await typeform.response.list({
  form_id: 'abc123',
  page_size: 100,
  since: '2025-01-01T00:00:00Z',
  until: '2025-12-31T23:59:59Z',
})

// Get specific response
const response = await typeform.response.get({
  form_id: 'abc123',
  response_id: 'resp123',
})

// Delete responses
await typeform.response.delete({
  form_id: 'abc123',
  included_response_ids: ['resp1', 'resp2'],
})
```

## Webhooks

Typeform sends webhook events for form responses:

```typescript
import { TypeformWebhookHandler, WebhookEventRouter } from '@dotdo/sdk.do/integrations/typeform'

const handler = new TypeformWebhookHandler(process.env.TYPEFORM_WEBHOOK_SECRET)
const event = await handler.handleRequest(request)

const router = new WebhookEventRouter()
router.onFormResponse(async (event) => {
  console.log('New response:', event.data)
  console.log('Form ID:', event.data.form_id)
  console.log('Response ID:', event.data.response_id)
  console.log('Answers:', event.data.answers)
})

await router.route(event)
```

**Important**: Typeform signature includes a trailing newline character (`\n`) in the calculation.

## Best Practices

1. **Rate Limits**: Maximum 2 requests per second per account
2. **Response Pagination**: Use `page_size` up to 1000, then use `since`/`until` or `after`/`before` for large datasets
3. **Webhook Secrets**: Always set a secret for signature verification
4. **Token Security**: Keep personal access tokens secure, never commit to version control
5. **Scope Selection**: Only request scopes your application needs
6. **Error Handling**: Handle 429 rate limit errors with exponential backoff
7. **Large Response Sets**: For >1000 responses, use date filters (`since`/`until`) to narrow requests

## Rate Limiting

- **Create API**: 2 requests/second per account
- **Responses API**: 2 requests/second per account
- Exceeding limits returns 429 status code
- Use `Retry-After` header for retry timing
