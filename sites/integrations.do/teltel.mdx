# Teltel Integration

TelTel is a telecom operator and software provider offering tools for automated voice and SMS communication, including contact center software, Voice/SMS API, and VoIP/SIP trunking.

## Quick Start

```typescript
import { TeltelClient } from '@dotdo/sdk.do/integrations/teltel'

const teltel = new TeltelClient({
  apiKey: process.env.TELTEL_API_KEY,
})

const result = await teltel.action.execute({
  action: 'example_action',
  parameters: {},
})
```
