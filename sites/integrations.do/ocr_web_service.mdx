# Ocr web service Integration

OCR Web Service provides SOAP and REST APIs for integrating Optical Character Recognition (OCR) technology into software products, mobile devices, or other web services.

## Quick Start

```typescript
import { OcrwebserviceClient } from '@dotdo/sdk.do/integrations/ocr_web_service'

const ocr_web_service = new OcrwebserviceClient({
  apiKey: process.env.OCR_WEB_SERVICE_API_KEY,
})

const result = await ocr_web_service.action.execute({
  action: 'example_action',
  parameters: {},
})
```

Ocr web service provides 4 actions through the Composio API.
