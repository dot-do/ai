# Sendgrid Integration

SendGrid is a cloud-based email delivery platform providing transactional and marketing email services, with APIs for integration, analytics, and scalability

## Quick Start

```typescript
import { SendgridClient } from '@dotdo/sdk.do/integrations/sendgrid'

const sendgrid = new SendgridClient({
  apiKey: process.env.SENDGRID_API_KEY,
})

const result = await sendgrid.action.execute({
  action: 'example_action',
  parameters: {},
})
```
