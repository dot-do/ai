# Simplesat Integration

Simplesat captures customer feedback and CSAT scores through surveys, integrating directly with helpdesk systems for real-time performance insights

## Quick Start

```typescript
import { SimplesatClient } from '@dotdo/sdk.do/integrations/simplesat'

const simplesat = new SimplesatClient({
  apiKey: process.env.SIMPLESAT_API_KEY,
})

const result = await simplesat.action.execute({
  action: 'example_action',
  parameters: {},
})
```
