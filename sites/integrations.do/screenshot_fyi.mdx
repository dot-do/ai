# Screenshot fyi Integration

Take stunning, premium quality screenshots of any website with one API call.

## Quick Start

```typescript

const screenshot_fyi = new ScreenshotfyiClient({
  apiKey: process.env.SCREENSHOT_FYI_API_KEY,
})

const result = await screenshot_fyi.action.execute({
  action: 'example_action',
  parameters: {},
})
```
