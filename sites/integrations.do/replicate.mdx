# Replicate Integration

Replicate allows users to run AI models via a cloud API without managing infrastructure.

## Quick Start

```typescript
import { ReplicateClient } from '@dotdo/sdk.do/integrations/replicate'

const replicate = new ReplicateClient({
  apiKey: process.env.REPLICATE_API_KEY,
})

const result = await replicate.action.execute({
  action: 'example_action',
  parameters: {},
})
```
