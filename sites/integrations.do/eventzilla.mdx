# Eventzilla Integration

Eventzilla is an event management platform that provides tools for creating, promoting, and managing events, including ticketing, registration, and attendee management.

## Quick Start

```typescript
import { EventzillaClient } from '@dotdo/sdk.do/integrations/eventzilla'

const eventzilla = new EventzillaClient({
  apiKey: process.env.EVENTZILLA_API_KEY,
})

const result = await eventzilla.action.execute({
  action: 'example_action',
  parameters: {},
})
```
