TimeCamp is a time tracking solution designed to help businesses of all sizes track time for projects to maximize their profits.

## Quick Start

```typescript

const timecamp = new TimecampClient({
  apiKey: process.env.TIMECAMP_API_KEY,
})

const result = await timecamp.action.execute({
  action: 'example_action',
  parameters: {},
})
```
