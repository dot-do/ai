## Quick Start

```typescript
import { Auth0Client } from '@dotdo/sdk.do/integrations/auth0'

const auth0 = new Auth0Client({
  apiKey: process.env.AUTH0_API_KEY,
})

const result = await auth0.action.execute({
  action: 'example_action',
  parameters: {},
})
```
