# Pingdom Integration

Pingdom is a web performance monitoring service that allows users to monitor the uptime and performance of websites, servers, and applications.

## Quick Start

```typescript
import { PingdomClient } from '@dotdo/sdk.do/integrations/pingdom'

const pingdom = new PingdomClient({
  apiKey: process.env.PINGDOM_API_KEY,
})

const result = await pingdom.action.execute({
  action: 'example_action',
  parameters: {},
})
```
