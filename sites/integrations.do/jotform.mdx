# Jotform Integration

Jotform is an online form builder that allows users to create and manage forms for various purposes, including data collection, surveys, and more.

## Quick Start

```typescript
import { JotformClient } from '@dotdo/sdk.do/integrations/jotform'

const jotform = new JotformClient({
  apiKey: process.env.JOTFORM_API_KEY,
})

const result = await jotform.action.execute({
  action: 'example_action',
  parameters: {},
})
```
