# Signaturely Integration

Signaturely is an electronic signature platform that allows users to sign documents online, offering features like automated signature requests, templates, and integrations with other platforms.

## Quick Start

```typescript
import { SignaturelyClient } from '@dotdo/sdk.do/integrations/signaturely'

const signaturely = new SignaturelyClient({
  apiKey: process.env.SIGNATURELY_API_KEY,
})

const result = await signaturely.action.execute({
  action: 'example_action',
  parameters: {},
})
```
