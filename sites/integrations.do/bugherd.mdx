# Bugherd Integration

BugHerd is a visual feedback and bug tracking tool for websites, allowing clients and teams to report issues directly on live sites.

## Quick Start

```typescript
import { BugherdClient } from '@dotdo/sdk.do/integrations/bugherd'

const bugherd = new BugherdClient({
  apiKey: process.env.BUGHERD_API_KEY,
})

const result = await bugherd.action.execute({
  action: 'example_action',
  parameters: {},
})
```
