# Botsonic Integration

Botsonic is a no-code AI chatbot builder that enables users to create and integrate AI chatbots into their websites without any coding knowledge.

## Quick Start

```typescript
import { BotsonicClient } from '@dotdo/sdk.do/integrations/botsonic'

const botsonic = new BotsonicClient({
  apiKey: process.env.BOTSONIC_API_KEY,
})

const result = await botsonic.action.execute({
  action: 'example_action',
  parameters: {},
})
```
