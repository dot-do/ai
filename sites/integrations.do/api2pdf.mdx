# Api2pdf Integration

Api2Pdf is a REST API that enables developers to generate PDFs from HTML, URLs, and various document formats using engines like wkhtmltopdf, Headless Chrome, and LibreOffice.

## Quick Start

```typescript
import { Api2pdfClient } from '@dotdo/sdk.do/integrations/api2pdf'

const api2pdf = new Api2pdfClient({
  apiKey: process.env.API2PDF_API_KEY,
})

const result = await api2pdf.action.execute({
  action: 'example_action',
  parameters: {},
})
```
