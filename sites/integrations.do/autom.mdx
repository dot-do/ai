# Autom Integration

Autom is a service that delivers lightning-fast search engine results page (SERP) outcomes for Google, Bing, and Brave, offering developers rapid access to search data with minimal latency.

## Quick Start

```typescript
import { AutomClient } from '@dotdo/sdk.do/integrations/autom'

const autom = new AutomClient({
  apiKey: process.env.AUTOM_API_KEY,
})

const result = await autom.action.execute({
  action: 'example_action',
  parameters: {},
})
```
