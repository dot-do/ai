# Mailersend Integration

MailerSend is a transactional email service designed for developers to integrate email sending capabilities into their applications.

## Quick Start

```typescript
import { MailersendClient } from '@dotdo/sdk.do/integrations/mailersend'

const mailersend = new MailersendClient({
  apiKey: process.env.MAILERSEND_API_KEY,
})

const result = await mailersend.action.execute({
  action: 'example_action',
  parameters: {},
})
```
