# Spotify Integration

Spotify is a digital music and podcast streaming service with millions of tracks, personalized playlists, and social sharing features

## Quick Start

```typescript
import { SpotifyClient } from '@dotdo/sdk.do/integrations/spotify'

const spotify = new SpotifyClient({
  apiKey: process.env.SPOTIFY_API_KEY,
})

const result = await spotify.action.execute({
  action: 'example_action',
  parameters: {},
})
```
