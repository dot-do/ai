# Openweather api Integration

Provides access to current weather data, forecasts, and historical weather data for any location worldwide.

## Quick Start

```typescript
import { OpenweatherapiClient } from '@dotdo/sdk.do/integrations/openweather_api'

const openweather_api = new OpenweatherapiClient({
  apiKey: process.env.OPENWEATHER_API_API_KEY,
})

const result = await openweather_api.action.execute({
  action: 'example_action',
  parameters: {},
})
```

Openweather api provides 17 actions through the Composio API.
