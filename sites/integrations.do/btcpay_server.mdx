# Btcpay server Integration

BTCPay Server is a free, open-source, self-hosted Bitcoin payment processor that enables merchants to accept Bitcoin payments without intermediaries.

## Quick Start

```typescript
import { BtcpayserverClient } from '@dotdo/sdk.do/integrations/btcpay_server'

const btcpay_server = new BtcpayserverClient({
  apiKey: process.env.BTCPAY_SERVER_API_KEY,
})

const result = await btcpay_server.action.execute({
  action: 'example_action',
  parameters: {},
})
```
