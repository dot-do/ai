# Crowdin Integration

Crowdin is a localization management platform that streamlines the translation process, offering tools for collaboration, content centralization, and workflow automation.

## Quick Start

```typescript
import { CrowdinClient } from '@dotdo/sdk.do/integrations/crowdin'

const crowdin = new CrowdinClient({
  apiKey: process.env.CROWDIN_API_KEY,
})

const result = await crowdin.action.execute({
  action: 'example_action',
  parameters: {},
})
```
