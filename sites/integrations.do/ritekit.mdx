# Ritekit Integration

RiteKit offers a suite of APIs designed to enhance social media engagement by providing tools for hashtag generation, link shortening, and content analysis.

## Quick Start

```typescript
import { RitekitClient } from '@dotdo/sdk.do/integrations/ritekit'

const ritekit = new RitekitClient({
  apiKey: process.env.RITEKIT_API_KEY,
})

const result = await ritekit.action.execute({
  action: 'example_action',
  parameters: {},
})
```
