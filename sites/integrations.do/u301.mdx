# U301 Integration

U301 provides an API to manage all U301 resources, currently under development.

## Quick Start

```typescript
import { U301Client } from '@dotdo/sdk.do/integrations/u301'

const u301 = new U301Client({
  apiKey: process.env.U301_API_KEY,
})

const result = await u301.action.execute({
  action: 'example_action',
  parameters: {},
})
```
