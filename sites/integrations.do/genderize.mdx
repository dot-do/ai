# Genderize Integration

Genderize is an API that predicts the gender of a person based on their first name, providing statistical probabilities for male or female classifications.

## Quick Start

```typescript
import { GenderizeClient } from '@dotdo/sdk.do/integrations/genderize'

const genderize = new GenderizeClient({
  apiKey: process.env.GENDERIZE_API_KEY,
})

const result = await genderize.action.execute({
  action: 'example_action',
  parameters: {},
})
```
