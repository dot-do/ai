# Parseur Integration

Parseur is an AI-powered data extraction tool that automates the parsing of emails, PDFs, and other documents into structured data.

## Quick Start

```typescript
import { ParseurClient } from '@dotdo/sdk.do/integrations/parseur'

const parseur = new ParseurClient({
  apiKey: process.env.PARSEUR_API_KEY,
})

const result = await parseur.action.execute({
  action: 'example_action',
  parameters: {},
})
```
