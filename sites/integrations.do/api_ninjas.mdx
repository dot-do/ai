# Api ninjas Integration

API Ninjas offers over 120 unique APIs across various categories, enabling developers to build real applications with real data.

## Quick Start

```typescript
import { ApininjasClient } from '@dotdo/sdk.do/integrations/api_ninjas'

const api_ninjas = new ApininjasClient({
  apiKey: process.env.API_NINJAS_API_KEY,
})

const result = await api_ninjas.action.execute({
  action: 'example_action',
  parameters: {},
})
```
