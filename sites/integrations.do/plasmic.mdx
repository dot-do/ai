# Plasmic Integration

Plasmic is a visual builder for the web, enabling teams to create and manage web pages and components without code.

## Quick Start

```typescript
import { PlasmicClient } from '@dotdo/sdk.do/integrations/plasmic'

const plasmic = new PlasmicClient({
  apiKey: process.env.PLASMIC_API_KEY,
})

const result = await plasmic.action.execute({
  action: 'example_action',
  parameters: {},
})
```
