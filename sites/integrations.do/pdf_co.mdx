# Pdf co Integration

PDF.co is a secure, cost-effective, and scalable API platform offering a suite of web APIs for tasks such as PDF extraction, generation, editing, splitting, merging, form filling, and barcode processing.

## Quick Start

```typescript
import { PdfcoClient } from '@dotdo/sdk.do/integrations/pdf_co'

const pdf_co = new PdfcoClient({
  apiKey: process.env.PDF_CO_API_KEY,
})

const result = await pdf_co.action.execute({
  action: 'example_action',
  parameters: {},
})
```
