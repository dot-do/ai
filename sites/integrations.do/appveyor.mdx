# Appveyor Integration

AppVeyor is a hosted continuous integration service for building and deploying applications.

## Quick Start

```typescript
import { AppveyorClient } from '@dotdo/sdk.do/integrations/appveyor'

const appveyor = new AppveyorClient({
  apiKey: process.env.APPVEYOR_API_KEY,
})

const result = await appveyor.action.execute({
  action: 'example_action',
  parameters: {},
})
```
