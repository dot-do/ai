# Docusign Integration

DocuSign provides eSignature and digital agreement solutions, enabling businesses to send, sign, track, and manage documents electronically

## Quick Start

```typescript
import { DocusignClient } from '@dotdo/sdk.do/integrations/docusign'

const docusign = new DocusignClient({
  apiKey: process.env.DOCUSIGN_API_KEY,
})

const result = await docusign.action.execute({
  action: 'example_action',
  parameters: {},
})
```
