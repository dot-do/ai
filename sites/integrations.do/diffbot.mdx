# Diffbot Integration

Diffbot provides AI-powered tools to extract and structure data from web pages, transforming unstructured web content into structured, linked data.

## Quick Start

```typescript
import { DiffbotClient } from '@dotdo/sdk.do/integrations/diffbot'

const diffbot = new DiffbotClient({
  apiKey: process.env.DIFFBOT_API_KEY,
})

const result = await diffbot.action.execute({
  action: 'example_action',
  parameters: {},
})
```

Diffbot provides 14 actions through the Composio API.
