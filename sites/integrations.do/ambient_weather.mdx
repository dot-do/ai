# Ambient weather Integration

Ambient Weather provides personal weather stations and an API for accessing real-time and historical weather data.

## Quick Start

```typescript
import { AmbientweatherClient } from '@dotdo/sdk.do/integrations/ambient_weather'

const ambient_weather = new AmbientweatherClient({
  apiKey: process.env.AMBIENT_WEATHER_API_KEY,
})

const result = await ambient_weather.action.execute({
  action: 'example_action',
  parameters: {},
})
```
