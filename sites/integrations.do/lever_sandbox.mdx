# Lever sandbox Integration

Lever Sandbox is a test environment for Lever's ATS, allowing organizations to experiment with configurations, workflows, and integrations without affecting production data

## Quick Start

```typescript
import { LeversandboxClient } from '@dotdo/sdk.do/integrations/lever_sandbox'

const lever_sandbox = new LeversandboxClient({
  apiKey: process.env.LEVER_SANDBOX_API_KEY,
})

const result = await lever_sandbox.action.execute({
  action: 'example_action',
  parameters: {},
})
```
