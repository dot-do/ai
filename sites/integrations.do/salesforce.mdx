# Salesforce Integration

Salesforce is a leading CRM platform integrating sales, service, marketing, and analytics to build customer relationships and drive business growth

## Quick Start

```typescript
import { SalesforceClient } from '@dotdo/sdk.do/integrations/salesforce'

const salesforce = new SalesforceClient({
  apiKey: process.env.SALESFORCE_API_KEY,
})

const result = await salesforce.action.execute({
  action: 'example_action',
  parameters: {},
})
```
