# Finage Integration

Finage provides secure RESTful APIs for real-time and historical financial market data covering stocks, forex, cryptocurrencies, indices, ETFs, and commodities.

## Quick Start

```typescript
import { FinageClient } from '@dotdo/sdk.do/integrations/finage'

const finage = new FinageClient({
  apiKey: process.env.FINAGE_API_KEY,
})

const result = await finage.action.execute({
  action: 'example_action',
  parameters: {},
})
```
