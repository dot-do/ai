# Google Calendar Integration

Google Calendar is a time management tool providing scheduling features, event reminders, and integration with email and other apps for streamlined organization

## Quick Start

```typescript
import { GoogleCalendarClient } from '@dotdo/sdk.do/integrations/googlecalendar'

const googlecalendar = new GoogleCalendarClient({
  apiKey: process.env.GOOGLECALENDAR_API_KEY,
})

const result = await googlecalendar.action.execute({
  action: 'example_action',
  parameters: {},
})
```
