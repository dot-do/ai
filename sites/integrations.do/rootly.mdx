# Rootly Integration

Rootly is an AI-native incident management platform that automates workflows and collaboration, integrating with Slack, PagerDuty, and other tools to streamline incident response.

## Quick Start

```typescript
import { RootlyClient } from '@dotdo/sdk.do/integrations/rootly'

const rootly = new RootlyClient({
  apiKey: process.env.ROOTLY_API_KEY,
})

const result = await rootly.action.execute({
  action: 'example_action',
  parameters: {},
})
```
