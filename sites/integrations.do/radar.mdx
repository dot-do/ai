# Radar Integration

Radar is a full-stack location infrastructure platform offering SDKs and APIs for geofencing, location tracking, geocoding, search, routing, and maps.

## Quick Start

```typescript
import { RadarClient } from '@dotdo/sdk.do/integrations/radar'

const radar = new RadarClient({
  apiKey: process.env.RADAR_API_KEY,
})

const result = await radar.action.execute({
  action: 'example_action',
  parameters: {},
})
```
