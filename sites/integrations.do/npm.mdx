# Npm Integration

npm is the default package manager for JavaScript and Node.js, facilitating the sharing and reuse of code, managing dependencies, and streamlining project workflows.

## Quick Start

```typescript
import { NpmClient } from '@dotdo/sdk.do/integrations/npm'

const npm = new NpmClient({
  apiKey: process.env.NPM_API_KEY,
})

const result = await npm.action.execute({
  action: 'example_action',
  parameters: {},
})
```
