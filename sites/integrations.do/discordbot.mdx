# Discordbot Integration

Discordbot refers to automated programs on Discord servers, performing tasks like moderation, music playback, and user engagement to enhance community interactions

## Quick Start

```typescript
import { DiscordbotClient } from '@dotdo/sdk.do/integrations/discordbot'

const discordbot = new DiscordbotClient({
  apiKey: process.env.DISCORDBOT_API_KEY,
})

const result = await discordbot.action.execute({
  action: 'example_action',
  parameters: {},
})
```
