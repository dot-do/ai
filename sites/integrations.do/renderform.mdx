# Renderform Integration

RenderForm is a user-friendly automation and design tool that enables the creation of templates and images with custom content such as images, texts, and QR codes.

## Quick Start

```typescript
import { RenderformClient } from '@dotdo/sdk.do/integrations/renderform'

const renderform = new RenderformClient({
  apiKey: process.env.RENDERFORM_API_KEY,
})

const result = await renderform.action.execute({
  action: 'example_action',
  parameters: {},
})
```
