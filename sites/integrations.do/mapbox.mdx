# Mapbox Integration

Mapbox is a platform that provides mapping, navigation, and location data services for developers to integrate into their applications.

## Quick Start

```typescript
import { MapboxClient } from '@dotdo/sdk.do/integrations/mapbox'

const mapbox = new MapboxClient({
  apiKey: process.env.MAPBOX_API_KEY,
})

const result = await mapbox.action.execute({
  action: 'example_action',
  parameters: {},
})
```
