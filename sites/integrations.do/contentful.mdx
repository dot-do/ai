# Contentful Integration

Contentful is a headless CMS allowing developers to create, manage, and distribute content across multiple channels and devices with an API-first approach

## Quick Start

```typescript
import { ContentfulClient } from '@dotdo/sdk.do/integrations/contentful'

const contentful = new ContentfulClient({
  apiKey: process.env.CONTENTFUL_API_KEY,
})

const result = await contentful.action.execute({
  action: 'example_action',
  parameters: {},
})
```

## Actions

Contentful provides 3 actions through the Composio API.
