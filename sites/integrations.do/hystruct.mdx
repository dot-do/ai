# Hystruct Integration

Hystruct is an AI-driven web scraping tool that simplifies data extraction from websites using predefined schemas, supporting various data fields like job listings and e-commerce.

## Quick Start

```typescript
import { HystructClient } from '@dotdo/sdk.do/integrations/hystruct'

const hystruct = new HystructClient({
  apiKey: process.env.HYSTRUCT_API_KEY,
})

const result = await hystruct.action.execute({
  action: 'example_action',
  parameters: {},
})
```

## Actions

Hystruct provides 1 actions through the Composio API.
