# Chatfai Integration

ChatFAI is an AI-powered platform that enables users to engage in interactive conversations with AI-generated versions of their favorite fictional characters from various media.

## Quick Start

```typescript
import { ChatfaiClient } from '@dotdo/sdk.do/integrations/chatfai'

const chatfai = new ChatfaiClient({
  apiKey: process.env.CHATFAI_API_KEY,
})

const result = await chatfai.action.execute({
  action: 'example_action',
  parameters: {},
})
```
