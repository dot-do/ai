# Wufoo Integration

Online form builder and data collection platform.

## Quick Start

```typescript
import { WufooClient } from '@dotdo/sdk.do/integrations/wufoo'

const wufoo = new WufooClient({
  apiKey: process.env.WUFOO_API_KEY,
  subdomain: 'yoursubdomain',
})

const forms = await wufoo.form.list()
```
