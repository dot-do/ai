# Veriphone Integration

Veriphone is a REST-based JSON API that provides phone number validation services, allowing developers to verify the validity, type, carrier, and region of phone numbers.

## Quick Start

```typescript
import { VeriphoneClient } from '@dotdo/sdk.do/integrations/veriphone'

const veriphone = new VeriphoneClient({
  apiKey: process.env.VERIPHONE_API_KEY,
})

const result = await veriphone.action.execute({
  action: 'example_action',
  parameters: {},
})
```
