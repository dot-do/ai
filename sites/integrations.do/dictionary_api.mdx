# Dictionary api Integration

The Merriam-Webster Dictionary API provides developers with access to comprehensive dictionary and thesaurus content, including definitions, etymologies, audio pronunciations, synonyms, and antonyms.

## Quick Start

```typescript
import { DictionaryapiClient } from '@dotdo/sdk.do/integrations/dictionary_api'

const dictionary_api = new DictionaryapiClient({
  apiKey: process.env.DICTIONARY_API_API_KEY,
})

const result = await dictionary_api.action.execute({
  action: 'example_action',
  parameters: {},
})
```

## Actions

Dictionary api provides 1 actions through the Composio API.
