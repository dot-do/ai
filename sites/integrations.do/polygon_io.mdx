# Polygon io Integration

Polygon.io provides real-time and historical market data APIs for stocks, options, forex, and cryptocurrencies, enabling developers to build financial applications with ease.

## Quick Start

```typescript
import { PolygonioClient } from '@dotdo/sdk.do/integrations/polygon_io'

const polygon_io = new PolygonioClient({
  apiKey: process.env.POLYGON_IO_API_KEY,
})

const result = await polygon_io.action.execute({
  action: 'example_action',
  parameters: {},
})
```
