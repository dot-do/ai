# Kickbox Integration

Kickbox provides an API for real-time email verification, list cleaning, and quality scoring to ensure email deliverability.

## Quick Start

```typescript
import { KickboxClient } from '@dotdo/sdk.do/integrations/kickbox'

const kickbox = new KickboxClient({
  apiKey: process.env.KICKBOX_API_KEY,
})

const result = await kickbox.action.execute({
  action: 'example_action',
  parameters: {},
})
```
