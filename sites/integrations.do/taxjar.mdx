# Taxjar Integration

TaxJar provides a comprehensive sales tax API for real-time tax calculations, reporting, and filing.

## Quick Start

```typescript
import { TaxjarClient } from '@dotdo/sdk.do/integrations/taxjar'

const taxjar = new TaxjarClient({
  apiKey: process.env.TAXJAR_API_KEY,
})

const result = await taxjar.action.execute({
  action: 'example_action',
  parameters: {},
})
```
