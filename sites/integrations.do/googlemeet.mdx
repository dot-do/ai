# Google Meet Integration

Google Meet is a secure video conferencing platform that integrates with Google Workspace, facilitating remote meetings, screen sharing, and chat

## Quick Start

```typescript
import { GoogleMeetClient } from '@dotdo/sdk.do/integrations/googlemeet'

const googlemeet = new GoogleMeetClient({
  apiKey: process.env.GOOGLEMEET_API_KEY,
})

const result = await googlemeet.action.execute({
  action: 'example_action',
  parameters: {},
})
```
