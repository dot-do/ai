# Onedesk Integration

OneDesk is an all-in-one platform combining help desk and project management functionalities, enabling teams to manage customer support and project tasks seamlessly.

## Quick Start

```typescript
import { OnedeskClient } from '@dotdo/sdk.do/integrations/onedesk'

const onedesk = new OnedeskClient({
  apiKey: process.env.ONEDESK_API_KEY,
})

const result = await onedesk.action.execute({
  action: 'example_action',
  parameters: {},
})
```
