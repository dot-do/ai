# Survey monkey Integration

SurveyMonkey is an online survey development platform that enables users to create, distribute, and analyze surveys for various purposes.

## Quick Start

```typescript
import { SurveymonkeyClient } from '@dotdo/sdk.do/integrations/survey_monkey'

const survey_monkey = new SurveymonkeyClient({
  apiKey: process.env.SURVEY_MONKEY_API_KEY,
})

const result = await survey_monkey.action.execute({
  action: 'example_action',
  parameters: {},
})
```
