# Coinbase Integration

Coinbase is a platform for buying, selling, transferring, and storing cryptocurrency.

## Quick Start

```typescript
import { CoinbaseClient } from '@dotdo/sdk.do/integrations/coinbase'

const coinbase = new CoinbaseClient({
  apiKey: process.env.COINBASE_API_KEY,
})

const result = await coinbase.action.execute({
  action: 'example_action',
  parameters: {},
})
```
