# Statuscake Integration

StatusCake is a website monitoring platform that provides observability for applications, offering features like uptime monitoring, page speed monitoring, SSL monitoring, and more.

## Quick Start

```typescript
import { StatuscakeClient } from '@dotdo/sdk.do/integrations/statuscake'

const statuscake = new StatuscakeClient({
  apiKey: process.env.STATUSCAKE_API_KEY,
})

const result = await statuscake.action.execute({
  action: 'example_action',
  parameters: {},
})
```
