# Faceup Integration

FaceUp is an anonymous reporting tool designed for companies and schools, enabling employees and students to safely report issues and misconduct.

## Quick Start

```typescript
import { FaceupClient } from '@dotdo/sdk.do/integrations/faceup'

const faceup = new FaceupClient({
  apiKey: process.env.FACEUP_API_KEY,
})

const result = await faceup.action.execute({
  action: 'example_action',
  parameters: {},
})
```
