# Veo Integration

Veo 3 is Google's state-of-the-art model for generating high-fidelity 8s 720p videos with natively generated audio via the Gemini API.

## Quick Start

```typescript
import { VeoClient } from '@dotdo/sdk.do/integrations/veo'

const veo = new VeoClient({
  apiKey: process.env.VEO_API_KEY,
})

const result = await veo.action.execute({
  action: 'example_action',
  parameters: {},
})
```
