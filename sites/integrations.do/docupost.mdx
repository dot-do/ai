# Docupost Integration

DocuPost is a print and mail service that enables users to send digital files as physical letters and postcards via the U.S. Postal Service.

## Quick Start

```typescript
import { DocupostClient } from '@dotdo/sdk.do/integrations/docupost'

const docupost = new DocupostClient({
  apiKey: process.env.DOCUPOST_API_KEY,
})

const result = await docupost.action.execute({
  action: 'example_action',
  parameters: {},
})
```
