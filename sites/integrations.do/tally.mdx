# Tally Integration

Tally is a form-building platform that allows users to create forms, collect responses, and integrate with various tools and services.

## Quick Start

```typescript
import { TallyClient } from '@dotdo/sdk.do/integrations/tally'

const tally = new TallyClient({
  apiKey: process.env.TALLY_API_KEY,
})

const result = await tally.action.execute({
  action: 'example_action',
  parameters: {},
})
```
