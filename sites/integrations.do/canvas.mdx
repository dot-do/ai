# Canvas Integration

Canvas is a learning management system supporting online courses, assignments, grading, and collaboration, widely used by schools and universities for virtual classrooms

## Quick Start

```typescript
import { CanvasClient } from '@dotdo/sdk.do/integrations/canvas'

const canvas = new CanvasClient({
  apiKey: process.env.CANVAS_API_KEY,
})

const result = await canvas.action.execute({
  action: 'example_action',
  parameters: {},
})
```
