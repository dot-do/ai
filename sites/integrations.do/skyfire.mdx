# Skyfire Integration

Skyfire enables AI agents to autonomously transact and pay for services, creating a seamless payment infrastructure for AI applications.

## Quick Start

```typescript
import { SkyfireClient } from '@dotdo/sdk.do/integrations/skyfire'

const skyfire = new SkyfireClient({
  apiKey: process.env.SKYFIRE_API_KEY,
})

const result = await skyfire.action.execute({
  action: 'example_action',
  parameters: {},
})
```
