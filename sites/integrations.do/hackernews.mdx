# Hackernews Integration

Hacker News is a tech-focused news aggregator by Y Combinator, featuring user-submitted stories and discussions on startups, programming, and emerging trends

## Quick Start

```typescript
import { HackernewsClient } from '@dotdo/sdk.do/integrations/hackernews'

const hackernews = new HackernewsClient({
  apiKey: process.env.HACKERNEWS_API_KEY,
})

const result = await hackernews.action.execute({
  action: 'example_action',
  parameters: {},
})
```
