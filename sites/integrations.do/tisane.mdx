# Tisane Integration

Tisane API is a natural language processing tool that detects problematic content, extracts topics, and performs aspect-based sentiment analysis across 27 languages.

## Quick Start

```typescript
import { TisaneClient } from '@dotdo/sdk.do/integrations/tisane'

const tisane = new TisaneClient({
  apiKey: process.env.TISANE_API_KEY,
})

const result = await tisane.action.execute({
  action: 'example_action',
  parameters: {},
})
```
