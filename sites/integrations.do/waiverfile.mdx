# Waiverfile Integration

WaiverFile is an electronic waiver form system that allows customers to sign release waivers from any device, streamlining the waiver management process.

## Quick Start

```typescript
import { WaiverfileClient } from '@dotdo/sdk.do/integrations/waiverfile'

const waiverfile = new WaiverfileClient({
  apiKey: process.env.WAIVERFILE_API_KEY,
})

const result = await waiverfile.action.execute({
  action: 'example_action',
  parameters: {},
})
```
