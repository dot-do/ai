# Gamma Integration

Gamma helps create beautiful, interactive content and presentations using AI. This integration enables programmatic generation via Gamma’s API.

## Quick Start

```typescript
import { GammaClient } from '@dotdo/sdk.do/integrations/gamma'

const gamma = new GammaClient({
  apiKey: process.env.GAMMA_API_KEY,
})

const result = await gamma.action.execute({
  action: 'example_action',
  parameters: {},
})
```
