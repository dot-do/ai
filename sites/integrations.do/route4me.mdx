# Route4me Integration

Route4Me provides a Last Mile Routing API offering automated solutions for route planning and optimization, tailored for logistics-intensive businesses.

## Quick Start

```typescript
import { Route4meClient } from '@dotdo/sdk.do/integrations/route4me'

const route4me = new Route4meClient({
  apiKey: process.env.ROUTE4ME_API_KEY,
})

const result = await route4me.action.execute({
  action: 'example_action',
  parameters: {},
})
```
