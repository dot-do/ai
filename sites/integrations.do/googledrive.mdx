# Google Drive Integration

Google Drive is a cloud storage solution for uploading, sharing, and collaborating on files across devices, with robust search and offline access

## Quick Start

```typescript
import { GoogleDriveClient } from '@dotdo/sdk.do/integrations/googledrive'

const googledrive = new GoogleDriveClient({
  apiKey: process.env.GOOGLEDRIVE_API_KEY,
})

const result = await googledrive.action.execute({
  action: 'example_action',
  parameters: {},
})
```
