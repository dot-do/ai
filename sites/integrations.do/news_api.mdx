# News api Integration

News API is a simple HTTP REST API for searching and retrieving live articles from all over the web.

## Quick Start

```typescript
import { NewsapiClient } from '@dotdo/sdk.do/integrations/news_api'

const news_api = new NewsapiClient({
  apiKey: process.env.NEWS_API_API_KEY,
})

const result = await news_api.action.execute({
  action: 'example_action',
  parameters: {},
})
```

## Actions

News api provides 3 actions through the Composio API.
