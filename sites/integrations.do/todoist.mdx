# Todoist Integration

Todoist is a task management tool allowing users to create to-do lists, set deadlines, and collaborate on projects with reminders and cross-platform syncing

## Quick Start

```typescript
import { TodoistClient } from '@dotdo/sdk.do/integrations/todoist'

const todoist = new TodoistClient({
  apiKey: process.env.TODOIST_API_KEY,
})

const result = await todoist.action.execute({
  action: 'example_action',
  parameters: {},
})
```

## Actions

Todoist provides 30 actions through the Composio API.

## Webhooks

Todoist supports 1 webhook triggers for real-time event notifications.
