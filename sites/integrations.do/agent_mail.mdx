# Agent mail Integration

AgentMail provides AI agents with their own email inboxes, enabling them to send, receive, and act upon emails for communication with services, people, and other agents.

## Quick Start

```typescript
import { AgentmailClient } from '@dotdo/sdk.do/integrations/agent_mail'

const agent_mail = new AgentmailClient({
  apiKey: process.env.AGENT_MAIL_API_KEY,
})

const result = await agent_mail.action.execute({
  action: 'example_action',
  parameters: {},
})
```
