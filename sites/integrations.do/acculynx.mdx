# Acculynx Integration

Using the AccuLynx API, data can be seamlessly exchanged between AccuLynx and other applications for greater efficiency and productivity.

## Quick Start

```typescript
import { AcculynxClient } from '@dotdo/sdk.do/integrations/acculynx'

const acculynx = new AcculynxClient({
  apiKey: process.env.ACCULYNX_API_KEY,
})

const result = await acculynx.action.execute({
  action: 'example_action',
  parameters: {},
})
```
