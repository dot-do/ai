# Pdf api io Integration

PDF-API.io offers a dynamic PDF generation API with a drag-and-drop template designer, allowing developers to create customizable PDFs using HTML and CSS.

## Quick Start

```typescript
import { PdfapiioClient } from '@dotdo/sdk.do/integrations/pdf_api_io'

const pdf_api_io = new PdfapiioClient({
  apiKey: process.env.PDF_API_IO_API_KEY,
})

const result = await pdf_api_io.action.execute({
  action: 'example_action',
  parameters: {},
})
```

## Actions

Pdf api io provides 1 actions through the Composio API.
