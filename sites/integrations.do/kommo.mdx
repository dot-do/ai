# Kommo Integration

Kommo CRM (formerly amoCRM) integration tool for managing customer relationships, sales pipelines, and business processes.

## Quick Start

```typescript
import { KommoClient } from '@dotdo/sdk.do/integrations/kommo'

const kommo = new KommoClient({
  apiKey: process.env.KOMMO_API_KEY,
})

const result = await kommo.action.execute({
  action: 'example_action',
  parameters: {},
})
```
