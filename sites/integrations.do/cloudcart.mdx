# Cloudcart Integration

CloudCart is an e-commerce platform that enables businesses to create and manage online stores efficiently.

## Quick Start

```typescript
import { CloudcartClient } from '@dotdo/sdk.do/integrations/cloudcart'

const cloudcart = new CloudcartClient({
  apiKey: process.env.CLOUDCART_API_KEY,
})

const result = await cloudcart.action.execute({
  action: 'example_action',
  parameters: {},
})
```
