# Kit Integration

Kit is a platform that allows creators to automate tasks and developers to build apps for the Kit App Store.

## Quick Start

```typescript
import { KitClient } from '@dotdo/sdk.do/integrations/kit'

const kit = new KitClient({
  apiKey: process.env.KIT_API_KEY,
})

const result = await kit.action.execute({
  action: 'example_action',
  parameters: {},
})
```
