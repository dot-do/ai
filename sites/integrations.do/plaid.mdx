# Plaid Integration

Plaid is a data network that powers fintech and digital finance products by connecting users' bank accounts to apps.

## Features

- **Accounts**: Retrieve bank account information
- **Transactions**: Access transaction history
- **Balance**: Get real-time balance data
- **Identity**: Retrieve account holder identity info
- **Auth**: Verify account and routing numbers
- **Webhooks**: Real-time notifications for data updates

## Quick Start

```typescript
import { PlaidClient } from '@dotdo/sdk.do/integrations/plaid'

const plaid = new PlaidClient({
  clientId: process.env.PLAID_CLIENT_ID,
  secret: process.env.PLAID_SECRET,
  environment: 'sandbox', // sandbox, development, or production
})

// Get accounts
const accounts = await plaid.account.list({
  access_token: 'access-token',
})

// Get transactions
const transactions = await plaid.transaction.list({
  access_token: 'access-token',
  start_date: '2025-01-01',
  end_date: '2025-12-31',
})
```

## Authentication

Plaid uses API key authentication with client ID and secret:

```bash
export PLAID_CLIENT_ID=your_client_id
export PLAID_SECRET=your_secret
```

## Resources

### Accounts

Retrieve account information:

```typescript
// List accounts
const accounts = await plaid.account.list({
  access_token: 'access-token',
})

// Get account balances
const balances = await plaid.account.balance({
  access_token: 'access-token',
})
```

### Transactions

Get transaction history:

```typescript
// List transactions
const transactions = await plaid.transaction.list({
  access_token: 'access-token',
  start_date: '2025-01-01',
  end_date: '2025-01-31',
})

// Sync transactions (incremental updates)
const sync = await plaid.transaction.sync({
  access_token: 'access-token',
  cursor: 'cursor-token',
})
```

### Balance

Get real-time balance:

```typescript
// Get balance
const balance = await plaid.balance.get({
  access_token: 'access-token',
})
```

### Identity

Retrieve identity information:

```typescript
// Get identity
const identity = await plaid.identity.get({
  access_token: 'access-token',
})
```

## Webhooks

Plaid supports webhooks with JWT signature verification:

```typescript
import { PlaidWebhookHandler, WebhookEventRouter } from '@dotdo/sdk.do/integrations/plaid'

const handler = new PlaidWebhookHandler(process.env.PLAID_CLIENT_ID, process.env.PLAID_SECRET)
const event = await handler.handleRequest(request)

const router = new WebhookEventRouter()
router.onTransactionsDefaultUpdate(async (event) => {
  console.log('New transactions available:', event.data)
})

router.onItemError(async (event) => {
  console.log('Item error:', event.data)
})

await router.route(event)
```

## Best Practices

1. **Link Token**: Use Link to connect user accounts securely
2. **Access Tokens**: Store access tokens securely
3. **Webhooks**: Use webhooks for transaction updates instead of polling
4. **Environments**: Use sandbox for testing, production for live data
5. **Error Handling**: Handle ITEM_LOGIN_REQUIRED errors gracefully
6. **Rate Limits**: Implement exponential backoff for rate limits
