# Textrazor Integration

TextRazor is a natural language processing API that extracts meaning, entities, and relationships from text, powering advanced content analysis and sentiment detection

## Quick Start

```typescript
import { TextrazorClient } from '@dotdo/sdk.do/integrations/textrazor'

const textrazor = new TextrazorClient({
  apiKey: process.env.TEXTRAZOR_API_KEY,
})

const result = await textrazor.action.execute({
  action: 'example_action',
  parameters: {},
})
```
