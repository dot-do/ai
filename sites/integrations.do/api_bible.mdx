# Api bible Integration

API.Bible allows developers to integrate Scripture content into their applications or websites for non-commercial purposes.

## Quick Start

```typescript
import { ApibibleClient } from '@dotdo/sdk.do/integrations/api_bible'

const api_bible = new ApibibleClient({
  apiKey: process.env.API_BIBLE_API_KEY,
})

const result = await api_bible.action.execute({
  action: 'example_action',
  parameters: {},
})
```
