# Bunnycdn Integration

BunnyCDN is a powerful content delivery network (CDN) offering an API to manage and deliver content globally with ease.

## Quick Start

```typescript
import { BunnycdnClient } from '@dotdo/sdk.do/integrations/bunnycdn'

const bunnycdn = new BunnycdnClient({
  apiKey: process.env.BUNNYCDN_API_KEY,
})

const result = await bunnycdn.action.execute({
  action: 'example_action',
  parameters: {},
})
```
