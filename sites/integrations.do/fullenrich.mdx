# Fullenrich Integration

FullEnrich is a B2B email and phone waterfall enrichment platform that aggregates contact information from over 15 premium vendors to find the emails and phone numbers of leads.

## Quick Start

```typescript
import { FullenrichClient } from '@dotdo/sdk.do/integrations/fullenrich'

const fullenrich = new FullenrichClient({
  apiKey: process.env.FULLENRICH_API_KEY,
})

const result = await fullenrich.action.execute({
  action: 'example_action',
  parameters: {},
})
```
