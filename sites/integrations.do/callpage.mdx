# Callpage Integration

CallPage is a lead capture tool that enables businesses to connect with website visitors through immediate phone callbacks, enhancing lead generation and sales conversion rates.

## Quick Start

```typescript
import { CallpageClient } from '@dotdo/sdk.do/integrations/callpage'

const callpage = new CallpageClient({
  apiKey: process.env.CALLPAGE_API_KEY,
})

const result = await callpage.action.execute({
  action: 'example_action',
  parameters: {},
})
```
