# Brandfetch Integration

Brandfetch offers an API that retrieves company logos, brand colors, and other visual assets, helping marketers and developers maintain consistent branding across apps

## Quick Start

```typescript
import { BrandfetchClient } from '@dotdo/sdk.do/integrations/brandfetch'

const brandfetch = new BrandfetchClient({
  apiKey: process.env.BRANDFETCH_API_KEY,
})

const result = await brandfetch.action.execute({
  action: 'example_action',
  parameters: {},
})
```
