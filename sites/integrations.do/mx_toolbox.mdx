# Mx toolbox Integration

MxToolbox provides a suite of network diagnostic and monitoring tools, including DNS lookups, blacklist checks, and email health analysis.

## Quick Start

```typescript
import { MxtoolboxClient } from '@dotdo/sdk.do/integrations/mx_toolbox'

const mx_toolbox = new MxtoolboxClient({
  apiKey: process.env.MX_TOOLBOX_API_KEY,
})

const result = await mx_toolbox.action.execute({
  action: 'example_action',
  parameters: {},
})
```
