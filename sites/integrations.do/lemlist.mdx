# Lemlist Integration

lemlist is a prospecting tool that automates multichannel outreach, enabling users to find leads with valid contact information and reach them across email, LinkedIn, or calls with personalized messages.

## Quick Start

```typescript
import { LemlistClient } from '@dotdo/sdk.do/integrations/lemlist'

const lemlist = new LemlistClient({
  apiKey: process.env.LEMLIST_API_KEY,
})

const result = await lemlist.action.execute({
  action: 'example_action',
  parameters: {},
})
```
