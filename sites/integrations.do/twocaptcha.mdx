# Twocaptcha Integration

2Captcha is a service that provides automated CAPTCHA solving solutions, enabling developers to bypass various types of CAPTCHAs in their applications.

## Quick Start

```typescript
import { TwocaptchaClient } from '@dotdo/sdk.do/integrations/twocaptcha'

const twocaptcha = new TwocaptchaClient({
  apiKey: process.env.TWOCAPTCHA_API_KEY,
})

const result = await twocaptcha.action.execute({
  action: 'example_action',
  parameters: {},
})
```
