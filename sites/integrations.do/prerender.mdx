# Prerender Integration

Prerender.io is a service that pre-renders JavaScript-heavy websites to improve SEO by serving static HTML versions to search engine crawlers.

## Quick Start

```typescript
import { PrerenderClient } from '@dotdo/sdk.do/integrations/prerender'

const prerender = new PrerenderClient({
  apiKey: process.env.PRERENDER_API_KEY,
})

const result = await prerender.action.execute({
  action: 'example_action',
  parameters: {},
})
```
