# Browseai Integration

Browse.ai allows you to turn any website into an API using its advanced web automation and data extraction tools, enabling easy monitoring and data retrieval from websites.

## Quick Start

```typescript
import { BrowseaiClient } from '@dotdo/sdk.do/integrations/browseai'

const browseai = new BrowseaiClient({
  apiKey: process.env.BROWSEAI_API_KEY,
})

const result = await browseai.action.execute({
  action: 'example_action',
  parameters: {},
})
```
