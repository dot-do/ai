# Bigml Integration

BigML is a comprehensive machine learning platform that simplifies the creation and deployment of predictive models through an intuitive web interface and a REST API.

## Quick Start

```typescript
import { BigmlClient } from '@dotdo/sdk.do/integrations/bigml'

const bigml = new BigmlClient({
  apiKey: process.env.BIGML_API_KEY,
})

const result = await bigml.action.execute({
  action: 'example_action',
  parameters: {},
})
```
