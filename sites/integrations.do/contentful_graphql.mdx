# Contentful graphql Integration

The Contentful GraphQL Content API allows developers to query and deliver content using GraphQL, providing a flexible and efficient way to access content stored in Contentful.

## Quick Start

```typescript
import { ContentfulgraphqlClient } from '@dotdo/sdk.do/integrations/contentful_graphql'

const contentful_graphql = new ContentfulgraphqlClient({
  apiKey: process.env.CONTENTFUL_GRAPHQL_API_KEY,
})

const result = await contentful_graphql.action.execute({
  action: 'example_action',
  parameters: {},
})
```

Contentful graphql provides 3 actions through the Composio API.
