# Advanced Patterns

Master advanced Recruiting AI Agent patterns for complex use cases.

## Multi-Step Workflows

Execute complex multi-step workflows:

```typescript
import $, { agent, db } from 'sdk.do'

const recruitingAgent = await agent.create({
  $role: $.RecruitingAgent,
  model: 'claude-sonnet-4.5',
})

// Define workflow
const workflow = {
  steps: [
    { name: 'step1', action: 'analyze' },
    { name: 'step2', action: 'plan' },
    { name: 'step3', action: 'execute' },
    { name: 'step4', action: 'review' },
  ],
  parallel: false,
}

// Execute workflow
const result = await recruitingAgent.executeWorkflow(workflow)
```

## Parallel Processing

Process multiple tasks in parallel:

```typescript
// Parallel execution
const tasks = Array.from({ length: 10 }, (_, i) => ({
  task: `Task ${i}`,
  context: { id: i },
}))

const results = await Promise.all(tasks.map((task) => recruitingAgent.execute(task)))

console.log(`Completed ${results.length} tasks`)
```

## Conditional Execution

Execute tasks conditionally:

```typescript
// Conditional logic
const result = await recruitingAgent.execute({
  task: 'Initial analysis',
  conditions: {
    if_success: 'continue_to_execution',
    if_failure: 'escalate_to_human',
    if_uncertain: 'request_more_data',
  },
})

// Handle result based on condition
switch (result.next_action) {
  case 'continue_to_execution':
    await recruitingAgent.execute({ task: 'Execute plan' })
    break
  case 'escalate_to_human':
    await notifyHuman(result)
    break
  case 'request_more_data':
    const additionalData = await gatherData()
    await recruitingAgent.execute({ task: 'Re-analyze', data: additionalData })
    break
}
```

## Event-Driven Architecture

Build event-driven systems:

```typescript
import $, { agent, on, send } from 'sdk.do'

const recruitingAgent = await agent.create({
  $role: $.RecruitingAgent,
  event_driven: true,
})

// Subscribe to events
await on($.Event.triggered, async (event) => {
  const result = await recruitingAgent.execute({
    task: 'Handle event',
    event: event,
  })

  // Publish result
  await send($.Result.completed, result)
})

// Chain events
await on($.Result.completed, async (result) => {
  if (result.requires_followup) {
    await send($.Event.triggered, {
      type: 'followup',
      data: result,
    })
  }
})
```

## Multi-Agent Collaboration

Coordinate multiple agents:

```typescript
import $, { agent } from 'sdk.do'

// Create specialized agents
const agent1 = await agent.create({
  $role: $.RecruitingAgent,
  specialization: 'analysis',
})

const agent2 = await agent.create({
  $role: $.RecruitingAgent,
  specialization: 'execution',
})

const agent3 = await agent.create({
  $role: $.RecruitingAgent,
  specialization: 'monitoring',
})

// Collaborative workflow
const analysis = await agent1.analyze(task)
const execution = await agent2.execute({
  plan: analysis.recommendations,
})
const monitoring = await agent3.monitor({
  execution: execution,
})
```

## Advanced Error Handling

Comprehensive error handling:

```typescript
const recruitingAgent = await agent.create({
  $role: $.RecruitingAgent,
  error_handling: {
    retry_strategy: 'exponential_backoff',
    max_retries: 5,
    fallback_enabled: true,
    circuit_breaker: {
      enabled: true,
      threshold: 0.5,
      timeout: 60000,
    },
  },
})

// Automatic error recovery
try {
  const result = await recruitingAgent.execute(task)
} catch (error) {
  if (error.recoverable) {
    // Automatic retry with backoff
    console.log('Retrying...')
  } else {
    // Escalate to human
    await escalate(error)
  }
}
```

## Monitoring and Observability

Monitor agent performance:

```typescript

const recruitingAgent = await agent.create({

  comparison: 'greater_than',
  action: 'notify',
})
```
