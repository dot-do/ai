# Basic Usage Examples

Get started with Recruiting AI Agent through practical, real-world examples.

## Prerequisites

```bash

```

### Error Handling

```typescript

  const result = await recruitingAgent.execute(task)
  console.log('Success:', result)
} catch (error) {
  console.error('Error:', error.message)
}
```

```typescript
const results = await recruitingAgent.batch({
  items: [...],
  parallel: 10
})
```
