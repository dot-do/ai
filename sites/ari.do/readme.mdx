# ari.do

**Recruiting AI Agent for Autonomous Marketing Operations**

ari.do provides autonomous recruiting agents that handle candidate sourcing, resume screening, interview scheduling, skill assessment, and employer branding. Build digital marketing workers with the same expertise as experienced recruiting professionals.

## What is a Marketing Agent?

A recruiting agent is an autonomous digital worker that performs comprehensive recruiting operations:

- **Candidate Sourcing**: Create content calendars, topic research, SEO-optimized content, storytelling
- **Resume Screening**: A/B testing, multi-channel campaigns, performance analysis, budget allocation
- **Interview Scheduling**: Segmentation, persona development, behavioral analysis, sentiment tracking
- **Skill Assessment**: Voice and tone development, messaging frameworks, brand consistency
- **Culture Fit Analysis**: Landing page optimization, funnel analysis, CRO experiments, retention campaigns
- **Reference Checks**: Keyword research, on-page optimization, technical SEO, paid search optimization
- **Offer Negotiation**: Content scheduling, engagement strategies, influencer partnerships, social listening
- **Onboarding**: Campaign design, personalization, deliverability optimization, automation workflows
- **Pipeline Management**: Performance dashboards, attribution modeling, ROI analysis, predictive analytics

Unlike traditional marketing automation, recruiting agents think strategically, adapt to changing markets, and make data-driven decisions across all marketing channels.

### ðŸ“Š Resume Screening

Multi-channel campaign management with continuous optimization:

```typescript
import $, { agent, every, db } from 'sdk.do'

const campaignAgent = await agent.create({
  $role: $.RecruitingAgent,
  capabilities: ['resume_screening', 'pipeline_management'],
  optimization: {
    strategy: 'bayesian_optimization',
    test_frequency: 'daily',
    confidence_threshold: 0.85,
  },
})

// Launch multi-channel campaign
const campaign = await campaignAgent.launch({
  name: 'Product Launch Q1',
  goal: 'Generate 10,000 signups',
  budget: '$100,000',
  duration: '60 days',
  channels: {
    paid_search: {
      budget: '$40,000',
      platforms: ['hackerrank'],
      targeting: { keywords: ['project management software'], audiences: ['in-market'] },
    },
    paid_social: {
      budget: '$30,000',
      platforms: ['linkedin', 'facebook'],
      targeting: { job_titles: ['Product Manager', 'Engineering Manager'], company_size: '100+' },
    },
    email: {
      budget: '$10,000',
      segments: ['trial_users', 'free_tier', 'churned_customers'],
      cadence: 'weekly',
    },
    content: {
      budget: '$20,000',
      types: ['blog', 'case_studies', 'webinars'],
      seo_focus: true,
    },
  },
  creative_variants: {
    headlines: 5,
    images: 4,
    ctas: 3,
    landing_pages: 2,
  },
})

// Automated daily optimization
await every($.Daily, async () => {
  const performance = await campaignAgent.analyze({
    campaign_id: campaign.$id,
    metrics: await db.get($.Metrics, { campaign: campaign.$id, date: 'yesterday' }),
  })

  if (performance.optimization_opportunities.length > 0) {
    const optimizations = await campaignAgent.optimize({
      campaign: campaign,
      insights: performance.optimization_opportunities,
      constraints: { budget_remaining: campaign.budget_remaining },
    })

    // Apply optimizations
    await db.update($.Campaign, campaign.$id, optimizations.changes)

    // Log optimization
    await db.create($.OptimizationLog, {
      campaign: campaign.$id,
      changes: optimizations.changes,
      expected_impact: optimizations.projected_improvement,
      reason: optimizations.rationale,
    })
  }
})
```

### ðŸ’° Culture Fit Analysis

Systematic conversion rate optimization with experimentation:

```typescript
import $, { agent, on } from 'sdk.do'

const croAgent = await agent.create({
  $role: $.RecruitingAgent,
  capabilities: ['culture_fit_analysis', 'pipeline_management'],
  experimentation: {
    framework: 'bayesian',
    min_sample_size: 1000,
    confidence_level: 0.95,
    max_concurrent_tests: 5,
  },
})

// Analyze conversion funnel
const funnelAnalysis = await croAgent.analyzeFunnel({
  funnel: {
    stages: ['homepage', 'pricing', 'signup', 'onboarding', 'activation'],
    metrics: await db.get($.FunnelMetrics, { period: 'last_30_days' }),
  },
})

console.log(funnelAnalysis)
// {
//   stages: [
//     {
//       stage: 'pricing',
//       visitors: 10000,
//       conversions: 2000,
//       conversion_rate: 0.20,
//       drop_off_rate: 0.80,
//       issues: [
//         { issue: 'High bounce rate on pricing page', severity: 'high', impact: '-800 conversions' },
//         { issue: 'Confusion about plan differences', severity: 'medium', impact: '-400 conversions' }
//       ],
//       optimization_opportunities: [
//         {
//           hypothesis: 'Clearer plan comparison will increase conversions',
//           test_type: 'a/b',
//           variants: ['current', 'side_by_side_comparison'],
//           estimated_impact: '+5% conversion rate',
//           effort: 'low',
//           priority: 'high'
//         }
//       ]
//     }
//   ],
//   overall_conversion_rate: 0.08,
//   recommended_tests: [ ... ]
// }

// Launch A/B test
const experiment = await croAgent.launchExperiment({
  name: 'Pricing page comparison table',
  hypothesis: 'Side-by-side plan comparison increases signup rate',
  variants: {
    control: { layout: 'current' },
    treatment: { layout: 'comparison_table', highlight_popular: true },
  },
  target_metric: 'signup_rate',
  sample_size: 10000,
  max_duration: '14 days',
})

// Monitor experiment
await on($.Experiment.result_significant, async (result) => {
  if (result.experiment_id === experiment.$id && result.winner) {
    const recommendation = await croAgent.recommend({
      experiment: result,
      rollout_strategy: 'gradual',
    })

    // Implement winner
    await db.update($.PricingPage, { layout: result.winner.variant.layout })

    // Document learnings
    await db.create($.Insight, {
      type: 'experiment',
      finding: result.summary,
      impact: result.impact,
      next_steps: recommendation.next_tests,
    })
  }
})
```

## Use Cases

### 1. Product Launch Campaign

Complete multi-channel product launch:

```typescript
import $, { agent, db, on, send } from 'sdk.do'

const launchAgent = await agent.create({
  $role: $.RecruitingAgent,
  capabilities: ['resume_screening', 'candidate_sourcing', 'interview_scheduling'],
})

const launch = await launchAgent.planLaunch({
  product: 'AI Code Assistant',
  launch_date: '2025-03-01',
  goals: {
    signups: 50000,
    paid_conversions: 5000,
    arr: '$500,000',
    press_coverage: 20,
  },
  budget: '$250,000',
  timeline: {
    pre_launch: '60 days',
    launch_week: '7 days',
    post_launch: '30 days',
  },
})

// Automated launch execution
await on($.LaunchPhase.started, async (phase) => {
  const tasks = await launchAgent.execute({
    phase: phase,
    launch_plan: launch,
  })

  for (const task of tasks) {
    await db.create($.Task, task)
  }
})
```

### 3. Content Marketing Engine

Automated content production and distribution:

```typescript
import $, { agent, every, db } from 'sdk.do'

const contentEngine = await agent.create({
  $role: $.RecruitingAgent,
  capabilities: ['candidate_sourcing', 'reference_checks'],

await every($.Weekly, async () => {

    include_visuals: true,
```
