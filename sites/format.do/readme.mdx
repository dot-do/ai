# format.do

**Format dates, numbers, strings, currencies and data structures with localization**

format.do provides powerful utilities for data formatting in your Business-as-Code applications. Built with semantic patterns, type safety, and integration with the entire `.do` platform ecosystem.

## Quick Start

```bash
pnpm add format.do
```

```typescript
import $, { format } from 'format.do'

// Basic usage
const result = await format({
  // Your data
  data: 'example',
  // Options
  options: {},
})

console.log('Result:', result)
```

## Key Features

### Semantic Patterns

Use `$.Subject.predicate.Object` patterns for type-safe operations:

```typescript
import $ from 'format.do'

// Semantic patterns
const result = await format($.Data, {
  // Configuration
})
```

### Type Safety

Full TypeScript support with comprehensive type definitions:

```typescript
import type { FormatOptions, FormatResult } from 'format.do'

const options: FormatOptions = {
  // Type-safe configuration
}

const result: FormatResult = await format(options)
```

### Integration Ready

Seamlessly integrates with other `.do` platform services:

```typescript
import { format } from 'format.do'
import { db, ai } from 'sdk.do'

// Integrate with database
const data = await db.list($.Data)
const result = await format(data)

// Integrate with AI
const processed = await ai.generate('process', { data: result })
```

## TypeScript Support

format.do is built with TypeScript and provides full type safety:

```typescript
import type { FormatOptions, FormatResult, FormatError } from 'format.do'

// Type-safe operations
const options: FormatOptions = {
  // Configuration
}

try {
  const result: FormatResult = await format(options)

} catch (error: FormatError) {
  console.error('Error:', error.message)
}
```

## Best Practices

### Use Semantic Patterns

```typescript
// Good - semantic patterns
const result = await format($.Data, options)

// Avoid - string-based
const result = await format('Data', options)
```

### Handle Errors

```typescript
// Good - error handling
try {
  const result = await format(data)
  return result
} catch (error) {
  console.error('Failed:', error)
  throw error
}
```

### Use TypeScript

```typescript
// Good - type-safe
import type { Options } from 'format.do'
const options: Options = {}

// Avoid - untyped
const options = {}
```
