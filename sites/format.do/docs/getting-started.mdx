# Getting Started with format.do

Learn how to set up and use format.do for data formatting in your applications.

## Installation

Install format.do using pnpm:

```bash
pnpm add format.do
```

## Basic Setup

### 1. Import

```typescript
import { format } from 'format.do'
```

### 2. Configure

```typescript
// Optional configuration
const options = {
  // Configuration options
}
```

### 3. Use

```typescript
const result = await format({
  // Your data
})

console.log('Result:', result)
```

## First Example

Here's a complete working example:

```typescript
import { format } from 'format.do'

async function example() {
  try {
    const result = await format({
      data: 'example input',
      options: {
        format: 'json',
      },
    })

    console.log('Success:', result)
    return result
  } catch (error) {
    console.error('Error:', error)
    throw error
  }
}

// Run example
example()
```

## Configuration Options

format.do supports various configuration options:

```typescript
const options = {
  // Option 1
  option1: 'value',

  // Option 2
  option2: true,

  // Option 3
  option3: {
    nested: 'value',
  },
}

const result = await format(data, options)
```

## Error Handling

Handle errors appropriately:

```typescript
import { format, FormatError } from 'format.do'

try {
  const result = await format(data)
} catch (error) {
  if (error instanceof FormatError) {
    console.error('Domain-specific error:', error.message)
  } else {
    console.error('Unexpected error:', error)
  }
}
```

## Integration with SDK

Use with sdk.do:

```typescript

// Fetch data from database

```
