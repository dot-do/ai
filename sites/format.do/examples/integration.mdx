# Data Formatting Integration Examples

Learn how to integrate format.do with other `.do` platform services.

## Integration with SDK

```typescript
import { format } from 'format.do'
import { db, ai } from 'sdk.do'

// Fetch data from database
const data = await db.list($.Data)

// Process with format.do
const results = await format(data)

// Use AI to analyze
const analysis = await ai.generate('analyze', { results })
```

## Integration with Events

```typescript
import { format } from 'format.do'
import { on, send } from 'sdk.do'

// Process on event
on($.Data.received, async (data) => {
  const result = await format(data)
  send($.Data.processed, result)
})
```

## Integration with Workflows

```typescript
import { format } from 'format.do'
import { every } from 'sdk.do'

// Scheduled processing
every($.Daily, async () => {
  const data = await fetchDailyData()
  const results = await format(data)
  await saveResults(results)
})
```

## Integration with Queue

```typescript
import { format } from 'format.do'
import { queue } from 'queue.do'

const processQueue = queue($.Data.process)

processQueue.process(async (job) => {
  const result = await format(job.data)
  return result
})
```

## Integration with External APIs

```typescript
import { format } from 'format.do'
import { api } from 'sdk.do'

// Fetch from external API
const external = await api('https://api.example.com/data')

// Process with format.do
const result = await format(external)

// Send back
await api('https://api.example.com/results', {
  method: 'POST',
  body: JSON.stringify(result),
})
```
