# format.do API Reference

Complete API documentation for format.do functions, types, and interfaces.

## Functions

### format()

Main function for data formatting.

**Signature:**

```typescript
async function format<T = any>(data: T | FormatInput, options?: FormatOptions): Promise<FormatResult<T>>
```

**Parameters:**

- `data` - Input data to process
- `options` - Optional configuration options

**Returns:** Promise resolving to result object

**Example:**

```typescript
const result = await format({
  data: 'input',
  options: { format: 'json' },
})
```

### format.create()

Create a reusable instance.

**Signature:**

```typescript
function create(config: FormatConfig): FormatInstance
```

**Parameters:**

- `config` - Instance configuration

**Returns:** Instance object

**Example:**

```typescript
const instance = format.create({
  timeout: 30000,
  retries: 3,
})

const result = await instance.process(data)
```

### format.batch()

Process multiple items.

**Signature:**

```typescript
async function batch<T>(items: T[], options?: BatchOptions): Promise<FormatResult<T>[]>
```

**Parameters:**

- `items` - Array of items to process
- `options` - Optional batch options

**Returns:** Promise resolving to array of results

**Example:**

```typescript
const results = await format.batch([item1, item2, item3])
```

### format.stream()

Process data as a stream.

**Signature:**

```typescript
async function* stream<T>(
  data: AsyncIterable<T>,
  options?: StreamOptions
): AsyncGenerator<FormatResult<T>>
```

**Parameters:**

- `data` - Async iterable data source
- `options` - Optional stream options

**Returns:** Async generator yielding results

**Example:**

```typescript
for await (const result of format.stream(dataStream)) {
  console.log('Result:', result)
}
```

## Types

### FormatOptions

Configuration options for format().

```typescript
interface FormatOptions {
  // Format options
  format?: 'json' | 'xml' | 'text'

  // Validation
  validate?: boolean
  strict?: boolean

  // Performance
  timeout?: number
  cache?: boolean
  cacheTTL?: number

  // Error handling
  throwOnError?: boolean
  retries?: number
}
```

### FormatResult

Result object returned by format().

```typescript
interface FormatResult<T = any> {
  // Success indicator
  success: boolean

  // Result data
  data?: T

  // Error information
  error?: string
  code?: string

  // Metadata
  metadata?: {
    duration: number
    timestamp: Date
    cached: boolean
  }
}
```

### FormatConfig

Configuration for creating instances.

```typescript
interface FormatConfig {
  // API configuration
  apiKey?: string
  endpoint?: string

  // Timeouts
  timeout?: number
  retries?: number

  // Defaults
  defaultOptions?: FormatOptions
}
```

### FormatInstance

Instance created by format.create().

```typescript
interface FormatInstance {
  // Process method
  process<T>(data: T, options?: FormatOptions): Promise<FormatResult<T>>

  // Batch method
  batch<T>(items: T[], options?: BatchOptions): Promise<FormatResult<T>[]>

  // Close instance
  close(): Promise<void>

  // Get metrics
  getMetrics(): Metrics
}
```

### FormatError

Custom error class.

```typescript
class FormatError extends Error {
  code: string
  details?: any

  constructor(message: string, code: string, details?: any)
}
```

## Events

### Instance Events

Instances emit events for monitoring:

```typescript
instance.on('start', (data) => {
  console.log('Processing started:', data)
})

instance.on('complete', (result) => {
  console.log('Processing completed:', result)
})

instance.on('error', (error) => {
  console.error('Processing error:', error)
})
```
