# invoices.do API Reference

## Types

### Invoice

```typescript
interface Invoice {
  $id: string
  $type: 'Invoice'
  invoiceNumber: string
  dateIssued: string
  customer: Person | Organization
  totalPaymentDue: PriceSpecification
  paymentDueDate: string
  paymentStatus: PaymentStatus
  referencesOrder?: string
  lineItems?: InvoiceItem[]
}
```

### PaymentStatus

```typescript
type PaymentStatus = 'PaymentDue' | 'PaymentComplete' | 'PaymentPartial' | 'PaymentOverdue'
```

## Functions

### db.create()

```typescript
const invoice = await db.create($.Invoice, invoiceData)
```

### db.get()

```typescript
const invoice = await db.get($.Invoice, 'invoice-id')
```

### db.list()

```typescript
const invoices = await db.list($.Invoice, {
  where: { paymentStatus: 'PaymentDue' },
})
```
