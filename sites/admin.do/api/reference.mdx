# API Reference

Complete API reference for admin.do.

## admin.users.create()

Create a new user.

```typescript
admin.users.create(options: UserOptions): Promise<User>
```

## admin.users.assignRole()

Assign a role to a user.

```typescript
admin.users.assignRole(userId: string, role: string): Promise<void>
```

## admin.users.hasPermission()

Check if user has a permission.

```typescript
admin.users.hasPermission(userId: string, permission: string): Promise<boolean>
```

## admin.config.set()

Set system configuration.

```typescript
admin.config.set(key: string, value: any): Promise<void>
```

## admin.monitor.getMetrics()

Get system metrics.

```typescript
admin.monitor.getMetrics(options: MetricsOptions): Promise<Metrics>
```

## admin.audit.log()

Log an administrative action.

```typescript
admin.audit.log(entry: AuditEntry): Promise<void>
```
