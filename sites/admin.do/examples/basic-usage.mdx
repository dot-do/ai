# Basic Usage

Basic examples for admin.do.

## Example 1: Create User

```typescript
import { admin } from 'admin.do'

const user = await admin.users.create({
  email: 'john@company.com',
  name: 'John Doe',
  role: 'developer',
  permissions: ['platform.read', 'database.write'],
})
```

## Example 2: Assign Role

```typescript
await admin.users.assignRole(user.$id, 'content-editor')
```

## Example 3: Check Permission

```typescript
const canPublish = await admin.users.hasPermission(user.$id, 'content.publish')

if (canPublish) {
  console.log('User can publish content')
}
```

## Example 4: Set Configuration

```typescript
await admin.config.set('feature_flags', {
  newUI: true,
  betaFeatures: false,
})
```

## Example 5: View Metrics

```typescript
const metrics = await admin.monitor.getMetrics({
  period: '24h',
  metrics: ['cpu', 'memory', 'requests'],
})

console.log(`CPU: ${metrics.cpu.average}%`)
```
