# Troubleshooting

Common issues and solutions for admin.do.

## Authentication Issues

### Users Cannot Log In

**Problem**: Users receive authentication errors

**Solutions**:

- Verify session secret is configured correctly
- Check database connection
- Confirm user account is active
- Review audit log for failed attempts

### Session Expires Too Quickly

**Problem**: Users logged out frequently

**Solutions**:

- Increase session timeout in configuration
- Check Redis connection for session store
- Verify cookie settings (domain, path)

## Permission Issues

### User Cannot Access Resource

**Problem**: Permission denied errors

**Solutions**:

- Verify user has required role/permissions
- Check permission inheritance chain
- Review RBAC configuration
- Confirm resource-level permissions

### Permission Changes Not Taking Effect

**Problem**: Updated permissions not working

**Solutions**:

- Invalidate user sessions after permission changes
- Clear permission cache
- Restart application if needed

## Monitoring Issues

### Metrics Not Updating

**Problem**: Dashboard shows stale data

**Solutions**:

- Check metrics collection service
- Verify time-series database connection
- Review metric ingestion logs
- Confirm data retention settings

### Missing Audit Logs

**Problem**: Actions not appearing in audit log

**Solutions**:

- Verify audit logging is enabled
- Check audit log database connection
- Review audit log service logs
- Confirm actions are properly instrumented
