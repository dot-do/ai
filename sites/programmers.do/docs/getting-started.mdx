# Getting Started with programmers.do

Build your first Programmer Collective agent in 5 minutes.

## Installation

```bash
pnpm add sdk.do
```

## Your First Programmer Collective Agent

```typescript
import $, { agent } from 'sdk.do'

const agentInstance = await agent.create({
  $type: 'Agent',
  name: 'Programmer Collective',
  capabilities: ['team_coordination', 'pair_programming', 'code_collaboration', 'knowledge_transfer', 'mentoring'],
  model: 'claude-sonnet-4.5',
})

// Execute task
const result = await agentInstance.execute({
  task: 'Team code review',
  context: {
    /* your context */
  },
})
```

## Core Capabilities

- **Team Coordination**: Development team coordination
- **Pair Programming**: pair programming
- **Code Collaboration**: knowledge sharing
