# Programmer Agent Collective API Reference

Complete API documentation for programmers.do.

## Agent Creation

```typescript
const programmercollective = await agent.create({
  $role: $.ProgrammerCollective,
  name: string,
  capabilities: string[],
  tools: Record<string, Tool>,
  model: string
})
```

## Execution

```typescript
const result = await programmercollective.execute({
  task: string,
  context?: Context
})
```

## Configuration

```typescript
await programmercollective.configure({
  model?: string,
  temperature?: number,
  capabilities?: string[]
})
```

## Events

```typescript
programmercollective.on('task_complete', (result) => {
  console.log(result)
})
```
