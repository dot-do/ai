# programmers.do

**Programmer Agent Collective for Autonomous Operations**

programmers.do provides autonomous programmer agent collectives that handle Team collaboration, pair programming, code reviews, knowledge sharing, mentoring. Build digital workers that can coordinate team code review with the same expertise as experienced professionals.

## Quick Start

```bash
pnpm add sdk.do
```

```typescript
import $, { agent } from 'sdk.do'

const programmercollective = await agent.create({
  $type: 'Agent',
  $role: $.ProgrammerCollective,
  name: 'Programmer Agent Collective',
  capabilities: ['team_coordination', 'pair_programming', 'code_collaboration', 'knowledge_transfer', 'peer_review', 'mentoring'],
  tools: ['github', 'slack', 'jira', 'confluence', 'pair_programming_tools'],
  model: 'claude-sonnet-4.5',
})

const result = await programmercollective.execute({
  task: 'Coordinate team code review',
  context: {
    /* your context */
  },
})
```

})
