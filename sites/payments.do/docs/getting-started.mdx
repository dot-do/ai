# Getting Started with payments.do

## Installation

```bash
pnpm add sdk.do
```

## Your First Payment

```typescript
import $, { db } from 'sdk.do'

const payment = await db.create($.PaymentRecord, {
  $type: 'PaymentRecord',
  paymentId: 'PAY-2025-001',
  paymentDate: new Date().toISOString(),
  paymentMethod: {
    $type: 'PaymentCard',
    name: 'Visa ending in 1234',
  },
  amount: {
    $type: 'MonetaryAmount',
    value: 299.99,
    currency: 'USD',
  },
  paymentStatus: 'PaymentComplete',
})
```

## Key Concepts

- **Payment Methods**: Cards, bank accounts, digital wallets
- **Authorization**: Reserve funds before capture
- **Capture**: Complete the payment
- **Refunds**: Return funds to customer
