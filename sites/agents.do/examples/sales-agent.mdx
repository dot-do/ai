# Sales Agent Example - SDR/BDR

Build an autonomous Sales Development Representative (SDR) agent that researches leads, sends personalized outreach, qualifies prospects, and books meetings.

## Overview

This example creates a complete SDR agent that:

1. **Researches** companies and decision makers
2. **Qualifies** leads based on fit criteria
3. **Personalizes** outreach based on insights
4. **Sends** emails with appropriate follow-up cadence
5. **Qualifies** responses and interest level
6. **Books** discovery meetings with qualified prospects
7. **Updates** CRM with all activities

## The Sales Agent

## Complete Workflow Example

```typescript
// End-to-end lead processing
async function processLead(lead: Lead) {
  // Step 1: Research
  console.log('Researching company...')
  const research = await sarahSDR.execute({
    task: 'research_company',
    input: { company: lead.company },
  })

  // Step 2: Qualify
  console.log('Qualifying lead...')
  const qualification = await sarahSDR.execute({
    task: 'qualify_lead',
    input: { lead, research },
  })

  if (!qualification.qualified) {
    console.log('Lead not qualified')
    await sarahSDR.tool.crm.updateLead(lead.id, {
      status: 'disqualified',
      reason: qualification.reasoning,
    })
    return
  }

  // Step 3: Craft outreach
  console.log('Crafting personalized email...')
  const email = await sarahSDR.execute({
    task: 'craft_outreach_email',
    input: { lead, research, qualification },
  })

  // Step 4: Request approval and send
  console.log('Requesting approval...')
  const approved = await sarahSDR.requestApproval({
    action: 'send_email',
    email: email,
  })

  if (approved) {
    await sarahSDR.tool.email.send(email)
    await sarahSDR.tool.crm.logActivity(lead.id, {
      type: 'email_sent',
      subject: email.subject,
      timestamp: new Date(),
    })
    console.log('Email sent!')
  }

  // Step 5: Schedule follow-ups
  await sarahSDR.execute({
    task: 'schedule_follow_up_sequence',
    input: { lead },
  })

  console.log('Lead processing complete')
}

// Process new lead
await processLead({
  id: 'lead-123',
  company: 'Acme Corp',
  contact: 'John Doe',
  email: 'john@acme.com',
  source: 'whitepaper_download',
  created_at: new Date(),
})
```
