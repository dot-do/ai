# Getting Started with customers.do

## Installation

```bash
pnpm add sdk.do
```

## Your First Customer

```typescript
import $, { db } from 'sdk.do'

const customer = await db.create($.Person, {
  $type: 'Person',
  name: 'Jane Smith',
  email: 'jane@example.com',
  telephone: '+1-555-0123',
  totalOrders: 0,
  totalSpent: 0,
  customerSince: new Date().toISOString(),
})
```

## Key Concepts

- **Customer Profile**: Core customer information
- **Segmentation**: Group customers by behavior
- **Loyalty Programs**: Reward repeat customers
- **Customer Analytics**: Track metrics and trends
