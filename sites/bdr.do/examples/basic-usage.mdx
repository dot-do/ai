# Basic Usage

Basic examples for working with BDR agents.

## Example 1: Create a BDR Agent

```typescript
import { $, bdr } from 'sdk.do'

const alex = await bdr.create({
  $type: 'BusinessDevelopmentRepresentative',
  name: 'Alex Morgan',
  focus: {
    verticals: ['Technology', 'FinTech'],
    partnershipTypes: ['Strategic Alliances'],
    dealSize: { min: 100000, target: 500000 },
  },
  model: 'claude-sonnet-4.5',
})
```

## Example 2: Identify Opportunities

```typescript
const opportunities = await alex.identifyOpportunities({
  criteria: {
    industry: 'FinTech',
    companySize: { min: 500, max: 10000 },
    complementary: true,
  },
  limit: 10,
})
```

## Example 3: Develop Partnership

```typescript
const partnership = await alex.developPartnership({
  company: 'Strategic Partner Inc',
  type: 'Integration Partner',
  approach: {
    timeline: '6_months',
    strategy: 'collaborative',
  },
})
```

## Example 4: Create Proposal

```typescript
const proposal = await alex.createProposal({
  partner: 'Strategic Partner Inc',
  type: 'Integration Partnership',
  structure: {
    term: '3_years',
    revenue: { model: 'revenue_share', split: { us: 0.7, them: 0.3 } },
  },
})
```

## Example 5: Monitor Partnership Health

```typescript
const health = await alex.monitorPartnership({
  partnerId: 'partner-123',
})

console.log(`Health: ${health.overallHealth}/100`)
```
