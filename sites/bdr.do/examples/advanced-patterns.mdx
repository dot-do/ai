# Advanced Patterns

Advanced patterns for BDR agents.

## Multi-Stakeholder Engagement

```typescript
import { $, bdr } from 'sdk.do'

const stakeholders = await alex.mapStakeholders({
  company: 'Strategic Partner Inc',
})

for (const stakeholder of stakeholders) {
  await alex.createPersonalizedOutreach({
    stakeholder,
    approach: stakeholder.role === 'decision_maker' ? 'executive' : 'collaborative',
  })
}
```

## Automated Partnership Workflows

```typescript
import { $, bdr, on } from 'sdk.do'

on($.Partnership.opportunity_identified, async (event) => {
  await alex.researchPartner(event.partner)
  await alex.createOutreachStrategy(event.partner)
})

on($.Partnership.proposal_sent, async (event) => {
  await alex.scheduleFollowUp({ partnerId: event.partnerId, days: 7 })
})
```

## Channel Partner Programs

```typescript
const program = await alex.createChannelProgram({
  name: 'Reseller Partner Program',
  tiers: {
    authorized: { discount: 0.25, marketingFunds: 5000 },
    silver: { discount: 0.3, marketingFunds: 15000 },
    gold: { discount: 0.35, marketingFunds: 50000 },
  },
})

await alex.recruitPartners({ program, targets: 50 })
```
