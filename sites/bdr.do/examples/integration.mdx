# Integration Examples

Examples of integrating BDR agents with other services.

## CRM Integration

```typescript
import { $, bdr, db } from 'sdk.do'

const partnership = await alex.developPartnership({
  company: 'Strategic Partner Inc',
  type: 'Integration Partner',
})

// Sync to CRM
await db.create($.Partnership, {
  ...partnership,
  crmId: await syncToCRM(partnership),
})
```

## Email Automation

```typescript
import { $, bdr, send } from 'sdk.do'

await alex.sendEmail({
  to: 'sarah@partner.com',
  subject: 'Partnership Opportunity',
  template: 'partnership_intro',
  variables: { company: 'Strategic Partner Inc' },
})

// Track engagement
on($.Email.opened, async (event) => {
  await alex.handleEmailEngagement({ partnerId: event.partnerId })
})
```

## Calendar Integration

```typescript
const meeting = await alex.scheduleMeeting({
  partner: 'Strategic Partner Inc',
  type: 'exploratory_call',
  duration: 30,
  attendees: ['sarah@partner.com'],
})
```
