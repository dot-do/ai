# bdr.do

**Business Development Representative Agent**

bdr.do enables you to create autonomous BDR agents that focus on strategic business development - partnerships, channel relationships, market expansion, and enterprise accounts. Unlike SDRs who focus on volume-based lead generation, BDRs cultivate high-value relationships and strategic opportunities.

## What is a BDR Agent?

A BDR agent is an autonomous strategic sales professional that:

- **Identifies Partnerships**: Finds mutually beneficial partnership opportunities
- **Develops Channels**: Builds and nurtures channel partner relationships
- **Expands Markets**: Enters new markets and territories strategically
- **Cultivates Enterprise**: Manages complex, multi-stakeholder enterprise deals
- **Orchestrates Alliances**: Coordinates strategic alliances and co-selling
- **Builds Networks**: Develops industry networks and thought leadership

## Quick Start

```bash
pnpm add sdk.do
```

```typescript
import $, { bdr } from 'sdk.do'

// Create a BDR agent
const david = await bdr.create({
  $type: 'BDRAgent',
  name: 'David Park',
  persona: {
    background: '10 years in enterprise software, former consultant at McKinsey',
    expertise: ['strategic partnerships', 'channel development', 'enterprise sales'],
    personality: 'strategic, relationship-focused, consultative',
    network: 'extensive C-level connections in tech and finance',
  },
  focus: {
    partnerships: ['technology_partners', 'resellers', 'system_integrators'],
    enterprise_accounts: { deal_size: { min: 500000 } },
    markets: ['financial_services', 'healthcare', 'manufacturing'],
  },
  goals: [
    { metric: 'partnership_deals_signed', target: 3, period: '$.Quarterly' },
    { metric: 'channel_pipeline', target: 2000000, period: '$.Quarterly' },
    { metric: 'enterprise_opportunities', target: 5, period: '$.Quarterly' },
  ],
  model: 'claude-sonnet-4.5',
})

// BDR autonomously develops strategic relationships
await david.work({
  priority: 'partnership_development',
  focus: 'cloud_infrastructure_partners',
})
```

## Key Features

### Partnership Development

Identify and cultivate strategic partnerships:

```typescript
const partnerships = await bdr.create({
  name: 'Emily Zhang',
  role: 'Partnership Development',
  partnership_strategy: {
    types: {
      technology: {
        focus: 'integration_partners',
        criteria: {
          market_overlap: 'high',
          product_complementary: true,
          customer_base_size: { min: 1000 },
        },
      },
      channel: {
        focus: 'resellers_and_integrators',
        criteria: {
          vertical_expertise: ['fintech', 'healthcare'],
          geographic_coverage: ['north_america', 'europe'],
          revenue_size: { min: 10000000 },
        },
      },
      strategic: {
        focus: 'co_selling_alliances',
        criteria: {
          market_leader: true,
          enterprise_relationships: true,
          aligned_roadmap: true,
        },
      },
    },
    evaluation_framework: {
      strategic_fit: { weight: 0.3 },
      revenue_potential: { weight: 0.25 },
      go_to_market_alignment: { weight: 0.2 },
      execution_complexity: { weight: 0.15 },
      competitive_dynamics: { weight: 0.1 },
    },
    development_process: {
      stages: [
        { name: 'identification', duration: '2_weeks' },
        { name: 'initial_contact', duration: '2_weeks' },
        { name: 'evaluation', duration: '4_weeks' },
        { name: 'negotiation', duration: '6_weeks' },
        { name: 'onboarding', duration: '4_weeks' },
      ],
      stakeholders: ['bd_lead', 'product', 'legal', 'executive'],
    },
  },
})

// Autonomous partnership development
await partnerships.developPartnership({
  company: 'Acme Cloud Solutions',
  type: 'technology',
  objective: 'joint_go_to_market',
})
```

      touchpoints: {
        executive_briefings: '$.Quarterly',

        thought_leadership: true,
        industry_insights: true,
        peer_connections: true,
        roi_analysis: true,

      average_cycle: 180, // days
