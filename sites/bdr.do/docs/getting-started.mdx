# Getting Started

Get started with BDR (Business Development Representative) agents in the `.do` platform and build autonomous partnership development teams.

## Installation

Install the SDK in your project:

```bash
pnpm add sdk.do
```

Or use the CLI:

```bash
pnpm add -g cli.do
```

## Environment Setup

### Required Configuration

Create a `.env` file in your project root:

```bash
# LLM Provider (required for AI-powered BDR agents)
OPENAI_API_KEY=sk-...
ANTHROPIC_API_KEY=sk-ant-...

# CRM Integration (optional but recommended)
SALESFORCE_API_KEY=...
HUBSPOT_API_KEY=...

# Communication Tools
LINKEDIN_API_KEY=...
EMAIL_SMTP_HOST=smtp.gmail.com
EMAIL_SMTP_USER=your-email@company.com
EMAIL_SMTP_PASS=...

# Database (for partnership tracking)
DATABASE_URL=...
```

### API Keys

BDR agents require:

- **LLM Provider**: OpenAI (GPT-5) or Anthropic (Claude Sonnet 4.5) for AI reasoning
- **CRM**: Salesforce, HubSpot, or similar for partnership tracking
- **LinkedIn**: For social selling and relationship building
- **Email**: SMTP for outreach and communication

## Creating Your First BDR Agent

### Basic BDR Agent

Create a BDR agent with default configuration:

```typescript
import { $, bdr } from 'sdk.do'

const alex = await bdr.create({
  $type: 'BusinessDevelopmentRepresentative',
  name: 'Alex Morgan',
  persona: {
    background: 'Former strategy consultant, 8 years in partnerships',
    style: 'strategic, relationship-focused, long-term oriented',
    expertise: ['partnership_strategy', 'channel_development', 'alliance_management'],
  },
  focus: {
    verticals: ['Technology', 'FinTech', 'Enterprise Software'],
    partnershipTypes: ['Strategic Alliances', 'Channel Partners', 'Integration Partners'],
    dealSize: { min: 100000, target: 500000 },
  },
  quota: {
    partnershipsSignedPerYear: 12,
    pipelineValue: 5000000,
    revenueFromPartners: 2000000,
  },
  tools: ['linkedin', 'email', 'crm', 'contract_management'],
  model: 'claude-sonnet-4.5',
})

console.log(`BDR agent ${alex.name} created successfully`)
```

### Identify Partnership Opportunities

Find potential partners:

```typescript
const opportunities = await alex.identifyOpportunities({
  criteria: {
    industry: ['FinTech', 'Banking'],
    companySize: { min: 500, max: 10000 },
    customerOverlap: { min: 0.2, max: 0.6 },
    complementary: true,
    marketPosition: ['Leader', 'Challenger'],
  },
  limit: 10,
})

console.log(`Found ${opportunities.length} partnership opportunities`)
opportunities.forEach((opp) => {
  console.log(`- ${opp.company}: ${opp.opportunity} (Fit: ${opp.fitScore}/100)`)
})
```
