# API Reference

Complete API reference for BDR (Business Development Representative) agents.

## bdr.create()

Create a new BDR agent.

```typescript
bdr.create(options: BDROptions): Promise<BDRAgent>
```

**Parameters:**

- `options.name` (string): Agent name
- `options.persona` (object): Agent persona and background
- `options.focus` (object): Focus areas (verticals, partnership types, deal size)
- `options.quota` (object): Performance quotas
- `options.tools` (string[]): Available tools
- `options.model` (string): LLM model ('gpt-5' | 'claude-sonnet-4.5')

## identifyOpportunities()

Identify partnership opportunities.

```typescript
agent.identifyOpportunities(criteria: OpportunityCriteria): Promise<Opportunity[]>
```

**Parameters:**

- `criteria.industry` (string | string[]): Target industries
- `criteria.companySize` (object): Company size range
- `criteria.complementary` (boolean): Require complementary offerings
- `criteria.limit` (number): Max opportunities to return

## developPartnership()

Develop a strategic partnership.

```typescript
agent.developPartnership(options: PartnershipOptions): Promise<Partnership>
```

**Parameters:**

- `options.company` (string): Partner company name
- `options.type` (string): Partnership type
- `options.approach` (object): Development approach

## createProposal()

Create a partnership proposal.

```typescript
agent.createProposal(options: ProposalOptions): Promise<Proposal>
```

**Parameters:**

- `options.partner` (string): Partner company name
- `options.type` (string): Partnership type
- `options.structure` (object): Partnership structure (term, revenue, etc.)

## monitorPartnership()

Monitor partnership health.

```typescript
agent.monitorPartnership(options: MonitorOptions): Promise<PartnerHealth>
```

**Parameters:**

- `options.partnerId` (string): Partnership ID

## getMetrics()

Get BDR performance metrics.

```typescript
agent.getMetrics(options: MetricsOptions): Promise<BDRMetrics>
```

**Parameters:**

- `options.period` (string): Time period ('last_7_days' | 'last_30_days' | 'last_quarter' | 'last_year')
