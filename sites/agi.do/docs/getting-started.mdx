# Getting Started

Build your first AGI agent with advanced reasoning, multi-domain expertise, and self-improvement capabilities in under 10 minutes.

## Installation

Install the SDK in your project:

```bash
pnpm add sdk.do
```

Or use the CLI:

```bash
pnpm add -g cli.do
```

## API Keys

AGI agents use LLM providers for reasoning and learning. Set up your API keys:

### Anthropic (Claude Models)

1. Create an account at [console.anthropic.com](https://console.anthropic.com)
2. Generate an API key from the API settings
3. Set the environment variable:

```bash
export ANTHROPIC_API_KEY=sk-ant-...
```

### OpenAI (GPT Models)

1. Create an account at [platform.openai.com](https://platform.openai.com)
2. Generate an API key from the API keys section
3. Set the environment variable:

```bash
export OPENAI_API_KEY=sk-...
```

## Your First AGI Agent

Create a simple AGI agent with advanced reasoning:

```typescript
import { $, agent } from 'sdk.do'

// Create AGI agent
const agiAgent = await agent.create({
  $type: $.Agent,
  $role: $.AGIAgent,
  name: 'Research Assistant',
  capabilities: ['advanced_reasoning', 'multi_domain_expertise'],
  model: 'claude-sonnet-4.5',
  temperature: 0.7,
})

// Execute task
const result = await agiAgent.execute({
  task: 'Analyze the impact of AI on software development productivity',
  context: {
    timeframe: '2020-2025',
    focus: ['automation', 'code generation', 'testing'],
  },
})

console.log(result.analysis)
console.log(result.reasoning)
```

Output:

```json
{
  "analysis": "AI has significantly increased software development productivity...",
  "reasoning": [
    "Step 1: Identified key AI technologies affecting development",
    "Step 2: Analyzed productivity metrics across the timeframe",
    "Step 3: Evaluated impact on automation and code generation",
    "Step 4: Synthesized findings into comprehensive analysis"
  ],
  "confidence": 0.92
}
```

## Core Capabilities

AGI agents support six core capabilities:

### 1. Advanced Reasoning

Multi-step reasoning with chain-of-thought and tree-of-thought strategies:

```typescript
import { $, agent } from 'sdk.do'

const reasoner = await agent.create({
  $role: $.AGIAgent,
  capabilities: ['advanced_reasoning'],
  reasoning: {
    strategy: 'chain-of-thought',
    depth: 5,
    explainSteps: true,
  },
})

const solution = await reasoner.reason({
  problem: 'Design a scalable notification system',
  constraints: ['10M users', 'real-time delivery', 'multi-channel'],
})

console.log(solution.steps)
console.log(solution.final)
```

### 2. Multi-Domain Expertise

Cross-domain knowledge integration:

```typescript
import { $, agent } from 'sdk.do'

const expert = await agent.create({
  $role: $.AGIAgent,
  capabilities: ['multi_domain_expertise'],
  domains: ['software_engineering', 'business_strategy', 'data_science', 'system_design'],
})

const analysis = await expert.analyze({
  problem: 'Launch strategy for a new SaaS product',
  consider: ['technical architecture', 'market positioning', 'pricing', 'go-to-market'],
})
```

### 3. Self-Improvement

Agents that learn from experience:

```typescript
import { $, agent } from 'sdk.do'

const learner = await agent.create({
  $role: $.AGIAgent,
  capabilities: ['self_improvement'],
  learning: {
    enabled: true,
    strategy: 'experience_replay',
    memory_size: 10000,
  },
})

// Agent improves with feedback
const result = await learner.execute({ task: 'Code review' })

await learner.feedback({
  task_id: result.id,
  outcome: 'success',
  metrics: { accuracy: 0.95 },
})
```

### 4. Meta-Learning

Learn how to learn more effectively:

```typescript
import { $, agent } from 'sdk.do'

const metaLearner = await agent.create({
  $role: $.AGIAgent,
  capabilities: ['meta_learning'],
  learning: {
    strategy: 'meta',
    adapt_to: ['task_type', 'domain', 'user_feedback'],
  },
})

// Agent adapts learning strategy based on task characteristics
```

### 5. Research Synthesis

Autonomous research and information gathering:

```typescript
import { $, agent, api } from 'sdk.do'

const researcher = await agent.create({
  $role: $.AGIAgent,
  capabilities: ['research_synthesis'],
  tools: {
    web_search: api.get('https://search.api.example.com'),
    arxiv: api.get('https://arxiv.org/api'),
  },
})

const report = await researcher.research({
  topic: 'Vector database optimization techniques',
  depth: 'comprehensive',
  deliverable: 'technical_report',
})
```

### 6. Strategic Thinking

Long-term planning and scenario analysis:

```typescript
import { $, agent } from 'sdk.do'

const strategist = await agent.create({
  $role: $.AGIAgent,
  capabilities: ['strategic_thinking'],
  planning: {
    horizon: '3_years',
    scenarios: 5,
    optimize_for: 'long_term_value',
  },
})

const strategy = await strategist.plan({
  objective: 'Achieve market leadership',
  current_state: { market_share: 0.15, revenue: '$10M' },
  constraints: { funding: '$50M', timeline: '36 months' },
})
```

## Configuration Options

### Basic Configuration

```typescript
import { $, agent } from 'sdk.do'

const agiAgent = await agent.create({
  // Identity
  $type: $.Agent,
  $role: $.AGIAgent,
  name: 'Strategic Advisor',

  // Capabilities
  capabilities: ['advanced_reasoning', 'multi_domain_expertise', 'strategic_thinking'],

  // Model configuration
  model: 'claude-sonnet-4.5',
  temperature: 0.7,
  maxTokens: 8192,

  // Reasoning configuration
  reasoning: {
    strategy: 'chain-of-thought',
    depth: 5,
    breadth: 3,
    explainSteps: true,
  },
})
```

### Advanced Configuration

```typescript
import { $, agent, db } from 'sdk.do'

const agiAgent = await agent.create({
  $type: $.Agent,
  $role: $.AGIAgent,
  name: 'Advanced AGI System',

  // Multiple capabilities
  capabilities: ['advanced_reasoning', 'multi_domain_expertise', 'self_improvement', 'meta_learning', 'research_synthesis', 'strategic_thinking'],

  // Domain expertise
  domains: ['software_engineering', 'data_science', 'business_strategy', 'economics', 'psychology', 'system_design'],

  // Learning configuration
  learning: {
    enabled: true,
    strategy: 'experience_replay',
    memory_size: 10000,
    update_frequency: 'task',
    adapt_to: ['task_type', 'domain', 'user_feedback'],
  },

  // Reasoning configuration
  reasoning: {
    strategy: 'graph-of-thought',
    depth: 7,
    breadth: 4,
    explainSteps: true,
    considerAlternatives: true,
  },

  // Planning configuration
  planning: {
    horizon: '3_years',
    scenarios: 5,
    monte_carlo_iterations: 1000,
    optimize_for: 'long_term_value',
  },

  // Knowledge integration
  knowledge_integration: 'semantic_fusion',
  knowledge_graph: {
    enabled: true,
    store: db,
  },

  // Model configuration
  model: 'claude-sonnet-4.5',
  temperature: 0.7,
  maxTokens: 8192,
})
```

## Environment Variables

Create a `.env` file in your project root:

```bash
# Required: API keys for LLM providers
ANTHROPIC_API_KEY=sk-ant-...
OPENAI_API_KEY=sk-...

# Optional: AGI configuration
AGI_DEFAULT_MODEL=claude-sonnet-4.5
AGI_DEFAULT_TEMPERATURE=0.7
AGI_ENABLE_LEARNING=true
AGI_ENABLE_KNOWLEDGE_GRAPH=true
AGI_MEMORY_SIZE=10000

# Optional: Performance tuning
AGI_MAX_REASONING_DEPTH=7
AGI_MAX_REASONING_BREADTH=4
AGI_CACHE_ENABLED=true
AGI_CACHE_TTL=3600
```

## Error Handling

Handle errors gracefully:

```typescript
import { $, agent } from 'sdk.do'

try {
  const agiAgent = await agent.create({
    $role: $.AGIAgent,
    capabilities: ['advanced_reasoning'],
    model: 'claude-sonnet-4.5',
  })

  const result = await agiAgent.execute({
    task: 'Complex analysis task',
  })

  console.log(result)
} catch (error) {
  if (error.code === 'INSUFFICIENT_CAPABILITIES') {
    console.error('Agent lacks required capabilities for this task')
  } else if (error.code === 'REASONING_DEPTH_EXCEEDED') {
    console.error('Reasoning depth limit exceeded, simplify task')
  } else if (error.code === 'PROVIDER_UNAVAILABLE') {
    console.error('LLM provider is currently unavailable')
  } else {
    console.error('Task execution failed:', error.message)
  }
}
```

## Common Patterns

### Pattern 1: Combining Capabilities

```typescript
import { $, agent } from 'sdk.do'

// Combine reasoning + expertise + research
const comprehensive = await agent.create({
  $role: $.AGIAgent,
  capabilities: ['advanced_reasoning', 'multi_domain_expertise', 'research_synthesis'],
})

const analysis = await comprehensive.execute({
  task: 'Comprehensive market analysis with strategic recommendations',
})
```

### Pattern 2: Learning Agent

```typescript
import { $, agent } from 'sdk.do'

// Self-improving agent
const learner = await agent.create({
  $role: $.AGIAgent,
  capabilities: ['advanced_reasoning', 'self_improvement'],
  learning: { enabled: true },
})

// Execute and provide feedback loop
for (const task of tasks) {
  const result = await learner.execute(task)

  await learner.feedback({
    task_id: result.id,
    outcome: evaluateResult(result),
    metrics: calculateMetrics(result),
  })
}
```

### Pattern 3: Multi-Agent System

```typescript
import { $, agent } from 'sdk.do'

// Specialized agents
const researcher = await agent.create({
  $role: $.AGIAgent,
  capabilities: ['research_synthesis'],
})

const strategist = await agent.create({
  $role: $.AGIAgent,
  capabilities: ['strategic_thinking'],
})

const analyst = await agent.create({
  $role: $.AGIAgent,
  capabilities: ['advanced_reasoning', 'multi_domain_expertise'],
})

// Orchestrated workflow
const research = await researcher.research({ topic: 'Market trends' })
const strategy = await strategist.plan({ insights: research })
const analysis = await analyst.analyze({ strategy: strategy })
```
