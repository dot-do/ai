# Basic Usage Examples

Simple examples to get started with agi.do.

## Example 1: Simple Reasoning Task

Perform basic reasoning with an AGI agent:

```typescript
import { agent } from 'sdk.do'

// Create AGI agent
const reasoningAgent = await agent.create({
  $type: $.Agent,
  $role: $.AGIAgent,
  name: 'Simple Reasoner',
  capabilities: ['advanced_reasoning'],
  model: 'claude-sonnet-4.5',
})

// Execute simple reasoning task
const result = await reasoningAgent.reason({
  problem: 'If a tree falls in the forest and no one is around, does it make a sound?',
  approach: 'philosophical_analysis',
})

console.log(result.conclusion)
console.log(result.reasoning_steps)
// Output:
// {
//   conclusion: "Yes, it makes a sound wave, but sound as a perceptual experience requires an observer.",
//   reasoning_steps: [
//     "Define 'sound' - physical phenomenon vs. perceptual experience",
//     "Analyze physical perspective: sound waves are produced",
//     "Analyze philosophical perspective: perception requires observer",
//     "Synthesize both perspectives into coherent conclusion"
//   ]
// }
```

## Example 2: Multi-Step Analysis

Break down complex problems into analytical steps:

```typescript
import { $, agent } from 'sdk.do'

const analyst = await agent.create({
  $role: $.AGIAgent,
  capabilities: ['advanced_reasoning', 'multi_domain_expertise'],
  reasoning: {
    strategy: 'chain-of-thought',
    depth: 5,
  },
})

// Analyze a business problem
const analysis = await analyst.analyze({
  problem: 'Why did our user retention drop 15% last quarter?',
  context: {
    product: 'SaaS analytics platform',
    metrics: {
      churn_rate: 0.18,
      nps: 42,
      feature_adoption: 0.65,
    },
  },
})

console.log(analysis)
// Output:
// {
//   root_causes: [
//     { cause: 'Poor onboarding', confidence: 0.85, evidence: [...] },
//     { cause: 'Missing key features', confidence: 0.72, evidence: [...] },
//     { cause: 'Price increase', confidence: 0.55, evidence: [...] }
//   ],
//   recommendations: [
//     'Redesign onboarding flow with progressive disclosure',
//     'Prioritize top 3 requested features',
//     'Implement customer success program'
//   ],
//   reasoning: { steps: [...], alternatives_considered: [...] }
// }
```

## Example 9: Pattern Recognition

Identify patterns in data and behavior:

```typescript

  $role: $.AGIAgent.Analyst,

  domains: ['data_science', 'psychology', 'business_strategy'],
})

// Analyze customer churn patterns
const customerData = await db.list($.Customer, {
  where: { churned: true },
  limit: 1000,

const patterns = await patternAgent.analyze({
```
