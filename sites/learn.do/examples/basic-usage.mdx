# Basic Usage

Fundamental learning platform operations.

## Creating a Course

```typescript
import { learn } from 'learn.do'

const course = await learn.createCourse({
  name: 'SDK Fundamentals',
  description: 'Learn SDK basics',
  level: 'beginner',
  duration: 'PT8H',
  modules: [
    {
      name: 'Getting Started',
      lessons: [
        {
          title: 'Introduction',
          content: '# Introduction\n\nWelcome to SDK...',
          duration: 'PT30M',
          exercises: [
            {
              question: 'What is SDK?',
              type: 'multiple-choice',
              options: ['A', 'B', 'C'],
              correct: 'A',
            },
          ],
        },
      ],
    },
  ],
})
```

## Enrolling Users

```typescript
// Enroll in course
await learn.enroll(user, course)

// Check enrollment
const enrolled = await learn.isEnrolled(user, course)

// Get enrolled courses
const courses = await learn.getUserCourses(user)
```

## Tracking Progress

```typescript
// Complete a lesson
await learn.completeLesson(user, lesson, {
  score: 95,
  timeSpent: 1800,
})

// Get progress
const progress = await learn.getProgress(user, course)
console.log(`Progress: ${progress.percentage}%`)
```
