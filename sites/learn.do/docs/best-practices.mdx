# learn.do Best Practices

Best practices for effective learning experiences.

## Course Structure

### Create Clear Learning Objectives

```typescript
const course = await learn.createCourse({
  name: 'Database Mastery',
  objectives: ['Understand database design principles', 'Master CRUD operations', 'Implement complex queries', 'Optimize performance'],
  prerequisites: ['sql-basics', 'programming-fundamentals'],
})
```

### Break Content into Modules

```typescript
const modules = [
  {
    name: 'Foundations',
    lessons: [
      /* fundamentals */
    ],
    duration: 'PT2H',
  },
  {
    name: 'Advanced Topics',
    lessons: [
      /* advanced */
    ],
    duration: 'PT4H',
  },
  {
    name: 'Real-World Projects',
    lessons: [
      /* projects */
    ],
    duration: 'PT6H',
  },
]
```

## Assessment Design

### Include Varied Question Types

```typescript
const assessment = await learn.createAssessment({
  name: 'SDK Fundamentals Test',
  questions: [
    {
      type: 'multiple-choice',
      question: 'What does db.create() return?',
      options: ['void', 'Promise', 'Entity', 'Error'],
      correct: 'Entity',
    },
    {
      type: 'code',
      question: 'Implement a function to create a business entity',
      solution: 'const business = await db.create($.Business, data)',
      tests: [
        /* unit tests */
      ],
    },
  ],
})
```
