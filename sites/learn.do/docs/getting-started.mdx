# Getting Started with learn.do

Complete guide to setting up your learning platform with learn.do.

## Installation

```bash
pnpm add learn.do
```

## Environment Setup

```bash
# Required
LEARN_DATABASE_URL=postgres://user:pass@localhost:5432/learn
LEARN_OPENAI_API_KEY=sk-your-key

# Optional
LEARN_VIDEO_CDN=https://cdn.learn.do
LEARN_LAB_ENV=sandbox
```

## Your First Learning Path

```typescript
import $, { learn } from 'learn.do'

async function createLearningSystem() {
  // Create a course
  const course = await learn.createCourse({
    name: 'SDK Fundamentals',
    description: 'Learn the basics of our SDK',
    level: 'beginner',
    duration: 'PT8H',
    modules: [
      {
        name: 'Introduction',
        lessons: [
          {
            title: 'What is SDK',
            content: '...',
            duration: 'PT30M',
          },
        ],
      },
    ],
  })

  // Create learning path
  const path = await learn.createPath({
    name: 'Developer Journey',
    courses: [course.$id, 'advanced-sdk', 'deployment'],
  })

  // Enroll user
  await learn.enroll(user, course)

  // Track progress
  await learn.trackProgress(user, course, {
    lessonId: 'intro-lesson-1',
    completed: true,
  })
}
```
