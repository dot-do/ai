# API Reference

## Core Functions

### learn.createCourse()

Create a new course.

```typescript
function createCourse(data: CourseData): Promise<Course>
```

### learn.enroll()

Enroll a user in a course.

```typescript
function enroll(user: User, course: Course): Promise<Enrollment>
```

### learn.trackProgress()

Track user progress.

```typescript
function trackProgress(user: User, course: Course, data: ProgressData): Promise<void>
```

### learn.certify()

Issue a certification.

```typescript
function certify(user: User, certification: string): Promise<Certificate>
```
