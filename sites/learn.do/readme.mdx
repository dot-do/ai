# learn.do

**Comprehensive learning platform with interactive tutorials, courses, certification, and skill paths**

learn.do is the education and learning hub for the `.do` platform, providing structured learning paths, interactive tutorials, video courses, hands-on labs, and professional certification programs for developers, engineers, and business professionals.

## Quick Start

```bash
pnpm add learn.do
```

```typescript
import $, { learn } from 'learn.do'

// Get learning path
const path = await learn.getPath($.Developer, {
  focus: 'backend',
  level: 'intermediate',
})

// Enroll in course
await learn.enroll(user, $.Course, 'building-with-sdk')

// Track progress
await learn.trackProgress(user, course, {
  completed: true,
  score: 95,
})

// Get recommendations
const recommended = await learn.recommend(user, {
  based: 'history',
  limit: 5,
})
```

## Key Features

### Learning Paths

Structured learning journeys for different roles:

```typescript
import { learn } from 'learn.do'

// Get path for developer
const devPath = await learn.getPath($.Developer, {
  focus: 'full-stack',
  level: 'beginner',
})

// Path includes ordered courses
devPath.courses.forEach((course, index) => {
  console.log(`${index + 1}. ${course.name}`)
  console.log(`   Duration: ${course.duration}`)
  console.log(`   Level: ${course.level}`)
})

// Create custom path
const customPath = await learn.createPath({
  name: 'Custom Backend Path',
  description: 'Specialized backend development',
  courses: ['intro-to-sdk', 'database-operations', 'api-development', 'deployment-strategies'],
  estimatedTime: 'PT40H',
})

// Track path progress
const progress = await learn.pathProgress(user, devPath)
console.log(`Progress: ${progress.completed}/${progress.total} courses`)
```

### Interactive Courses

Hands-on courses with code exercises:

```typescript
// Get course details
const course = await learn.getCourse('building-with-sdk')

console.log(course.name)
console.log(`Modules: ${course.modules.length}`)
console.log(`Lessons: ${course.totalLessons}`)
console.log(`Duration: ${course.duration}`)

// Enroll user
await learn.enroll(user, course)

// Get next lesson
const nextLesson = await learn.nextLesson(user, course)

// Complete lesson with exercise
await learn.completeLesson(user, nextLesson, {
  code: userCode,
  passed: true,
  score: 95,
})

// Get certificate
if (await learn.courseCompleted(user, course)) {
  const cert = await learn.getCertificate(user, course)
  console.log(`Certificate: ${cert.url}`)
}
```

### Hands-On Labs

Interactive coding environments:

```typescript
// Create lab session
const lab = await learn.startLab('database-queries', {
  userId: user.$id,
  environment: 'sandbox',
})

// Lab provides isolated environment
console.log(`Lab URL: ${lab.url}`)
console.log(`Environment: ${lab.environment}`)

// Run code in lab
const result = await learn.runInLab(lab, {
  code: `
    import { db } from 'sdk.do'
    const businesses = await db.list($.Business)
    return businesses.length
  `,
})

// Check lab completion
await learn.completeLab(lab, {
  tasks: ['task-1', 'task-2', 'task-3'],
  allPassed: true,
})
```

### Video Tutorials

Interactive video content:

```typescript
// Get video course
const video = await learn.getVideo('intro-to-platform')

// Video with chapters and interactive elements
video.chapters.forEach((chapter) => {
  console.log(`${chapter.start} - ${chapter.title}`)
})

// Track viewing
await learn.trackVideo(user, video, {
  position: '00:15:30',
  completed: false,
})

// Interactive questions in video
video.questions.forEach((q) => {
  console.log(`At ${q.timestamp}: ${q.question}`)
})
```

### Skill Assessment

Test and certify skills:

```typescript
// Take assessment
const assessment = await learn.startAssessment('sdk-basics', {
  userId: user.$id,
  timeLimit: 'PT60M',
})

// Submit answers
await learn.submitAssessment(assessment, {
  answers: userAnswers,
})

// Get results
const results = await learn.getResults(assessment)
console.log(`Score: ${results.score}%`)
console.log(`Passed: ${results.passed}`)
console.log(`Percentile: ${results.percentile}`)

// Get certification
if (results.passed && results.score >= 80) {
  const cert = await learn.certify(user, 'sdk-professional')
}
```

## Use Cases

### Developer Onboarding

```typescript
import { learn } from 'learn.do'

async function onboardDeveloper(developer) {
  // Create onboarding path
  const path = await learn.createPath({
    name: 'Developer Onboarding',
    courses: ['platform-overview', 'sdk-fundamentals', 'database-basics', 'building-first-app'],
  })

  // Enroll developer
  await learn.enroll(developer, path)

  // Schedule check-ins
  every($.Weekly, async () => {
    const progress = await learn.pathProgress(developer, path)

    if (progress.percentage < 50) {
      await sendReminder(developer, {
        message: "Keep learning! You're halfway there.",
        nextLesson: progress.nextLesson,
      })
    }
  })

  // Celebrate completion
  on($.PathCompleted, async (event) => {
    if (event.userId === developer.$id && event.pathId === path.$id) {
      await celebrate(developer)
      await learn.certify(developer, 'platform-developer')
    }
  })
}
```

### Corporate Training

```typescript
async function manageCorporateTraining(organization) {
  // Create team
  const team = await learn.createTeam({
    name: organization.name,
    members: organization.developers,
    plan: 'enterprise',
  })

  // Assign training
  await learn.assignTraining(team, {
    paths: ['full-stack-developer', 'cloud-architect'],
    deadline: '2025-06-30',
  })

  // Track team progress
  const progress = await learn.teamProgress(team)
  console.log(`Team completion: ${progress.averageCompletion}%`)
  console.log(`Certifications: ${progress.certifications}`)

  // Generate reports
  const report = await learn.generateReport(team, {
    period: '2025-Q1',
    metrics: ['completion', 'time-spent', 'scores'],
  })
}
```
